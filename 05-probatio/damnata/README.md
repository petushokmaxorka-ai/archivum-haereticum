### V. СТРАТЕГИЯ TRANSMISSIO + RESISTENTIA

**DAMNATA** — **стратегия** **TRANSMISSIO** + **RESISTENTIA** (**передача**/**сопротивление**):

> **«**DAMNATA** — **«**передача**»** **ложного** **корпуса** (**«**фальсификации**»**/**«**подделки**»**); **«**сопротивление**»** **критике** (**«**сторонники** **не** **признают** **опровержение**»**); **«**наша** **самокритика**»** (**«**храним** **как** **предостережение**»**); **«**Велесова** **книга**/**Славяно-арии** = **«**общий** **закон** **опровергнутого**»**»** — **это** **TRANSMISSIO** **+** **RESISTENTIA**.

**Сравнение** **с** **другими** **стратегиями**:
- **DAMNATA** — **TRANSMISSIO** + **RESISTENTIA** (**передача** + **сопротивление**)
- **PROBATA** — **REPLICATIO** + **TRANSMISSIO** (**копирование** + **передача**)
- **Староверы** — **TRANSMISSIO** + **RESISTENTIA** (**передача** + **сопротивление**)
- **Средиземноморье** — **CONCURRENTIA** (**соперничество**)
- **Славия** — **REPLICATIO** (**копирование**)
