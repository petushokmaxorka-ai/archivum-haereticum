### III. СТРАТЕГИЯ REPLICATIO + TRANSMISSIO

**PROBATA** — **стратегия** **REPLICATIO** + **TRANSMISSIO** (**копирование**/**передача**):

> **«**PROBATA** — **«**копии**»** **первоначального** **знания** (**«**общий** **закон** **Божий**»**); **«**передача**»** **через** **историю** (**«**каналы**»**); **«**доказательство**»** **текстологическое** **+** **историческое** **+** **археологическое**; **«**Септуагинта**/**Кумран**/**Наг** **Х**а**м**а**д**ди**/**Масоретский** **текст** = **«**общий** **закон** **доказанного** **корпуса**»**»** — **это** **REPLICATIO** **+** **TRANSMISSIO**.

**Сравнение** **с** **другими** **стратегиями**:
- **PROBATA** — **REPLICATIO** + **TRANSMISSIO** (**копирование** + **передача**)
- **DAMNATA** — **TRANSMISSIO** + **RESISTENTIA** (**передача** + **сопротивление**)
- **Средиземноморье** — **CONCURRENTIA** (**соперничество**)
- **Славия** — **REPLICATIO** (**копирование**)
- **Ислам** — **CONCURRENTIA** + **NODUS** (**соперничество** + **узел**)
- **Утопическое** **священное** — **ISOLATIO** + **TRANSMISSIO** (**изоляция** + **передача**)
