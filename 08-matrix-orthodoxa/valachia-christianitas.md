### I. СВОДКА

**Валахия христианская** = **«х**р**и**с**т**и**а**н**с**т**в**о** В**а**л**а**х**и**и**»**:

**Время**: **I-XXI вв.** (от Святого Андрея до РумПЦ)
**Место**: **Валахия** (Арджеш, Куртя-де-Арджеш, Бухарест, Яссы)

**Ш**е**с**т**ь** **п**е**р**и**о**д**о**в**:
- **Д**а**к**о**-**р**и**м**с**к**о**е** **х**р**и**с**т**и**а**н**с**т**в**о** (**I**-**I**V** вв.**)** — *«**С**в**. **А**н**д**р**е**й**»**

- **С**л**а**в**я**н**с**к**о**е** **х**р**и**с**т**и**а**н**с**т**в**о** (**V**I**-**X** вв.**)** — *«**С**в**. **К**и**р**и**л**л** + **С**в**. **М**е**ф**о**д**и**й**»**
- **М**и**т**р**о**п**о**л**и**я** В**а**л**а**х**с**к**а**я** (**1**3**5**9**-**1**8**5**9**)** — *«**М**и**т**р**о**п**о**л**и**т** **В**а**л**а**х**с**к**и**й**»**
- **Ф**а**н**а**р**и**о**т**с**к**о**е** **п**р**а**в**о**с**л**а**в**и**е** (**1**7**1**1**-**1**8**2**1**)** — *«**Ф**а**н**а**р**и**о**т**ы**»**
- **Р**у**м**ы**н**с**к**а**я** **П**р**а**в**о**с**л**а**в**н**а**я** **Ц**е**р**к**о**в**ь** (**1**9**2**5**-**X**X**I** в**.)** — *«**А**в**т**о**к**е**ф**а**л**и**я**»**

**И**с**т**о**ч**н**и**к**и**:
- **«**Ж**и**т**и**е** **С**в**. **Н**и**к**о**л**а**»** (**X**I** в**.)** — *«**п**а**т**р**о**н**»**
- **«**М**и**т**р**о**п**о**л**и**т** **В**а**л**а**х**с**к**и**й**»** (**1**3**5**9**)** — *«**а**в**т**о**к**е**ф**а**л**и**я**»**
- **«**Р**у**м**ы**н**с**к**а**я** **П**р**а**в**о**с**л**а**в**н**а**я** **Ц**е**р**к**о**в**ь**»** (**1**9**2**5**)** — *«**А**в**т**о**к**е**ф**а**л**и**я**»*

**С**т**р**а**т**е**г**и**я** **TRANSMISSIO + NODUS + EDITIO**:
- **TRANSMISSIO**: *«**З**а**л**м**о**к**с**и**с** + **Г**е**б**л**е**и**з**и**с** → **Р**о**д** + **П**е**р**у**н** → **С**в**. **Н**и**к**о**л**а** + **И**и**с**у**с** **Х**р**и**с**т**о**с**»**
- **NODUS**: *«**В**а**л**а**х**и**я** = **у**з**е**л** **п**р**а**в**о**с**л**а**в**и**я** + **к**а**т**о**л**и**ц**и**з**м**а** + **у**н**и**а**т**с**т**в**а**»**
- **EDITIO**: *«**К**у**р**т**е**а** **д**е** **А**р**д**ж**е**ш**»** + **«**Р**у**м**П**Ц**»** (**1**9**2**5**)**»**

**Г**л**а**в**н**ы**е** **т**е**м**ы**:
- **К**у**р**т**е**а** **д**е** **А**р**д**ж**е**ш** (**1**5**1**7**-**1**5**2**1**)**
- **М**и**т**р**о**п**о**л**и**т** **А**н**т**и**м** **И**в**и**р**и**а**н**у**л** (**1**7**0**9**-**1**7**1**6**)

### XII. DAMNATA-ПРОТОКОЛ

**DAMNATA (нельзя писать как факт):**

1. «**П**р**а**в**о**с**л**а**в**и**е** = **р**у**м**ы**н**с**к**о**е**»** — DAMNATA. **«**П**р**а**в**о**с**л**а**в**и**е** = **В**и**з**а**н**т**и**й**с**к**о**е**»**
2. «**Р**у**м**ы**н**ы** = **и**с**т**и**н**н**ы**е** **п**р**а**в**о**с**л**а**в**н**ы**е**»** — DAMNATA. **«**Р**у**м**ы**н**ы** = **п**р**а**в**о**с**л**а**в**н**ы**е** + **к**а**т**о**л**и**к**и** + **у**н**и**а**т**ы**»*
3. «**К**а**т**о**л**и**ц**и**з**м** = **ч**у**ж**о**й**»** — DAMNATA. **«**К**а**т**о**л**и**ц**и**з**м** = **н**е**з**н**а**ч**и**т**е**л**ь**н**ы**й**»*
4. «**В**а**л**а**х**и**я** = **в**е**ч**н**о** **п**р**а**в**о**с**л**а**в**н**а**я**»** — DAMNATA. **«**В**а**л**а**х**и**я** = **д**а**к**о**-**р**о**м**а**н**с**к**о**-**п**р**а**в**о**с**л**а**в**н**а**я**»**
5. «**С**в**. **А**н**д**р**е**й** = **п**р**о**п**о**в**е**д**н**и**к** В**а**л**а**х**и**и**»** — DAMNATA. **«**С**в**. **А**н**д**р**е**й** = **л**е**г**е**н**д**а**»*
6. «**С**в**. **Н**и**к**о**л**а** = **З**а**л**м**о**к**с**и**с**»** — DAMNATA. **«**С**в**. **Н**и**к**о**л**а** = **п**а**т**р**о**н** **п**у**т**е**ш**е**с**т**в**е**н**н**и**к**о**в**»*
7. «**К**у**р**т**е**а** **д**е** **А**р**д**ж**е**ш** = **А**т**е**н**с**к**о**г**о** **п**а**р**т**н**ё**р**а**»** — DAMNATA. **«**К**у**р**т**е**а** **д**е** **А**р**д**ж**е**ш** = **в**а**л**а**ш**с**к**а**я** **а**р**х**и**т**е**к**т**у**р**а**»*
8. «**Р**у**м**ы**н**с**к**а**я** **П**р**а**в**о**с**л**а**в**н**а**я** **Ц**е**р**к**о**в**ь** = **а**в**т**о**к**е**ф**а**л**ь**н**а**я**»** — **у**с**л**о**в**н**о** (**+** **X**X**I** в**.)**»**
9. «**У**н**и**я** = **п**о**б**е**д**а**»** — DAMNATA. **«**У**н**и**я** = **п**о**п**ы**т**к**а**»*
10. «**Ф**а**н**а**р**и**о**т**ы** = **ч**у**ж**и**е**»** — DAMNATA. **«**Ф**а**н**а**р**и**о**т**ы** = **г**р**е**к**и**-**а**д**м**и**н**и**с**т**р**а**т**о**р**ы**»*
11. «**М**а**л**а**я** **В**а**л**а**х**и**я** = **а**в**т**о**н**о**м**и**я**»** — DAMNATA. **«**М**а**л**а**я** **В**а**л**а**х**и**я** = **а**в**т**о**н**о**м**н**ы**й** **р**а**й**о**н**»*
12. «**С**в**. **А**н**т**и**м** = **п**е**р**в**ы**й** **п**р**о**с**в**е**т**и**т**е**л**ь**»** — **у**с**л**о**в**н**о** (**+** **с**л**о**ж**н**а**я** **ф**и**г**у**р**а**)**»**
