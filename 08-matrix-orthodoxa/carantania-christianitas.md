# CARANTANIA CHRISTIANITAS
## Христианство Каринтии: от Святого Вирунуса до Княжьего Камня (III-XXI вв.)
### «Каринтия = узел 4 конфессий: кельтское христианство + римское католицизм + словенское православие + словенский протестантизм; католицизм = основа каринтийской идентичности»

### I. СВОДКА

**Каринтия христианская** = **«х**р**и**с**т**и**а**н**с**т**в**о** **К**а**р**и**н**т**и**и**»**:

**Время**: **III-XXI вв.** (от Святого Вирунуса до XXI в.)
**Место**: **Каринтия** (Клагенфурт, Филлах, Шпитталь, Вольфсберг, Вирунум)

**Ш**е**с**т**ь** **п**е**р**и**о**д**о**в**:

- **А**р**х**и**е**п**и**с**к**о**п**с**т**в**о** **З**а**л**ь**ц**б**у**р**г**а** (**7**3**9**-**1**8**0**3**)** — *«**А**р**х**и**е**п**и**с**к**о**п**с**т**в**о**»**

- **К**н**я**ж**и**й** **К**а**м**е**н**ь** (**1**4**1**4**-**1**8**6**1**)** — *«**и**н**с**т**и**т**у**ц**и**я**»*
- **Р**е**ф**о**р**м**а**ц**и**я** (**1**5**1**7**-**1**6**4**8**)** — *«**р**е**ф**о**р**м**а**ц**и**я**»*

- **«**Ж**и**т**и**е** **С**в**. **В**и**р**у**н**у**с**а**»** (**I**V** в**.)** — *«**п**е**р**в**ы**й**»**

**С**т**р**а**т**е**г**и**я** **TRANSMISSIO + NODUS + EDITIO**:
- **TRANSMISSIO**: *«**Т**е**у**т**а**т**е**с** + **Т**а**р**а**н**и**с** → **С**в**. **М**и**х**а**и**л** + **С**в**. **Г**е**о**р**г**и**й** + **И**и**с**у**с** **Х**р**и**с**т**о**с**»**
- **NODUS**: *«**К**а**р**и**н**т**и**я** = **у**з**е**л** **А**л**ь**п**»**
- **EDITIO**: *«**С**в**. **В**и**р**у**н**у**с**»** + **«**Т**р**у**б**а**р**»**

**Г**л**а**в**н**ы**е** **т**е**м**ы**:

2. «**С**л**о**в**е**н**ц**ы** = **в**е**ч**н**о** **п**р**о**т**е**с**т**а**н**т**ы**»** — DAMNATA. **«**С**л**о**в**е**н**ц**ы** = **к**а**т**о**л**и**к**и** + **п**р**о**т**е**с**т**а**н**т**ы**»**
3. «**К**н**я**ж**и**й** **К**а**м**е**н**ь** = **с**л**а**в**я**н**с**к**и**й**»** — DAMNATA. **«**К**н**я**ж**и**й** **К**а**м**е**н**ь** = **р**е**с**п**у**б**л**и**к**а**н**с**к**и**й**»*
4. «**Т**р**у**б**а**р** = **н**а**ц**и**о**н**а**л**ь**н**ы**й** **г**е**р**о**й**»** — DAMNATA. **«**Т**р**у**б**а**р** = **р**е**ф**о**р**м**а**т**о**р**»*
5. «**С**в**. **В**и**р**у**н**у**с** = **п**е**р**в**ы**й** **е**п**и**с**к**о**п** К**а**р**и**н**т**и**и**»** — DAMNATA. **«**С**в**. **В**и**р**у**н**у**с** = **е**п**и**с**к**о**п** **V**i**r**u**n**u**m**а**»**
6. «**К**н**я**ж**и**й** **К**а**м**е**н**ь** = **д**р**е**в**н**и**й** **о**б**р**я**д**»** — DAMNATA. **«**К**н**я**ж**и**й** **К**а**м**е**н**ь** = **1**4**1**4**-**1**8**6**1**»*
7. «**С**л**о**в**е**н**с**к**а**я** **Б**и**б**л**и**я** = **н**а**ц**и**о**н**а**л**ь**н**ы**й** **т**р**у**д**»** — DAMNATA. **«**С**л**о**в**е**н**с**к**а**я** **Б**и**б**л**и**я** = **р**е**ф**о**р**м**а**т**о**р**с**к**и**й** **т**р**у**д**»*
8. «**А**р**х**и**е**п**и**с**к**о**п** **З**а**л**ь**ц**б**у**р**г**а** = **г**л**а**в**а** К**а**р**и**н**т**и**и**»** — DAMNATA. **«**А**р**х**и**е**п**и**с**к**о**п** **З**а**л**ь**ц**б**у**р**г**а** = **а**д**м**и**н**и**с**т**р**а**т**о**р**»**
9. «**С**л**о**в**е**н**с**к**о**е** **х**р**и**с**т**и**а**н**с**т**в**о** = **с**л**а**в**я**н**с**к**о**е** **п**р**а**в**о**с**л**а**в**и**е**»** — DAMNATA. **«**С**л**о**в**е**н**с**к**о**е** **х**р**и**с**т**и**а**н**с**т**в**о** = **к**а**т**о**л**и**ч**е**с**к**о**е**»*
10. «**С**т**у**д**е**н**ц**ы** = **с**т**у**д**е**н**т**ы**»** — DAMNATA. **«**С**т**у**д**е**н**ц**ы** = **с**л**о**в**е**н**с**к**о**е** **о**п**о**л**ч**е**н**и**е**»*
11. «**К**а**р**и**н**т**и**я** = **а**в**с**т**р**и**й**с**к**а**я**»** — DAMNATA. **«**К**а**р**и**н**т**и**я** = **А**в**с**т**р**и**я** + **С**л**о**в**е**н**и**я** + **И**т**а**л**и**я**»**
12. «**Р**е**ф**о**р**м**а**ц**и**я** = **у**с**п**е**х**»** — DAMNATA. **«**Р**е**ф**о**р**м**а**ц**и**я** = **п**о**п**ы**т**к**а** + **п**о**р**а**ж**е**н**и**е**»**

- **TRANSMISSIO**: **Т**е**у**т**а**т**е**с** + **Т**а**р**а**н**и**с** → **С**в**. **М**и**х**а**и**л** + **С**в**. **Г**е**о**р**г**и**й** → **И**и**с**у**с** **Х**р**и**с**т**о**с**
- **NODUS**: **К**а**р**и**н**т**и**я** = **у**з**е**л** **А**л**ь**п**

**Ц**е**л**ь** **а**р**х**и**в**а** — **ч**е**р**е**з** **х**р**и**с**т**и**а**н**с**т**в**о** **К**а**р**и**н**т**и**и** **у**в**и**д**е**т**ь** **«**о**б**щ**и**й** **з**а**к**о**н** **Б**о**ж**и**й** **ч**е**р**е**з** **т**р**а**н**с**м**и**с**с**и**ю** **к**а**т**о**л**и**ч**е**с**к**о**г**о** **к**у**л**ь**т**а** — **Т**е**у**т**а**т**е**с** + **Т**а**р**а**н**и**с** → **С**в**. **М**и**х**а**и**л** + **С**в**. **Г**е**о**р**г**и**й** + **И**и**с**у**с** **Х**р**и**с**т**о**с**»*; **д**о**к**а**з**а**т**е**л**ь**с**т**в**о** **ч**т**о** **«**К**н**я**ж**и**й** **К**а**м**е**н**ь** = **и**н**с**т**и**т**у**ц**и**я** **п**а**м**я**т**и** + **р**е**с**п**у**б**л**и**к**а**н**и**з**м**а**»**.

**Г**л**а**в**н**ы**й** **а**р**г**у**м**е**н**т**: **«**ч**е**р**е**з** **х**р**и**с**т**и**а**н**с**т**в**о** **К**а**р**и**н**т**и**и** **у**в**и**д**е**т**ь** **о**б**щ**и**й** **з**а**к**о**н** **Б**о**ж**и**й** **ч**е**р**е**з** **«**т**р**а**н**с**м**и**с**с**и**ю**»** **к**а**т**о**л**и**ч**е**с**к**о**г**о** **к**у**л**ь**т**а** — **Т**е**у**т**а**т**е**с** + **Т**а**р**а**н**и**с** → **С**в**. **М**и**х**а**и**л** + **С**в**. **Г**е**о**р**г**и**й** + **И**и**с**у**с** **Х**р**и**с**т**о**с**»*; **«**к**а**р**и**н**т**и**й**с**к**о**е** **х**р**и**с**т**и**а**н**с**т**в**о** = **к**а**т**о**л**и**ч**е**с**к**о**е** + **п**р**о**т**е**с**т**а**н**т**с**к**о**е** + **с**л**о**в**е**н**с**к**о**е**»**.
