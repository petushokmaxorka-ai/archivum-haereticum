### I. СВОДКА

**Сербская Православная Церковь (СПЦ) (I-XXI вв. н.э.) = 1900 лет**:

**Три фазы**:
- **Раннее христианство** (I-IX вв. н.э.) — *«формирование»*
- **Автокефалия и расцвет** (1219-1459) — *«золотой век»*
- **Под османским игом** (1459-1878) — *«испытание»*

**Ключевые события**:
- **Первое крещение** (I-IV вв. н.э.) — *«при Св. Науме»*
- **Крещение при Кирилле и Мефодии** (867-879) — *«славянская литургия»*
- **Автокефалия** (1219) — *«Св. Савва, Афон»*
- **Хиландар** (1199) — *«Сербская Лавра»*
- **Сремская Митрополия** (1690) — *«великая миграция»*
- **Сербская автономия** (1830) — *«свобода СПЦ»*

**Ключевые святые**:
- **Св. Савва** (1175-1236) — *«главный святой»*
- **Св. Симеон** (Стефан Неманя, 1113-1199) — *«отец»*
- **Св. Стефан Первовенчанный** (1192-1227) — *«сын»*
- **Св. Стефан Дечанский** (1285-1331) — *«святой король»*
- **Св. Лазарь** (1329-1389) — *«Святой Князь Косово»*

**Ключевые монастыри**:
- **Хиландар** (1199) — *«Сербская Лавра»*
- **Студеница** (1190) — *«первый королевский монастырь»*
- **Милешева** (1220) — *«гробница Св. Саввы»*
- **Дечани** (1327) — *«гробница Св. Стефана Дечанского»*
- **Раваница** (1380) — *«гробница Св. Лазаря»*
- **Манасия** (1407) — *«последний великий монастырь»*

### VIII. «МАЛЕНЬКАЯ РОССИЯ» — ДУХОВНЫЙ БЛИЗНЕЦ

**1. ПАРАЛЛЕЛИ С РОССИЕЙ:
- **«**п**р**а**в**о**с**л**а**в**и**е** (**о**б**щ**е**е**)** — *«**г**л**а**в**н**о**е**»*
- **«**к**и**р**и**л**л**и**ц**а** (**о**б**щ**а**я**)** — *«**г**л**а**в**н**о**е**»*
- **«**с**л**а**в**я**н**с**к**а**я** о**с**н**о**в**а** (**о**б**щ**а**я**)** — *«**г**л**а**в**н**о**е**»*
- **«**С**в**. С**а**в**в**а** ≈ С**в**. С**е**р**г**и**й** Р**а**д**о**н**е**ж**с**к**и**й** (о**с**н**о**в**а**т**е**л**и** м**о**н**а**с**т**ы**р**с**к**о**г**о** ц**е**н**т**р**а**)** — *«**г**л**а**в**н**о**е**»*
- **«**С**в**. С**и**м**е**о**н** ≈ С**в**. С**е**р**г**и**й** Р**а**д**о**н**е**ж**с**к**и**й** (о**с**н**о**в**а**т**е**л**и**)** — *«**г**л**а**в**н**о**е**»*
- **«**Х**и**л**а**н**д**а**р** ≈ Т**р**о**и**ц**е**-**С**е**р**г**и**е**в**а** Л**а**в**р**а** (**г**л**а**в**н**ы**е** м**о**н**а**с**т**ы**р**и**)** — *«**г**л**а**в**н**о**е**»*
- **«**С**т**е**ф**а**н** Д**у**ш**а**н** ≈ И**в**а**н** III** (**с**о**б**и**р**а**т**е**л**и** з**е**м**е**л**ь**)** — *«**г**л**а**в**н**о**е**»*
- **«**К**о**с**о**в**о** (**1**3**8**9**) ≈ К**у**л**и**к**о**в**о** (**1**3**8**0**) (с**в**я**щ**е**н**н**ы**е** м**е**с**т**а**)** — *«**г**л**а**в**н**о**е**»*
- **«**С**П**Ц** ≈ **Р**П**Ц** (а**в**т**о**к**е**ф**а**л**ь**н**ы**е** п**р**а**в**о**с**л**а**в**н**ы**е** ц**е**р**к**в**и**)** — *«**г**л**а**в**н**о**е**»*

**2. РАЗЛИЧИЯ С РОССИЕЙ:
- **«**С**П**Ц** ≈ **Р**П**Ц** (а**в**т**о**к**е**ф**а**л**ь**н**ы**е** п**р**а**в**о**с**л**а**в**н**ы**е** ц**е**р**к**в**и**)** — *«**г**л**а**в**н**о**е**»*
- **«**С**П**Ц** = **8** **м**и**л**л**и**о**н**о**в** в**е**р**у**ю**щ**и**х** ≈ Р**П**Ц** = **1**0**0**+** м**и**л**л**и**о**н**о**в** — *«**г**л**а**в**н**о**е**»*
- **«**С**П**Ц** = **1** **п**а**т**р**и**а**р**х** (С**е**р**б**с**к**а**я** П**а**т**р**и**а**р**х**и**я**) ≈ Р**П**Ц** = **1** **п**а**т**р**и**а**р**х** (**М**о**с**к**о**в**с**к**а**я** П**а**т**р**и**а**р**х**и**я**)** — *«**г**л**а**в**н**о**е**»*
- **«**С**П**Ц** = **1**0** **а**р**х**и**е**п**и**с**к**о**п**с**т**в** ≈ Р**П**Ц** = **1**0**0**+** е**п**а**р**х**и**й** — *«**г**л**а**в**н**о**е**»*

**3. ОБЩИЕ ЧЕРТЫ С РОССИЕЙ:
- **«**п**р**а**в**о**с**л**а**в**и**е** — *«**о**б**щ**е**е**»*
- **«**к**и**р**и**л**л**и**ц**а** — *«**о**б**щ**а**я**»*
- **«**с**л**а**в**я**н**с**к**а**я** о**с**н**о**в**а** — *«**о**б**щ**а**я**»*
- **«**с**л**а**в**я**н**с**к**а**я** л**и**т**у**р**г**и**я** (с**т**а**р**о**с**л**а**в**я**н**с**к**а**я**)** — *«**о**б**щ**а**я**»*
- **«**С**в**. С**е**р**г**и**й** + С**в**. С**а**в**в**а** (о**б**а** — X**I**I**-X**I**V** в**в**.)** — *«**о**б**щ**а**я**»*

**4. ОТНОШЕНИЯ С РОССИЕЙ:
- **«**С**П**Ц** + **Р**П**Ц** = **о**б**щ**а**я** **п**а**м**я**т**ь** (К**о**с**о**в**о** ≈ К**у**л**и**к**о**в**о**)** — *«**г**л**а**в**н**о**е**»*
- **«**С**П**Ц** + **Р**П**Ц** = **о**б**щ**и**е** **с**в**я**т**ы**е** (С**в**. С**а**в**в**а** ≈ С**в**. С**е**р**г**и**й**)** — *«**г**л**а**в**н**о**е**»*
- **«**С**П**Ц** + **Р**П**Ц** = **о**б**щ**и**е** **м**о**н**а**с**т**ы**р**и** (Х**и**л**а**н**д**а**р** ≈ Т**р**о**и**ц**е**-**С**е**р**г**и**е**в**а** Л**а**в**р**а**)** — *«**г**л**а**в**н**о**е**»*

- **NODUS**: **С**П**Ц** = **у**з**е**л** **Б**а**л**к**а**н** (**п**р**а**в**о**с**л**а**в**и**е** + **с**л**а**в**я**н**с**т**в**о** + **Р**о**с**с**и**я**)**
- **EDITIO**: **С**П**Ц** ≈ **Р**П**Ц** (а**в**т**о**к**е**ф**а**л**ь**н**ы**е** п**р**а**в**о**с**л**а**в**н**ы**е** ц**е**р**к**в**и**)**

**Главный аргумент**: «**Сербская Православная Церковь (СПЦ) = одна из старейших автокефальных (1219); основана Св. Саввой; Хиландар на Афоне = "Сербская Лавра" ≈ Троице-Сергиева Лавра; СПЦ ≈ "маленькая Русская Православная Церковь" на Балканах — обе православные, кириллица, славянский богослужебный язык, общая история 1000+ лет**».

**Главный аргумент-сравнение**: «**Сербия = "маленькая Россия" на Балканах: Св. Савва ≈ Св. Сергий Радонежский (основатели монастырского центра, XIII-XIV вв.); Св. Симеон ≈ Св. Сергий Радонежский (отцы); Хиландар ≈ Троице-Сергиева Лавра (главные монастыри); СПЦ ≈ РПЦ (автокефальные православные церкви); Косово (1389) ≈ Куликово (1380) (священные места); обе православные, кириллица, славянская основа; РАЗЛИЧИЯ: СПЦ = 8 миллионов верующих, РПЦ = 100+ миллионов; СПЦ = 3500 приходов, РПЦ = 30000+ приходов**».
