# ALBANIA CHRISTIANITAS
## Христианство Албании: узел 4 конфессий — православие + католицизм + ислам + бекташизм (I-XXI вв.)
### «Албания = уникальный узел: ~40% православие (юг) + ~15% католицизм (север) + ~60% ислам + активный бекташизм; национальная идентичность через двоеверие»

### I. СВОДКА

**Албания христианская** = **«х**р**и**с**т**и**а**н**с**т**в**о** А**л**б**а**н**и**и**»**:

**Время**: **I-XXI вв.** (от Святого Никиты до XXI в.)
**Место**: **Албания** (Дуррес, Тирана, Шкодер, Берат, Охрид, Корча, Гирокастра, Косово)

**Ш**е**с**т**ь** **п**е**р**и**о**д**о**в**:
- **Р**а**н**н**е**е** **х**р**и**с**т**и**а**н**с**т**в**о** (**I**-**I**V** вв.**)** — *«**С**в**. **Н**и**к**и**т**а**»**

- **К**л**и**м**е**н**т** + **Н**а**у**м** (**I**X**-**X** вв.**)** — *«**с**л**а**в**я**н**с**к**о**е** **б**о**г**о**с**л**у**ж**е**н**и**е**»**
- **О**с**м**а**н**с**к**о**е** **п**о**д**в**л**а**с**т**и**е** (**1**3**8**5**-**1**9**1**2**)** — *«**и**с**л**а**м**и**з**а**ц**и**я**»**
- **Н**а**ц**и**о**н**а**л**ь**н**о**е** **в**о**з**р**о**ж**д**е**н**и**е** (**1**8**3**0**-**1**9**1**2**)** — *«**а**в**т**о**к**е**ф**а**л**и**я**»**
- **Х**Х**-**X**X**I** вв. **А**л**ь**б**а**н**и**я** (**1**9**1**2**-**X**X**I** в**.)** — *«**н**а**ц**и**о**н**а**л**ь**н**о**е** **г**о**с**у**д**а**р**с**т**в**о**»**

**И**с**т**о**ч**н**и**к**и**:
- **«**Ж**и**т**и**е** **С**в**. **Н**и**к**и**т**ы**»** (**I**I**I** в**.)** — *«**г**л**а**в**н**ы**й** **и**с**т**о**ч**н**и**к**»**
- **«**О**х**р**и**д**с**к**а**я** **а**р**х**и**е**п**и**с**к**о**п**и**я**»** (**X**I** в**.)** — *«**г**л**а**в**н**а**я**»**
- **«**К**о**н**с**т**а**н**т**и**н**о**п**о**л**ь**с**к**а**я** **П**а**т**р**и**а**р**х**и**я**»** (**X**V**I**I** в**.)** — *«**г**л**а**в**н**а**я**»**

**С**т**р**а**т**е**г**и**я** **TRANSMISSIO + NODUS + EDITIO**:
- **TRANSMISSIO**: *«**З**е**в**с** + **Г**е**р**а** + **А**р**е**с** → **С**в**. **Н**и**к**о**л**а** + **Б**о**г**о**р**о**д**и**ц**а** + **С**в**. **Г**е**о**р**г**и**й** → **А**л**л**а**х** + **М**у**х**а**м**м**а**д** → **С**и**н**к**р**е**т**и**з**м**»**
- **NODUS**: *«**А**л**б**а**н**и**я** = **у**з**е**л** **4** **к**о**н**ф**е**с**с**и**й**»**
- **EDITIO**: *«**Ж**и**т**и**е** **С**в**. **Н**и**к**и**т**ы**»** + **«**О**х**р**и**д**с**к**а**я** **а**р**х**и**е**п**и**с**к**о**п**и**я**»**

**Г**л**а**в**н**ы**е** **т**е**м**ы**:
- **С**в**. **Н**и**к**о**л**а** (**X**V**-**X**V**I**I** вв**.)** — *«**а**л**ь**б**а**н**с**к**и**й** **п**а**т**р**о**н**»**
- **А**П**О**К** (**1**9**2**2** н**.**э**.)** — *«**а**в**т**о**к**е**ф**а**л**и**я**»**
