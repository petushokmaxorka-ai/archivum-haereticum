### V. СТРАТЕГИЯ ISOLATIO + NODUS

**Эзотерическое** **православие** — **стратегия** **ISOLATIO** + **NODUS** (**изоляция**/**узел**):

> **«**Эзотерическое** **православие** — **«**скрытая**»** **традиция** (**«**не** **для** **всех**»**); **«**узел**»** **между** **исламом**/**иудаизмом**/**христианством**; **«**Хидр**/**Мелхиседек**/**Иоанн** = **«**общий** **закон** **эзотерического** **православия**»**»** — **это** **ISOLATIO** **+** **NODUS**.

**Сравнение** **с** **другими** **стратегиями**:
- **Хидр**/**Мелхиседек**/**Иоанн** — **ISOLATIO** + **NODUS** (**изоляция** + **узел**)
- **Эфиопия**/**Армения** — **ISOLATIO**/**RESISTENTIA** (**изоляция**/**резистенция**)
- **Семитское** — **ISOLATIO**/**RESISTENTIA** (**изоляция**/**резистенция**)
- **Средиземноморье** — **CONCURRENTIA** (**соперничество**)
- **Славия** — **REPLICATIO** (**копирование**)
- **Ислам** — **CONCURRENTIA** + **NODUS** (**соперничество** + **узел**)
