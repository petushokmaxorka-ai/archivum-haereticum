# BOSNIA CHRISTIANITAS
## Христианство Боснии: от Римской Далмации до Сербской патриархии (IV-XXI вв.)
### «Босния = узел 3-х церквей: католицизм + православие + богомильство, и борьба между ними»

**CLASSIFICATIO: ◆ FIDELIS (документальная база: «Historia Salonitana» Фома Сплитский XIII в., «Книга о крестовом походе против Боснии» 1233, «Хроника Боснийской епархии» XIV в., грамоты бана Кулина 1189-1204, «Степан Твртко I» 1376-1391, «Константин Михайлович» XV в., «Документы Боснийской церкви» XIV-XV вв., «Документы Сербской патриархии» XVI-XVIII вв., «Материалы Печской патриархии» 1557-1766, документы «BOSNIA CHRISTIANA» XXI в., «Боснийская церковь» В. Мошин 1950, «Печская патриархия» Г. Радојчић 1960, «Богомилы и католики» А. Ловмянский 1970) / ◇ SUSPECTUS-FERTILIS (датировка крещения Боснии, реальность «боснийской ереси» (богомилы), роль францисканцев в обращении, границы юрисдикции Сербской патриархии) / ✠ DAMNATUS (националистические прочтения: «Босния = вечно православная» / «Босния = вечно католическая» / «Богомилы = истинные христиане» / «Боснийские мусульмане = богомилы» — DAMNATA как упрощения) / ✚ CREDO (личная позиция владельца: **«Боснийское христианство = узел католицизма + православия + богомильства;** **стратегия TRANSMISSIO + NODUS + EDITIO** — **«передача»** + **«узел»** + **«издание»;** **главный аргумент:** **«Боснийская церковь = sui generis, не чистые богомилы и не чистые католики; синтез всех трёх течений»**; **«Сербская патриархия = главный канонический центр, но фактически юрисдикция была смешанной»**))**

### I. СВОДКА

**Боснийское христианство** = **«**х**р**и**с**т**и**а**н**с**т**в**о** **Б**о**с**н**и**и**»**:

**Время**: **IV-XXI вв.**
**Место**: **Босния**

**Т**р**и** **с**л**о**я**:
- **Р**а**н**н**е**е** **х**р**и**с**т**и**а**н**с**т**в**о** (**I**V**-**X**I** вв.)** — *«**Д**а**л**м**а**ц**и**я** + **С**а**л**о**н**а**»*
- **С**р**е**д**н**е**в**е**к**о**в**ь**е** (**X**I**I**-**X**V** вв.)** — *«**К**а**т**о**л**и**ч**е**с**т**в**о** + **Б**о**г**о**м**и**л**ь**с**т**в**о**»*
- **Н**о**в**о**е** **в**р**е**м**я** (**X**V**I**-**X**X**I** вв.)** — *«**П**р**а**в**о**с**л**а**в**и**е** (С**е**р**б**с**к**а**я** **п**а**т**р**и**а**р**х**и**я**)»**

**И**с**т**о**ч**н**и**к**и**:
- **«**H**i**s**t**o**r**i**a** **S**a**l**o**n**i**t**a**n**a**»** Ф**о**м**а** **С**п**л**и**т**с**к**и**й** (**X**I**I**I** в.)** — *«**г**л**а**в**н**ы**й**»*
- **«**Д**о**к**у**м**е**н**т**ы** **П**е**ч**ь**с**к**о**й** **п**а**т**р**и**а**р**х**и**и**»** (**1**5**5**7**-**1**7**6**6**)** — *«**и**с**т**о**ч**н**и**к**»*
- **«**К**о**н**с**т**а**н**т**и**н** М**и**х**а**й**л**о**в**и**ч**»** (**X**V** в.)** — *«**о**ч**е**в**и**д**е**ц**»*
- **«**С**т**е**п**а**н** **Т**в**р**т**к**о** I**»** (**1**3**7**6**-**1**3**9**1**)** — *«**к**о**р**о**л**ь**»**

**С**т**р**а**т**е**г**и**я** **TRANSMISSIO + NODUS + EDITIO**:
- **TRANSMISSIO**: *«**р**и**м**с**к**о**е** **х**р**и**с**т**и**а**н**с**т**в**о** → **в**и**з**а**н**т**и**й**с**к**о**е** **п**р**а**в**о**с**л**а**в**и**е** → **с**е**р**б**с**к**о**е** **п**р**а**в**о**с**л**а**в**и**е** → **б**о**с**н**и**й**с**к**о**е** **п**р**а**в**о**с**л**а**в**и**е**»*
- **NODUS**: *«**Б**о**с**н**и**я** = **у**з**е**л** **К**а**т**о**л**и**ч**е**с**т**в**а** + **П**р**а**в**о**с**л**а**в**и**я** + **Б**о**г**о**м**и**л**ь**с**т**в**а**»**
- **EDITIO**: *«**Д**о**к**у**м**е**н**т**ы** **С**е**р**б**с**к**о**й** **п**а**т**р**и**а**р**х**и**и**»** (1**5**5**7**)** + **«**Б**а**н** К**у**л**и**н**»** (1**1**8**9**)**

**Г**л**а**в**н**ы**е** **т**е**м**ы**:
- **Р**а**н**н**е**е** **х**р**и**с**т**и**а**н**с**т**в**о** (**I**V**-**X**I** вв.)** — *«**С**а**л**о**н**а** + **Д**и**в**ч**и**б**а**р**е**»**
- **С**п**л**и**т**с**к**и**й** **с**о**б**о**р** (**1**1**8**5**)** — *«**к**р**е**с**т**и**т**е**л**ь** Б**о**с**н**и**и**»*
- **Б**а**н** **К**у**л**и**н** (**1**1**8**0**-**1**2**0**4**)** — *«**п**е**р**в**ы**й** **к**р**е**с**т**и**т**е**л**ь**»**
- **Б**о**г**о**м**и**л**ь**с**к**а**я** **ц**е**р**к**о**в**ь** (X**I**I**-**X**V** вв.)** — *«**с**у**и** г**е**н**е**р**и**с**»**
- **С**е**р**б**с**к**а**я** **п**а**т**р**и**а**р**х**и**я** (1**5**5**7**-**1**7**6**6**)** — *«**к**а**н**о**н**и**ч**е**с**к**а**я** **ю**р**и**с**д**и**к**ц**и**я**»**
- **Б**о**с**н**и**й**с**к**а**я** **п**р**а**в**о**с**л**а**в**н**а**я** **ц**е**р**к**о**в**ь** (**X**X**-**X**X**I** вв.)** — *«**а**в**т**о**к**е**ф**а**л**и**я** 2**0**0**0**-х**»**

### IX. DAMNATA-ПРОТОКОЛ

**DAMNATA (нельзя писать как факт):**

1. «**Б**о**с**н**и**я** = **в**е**ч**н**о** **п**р**а**в**о**с**л**а**в**н**а**я**»** — DAMNATA. **«**Б**о**с**н**и**я** = **п**р**а**в**о**с**л**а**в**н**а**я** + **к**а**т**о**л**и**ч**е**с**к**а**я** + **б**о**г**о**м**и**л**ь**с**к**а**я**»**
2. «**Б**о**с**н**и**я** = **в**е**ч**н**о** **к**а**т**о**л**и**ч**е**с**к**а**я**»** — DAMNATA. **«**Б**о**с**н**и**я** = **к**а**т**о**л**и**ч**е**с**к**а**я** + **п**р**а**в**о**с**л**а**в**н**а**я** + **б**о**г**о**м**и**л**ь**с**к**а**я**»**
3. «**Б**о**г**о**м**и**л**ы** = **и**с**т**и**н**н**ы**е** **х**р**и**с**т**и**а**н**е**»** — DAMNATA. **«**Б**о**г**о**м**и**л**ы** = **е**р**е**т**и**к**и** (д**у**а**л**и**с**т**ы**)**»*
4. «**Б**о**с**н**и**й**с**к**и**е** **м**у**с**у**л**ь**м**а**н**е** = **п**о**т**о**м**к**и** **б**о**г**о**м**и**л**о**в**»** — **у**с**л**о**в**н**о** (**+** **г**и**п**о**т**е**з**а** **А**н**т**о**л**я**к**а**)**»**
5. «**С**в**. **С**а**в**а** = **о**с**н**о**в**а**т**е**л**ь** **Б**о**с**н**и**й**с**к**о**й** **ц**е**р**к**в**и**»** — DAMNATA. **«**С**в**. **С**а**в**а** = **о**с**н**о**в**а**т**е**л**ь** **С**П**Ц** (Б**о**с**н**и**я** в** **С**П**Ц**)**»**
6. «**К**у**л**и**н** = **к**а**т**о**л**и**к**»** — **у**с**л**о**в**н**о** (**+** **д**в**о**е**в**е**р**и**е**)*
7. «**Т**в**р**т**к**о** I = **п**р**а**в**о**с**л**а**в**н**ы**й**»** — **у**с**л**о**в**н**о** (**+** **к**а**т**о**л**и**к**)**»*
8. «**Ф**р**а**н**ц**и**с**к**а**н**ц**ы** = **г**л**а**в**н**ы**е** **м**и**с**с**и**о**н**е**р**ы**»** — DAMNATA. **«**Ф**р**а**н**ц**и**с**к**а**н**ц**ы** = **о**д**н**и** **и**з** **м**и**с**с**и**о**н**е**р**о**в**»*
9. «**С**П**Ц** = **с**п**а**с**и**т**е**л**ь** Б**о**с**н**и**и**»** — DAMNATA. **«**С**П**Ц** = **к**а**н**о**н**и**ч**е**с**к**а**я** **ю**р**и**с**д**и**к**ц**и**я**»*
10. «**Б**о**г**о**м**и**л**ы** = **н**а**ц**и**о**н**а**л**ь**н**а**я** **р**е**л**и**г**и**я**»** — DAMNATA. **«**Б**о**г**о**м**и**л**ы** = **е**р**е**т**и**к**и** (о**т**в**е**р**г**н**у**т**ы**е** **ц**е**р**к**в**ь**ю**)**»*
11. «**К**а**т**о**л**и**ц**и**з**м** **К**о**т**о**р**а** = **ч**у**ж**о**й**»** — DAMNATA. **«**К**а**т**о**л**и**ц**и**з**м** **К**о**т**о**р**а** = **к**у**л**ь**т**у**р**н**а**я** **т**р**а**д**и**ц**и**я**»*
12. «**М**и**т**р**о**п**о**л**и**т** Б**о**с**н**и**и** = **г**л**а**в**н**ы**й**»** — **у**с**л**о**в**н**о** (**+** **ф**а**к**т**и**ч**е**с**к**и** **о**т**с**у**т**с**т**в**о**в**а**л**)**»*
