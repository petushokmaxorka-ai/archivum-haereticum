### VI. DAMNATA-ПРОТОКОЛ

**DAMNATA (нельзя писать как факт):**

1. «**Китеж** — **«**выдумка**»** — DAMNATA. **Китеж** = **«**документированное** **народное** **предание**»** (**«**не** **выдумка**»**)
2. «**Беловодье** — **«**миф**»** — DAMNATA. **Беловодье** = **«**реальная** **история** **поисков**»** (**«**XVII**–**XIX** **вв.**»**)
3. «**Старцы** — **«**сектанты**»** — DAMNATA. **Старцы** = **«**старообрядческие** **наставники**»** (**«**не** **сектанты**»**)
4. «**Китеж** = **«**реальный** **город**»** — DAMNATA. **Китеж** = **«**священная** **топография**»** (**«**не** **обязательно** **реальный** **город**»**)
5. «**Беловодье** = **«**Алтай**»** — DAMNATA. **Беловодье** = **«**возможно**, **Алтай**»** (**«**не** **доказано**»**)
6. «**Опоньское** **царство** = **«**Польша**»** — DAMNATA. **Опоньское** **царство** = **«**«**Польша**»** **в** **староверческих** **текстах**»** (**«**возможно**, **другая** **страна**»**)
7. «**Кудеяр** = **«**исторический** **разбойник**»** — DAMNATA. **Кудеяр** = **«**фольклорный** **персонаж**»** (**«**не** **обязательно** **исторический**»**)
8. «**Бухтарминские** **староверы** = **«**католики**»** — DAMNATA. **Бухтарминские** **староверы** = **«**русские** **старообрядцы** **на** **Алтае**»** (**«**«**католики**»** — **публицистический** **миф** **для** **дискредитации**»**)
