### DECRETUM

Сей Архив фиксирует итоги разбора гипотезы единого корпуса: все религии и мифы человечества — ветви одного свода, разошедшиеся дрейфом после единого истока. Архив не проповедует. Архив **классифицирует**: каждая запись несёт печать достоверности. Опровергнутое не уничтожается — ему ставится печать DAMNATUS.

### SIGILLA VERITATIS — ПЕЧАТИ ДОСТОВЕРНОСТИ

| Печать | Значение |
|---|---|
| ◆ **FIDELIS** | Подтверждено источниками и академическим консенсусом |
| ◇ **SUSPECTUS** | Рабочая гипотеза: не доказано, не опровергнуто, проверяемо |
| ✠ **DAMNATUS** | Опровергнуто протоколом. Хранить как предостережение |
