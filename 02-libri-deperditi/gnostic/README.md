### VI. СТРАТЕГИЯ TRANSMISSIO + NODUS

**Гностическая** **стратегия** **TRANSMISSIO** + **NODUS** (**передача**/**узел**):

> **«**Гностицизм** **—** **«**передача**»** **первоначального** **знания**, **утраченного** **в** **каноне**; **«**узел**»** **между** **Египтом**, **Сирией**, **Грецией**; **двоеверие** **с** **христианством**/**иудаизмом**/**герметизмом**; **«**Демиург**/**Плерома**/**София** = **«**общий** **закон** **гностиков**»**»** — **это** **TRANSMISSIO** **+** **NODUS**.

**Сравнение** **с** **другими** **стратегиями**:
- **Гностицизм** — **TRANSMISSIO** + **NODUS** (**передача** + **узел**)
- **Церковь** **Востока** — **TRANSMISSIO** (**передача**)
- **Балканы** — **NODUS**/**TRANSMISSIO** (**узел**/**передача**)
- **Средиземноморье** — **CONCURRENTIA** (**соперничество**)
- **Славия** — **REPLICATIO** (**копирование**)
- **Эфиопия**/**Семитское** — **ISOLATIO**/**RESISTENTIA**

### VIII. DAMNATA-ПРОТОКОЛ

**DAMNATA (нельзя писать как факт):**

1. «**Гностики** — **«**еретики**»** — DAMNATA. **Гностики** = **«**полноценная** **религиозная** **традиция**»** (**«**параллельная** **передача**»**)
2. «**Наг-Хаммади** — **«**еретическая** **подделка**»** — DAMNATA. **Наг-Хаммади** = **«**гностическая** **библиотека**»** (**IV** **в.**)
3. «**Герметизм** — **«**языческая** **мудрость**»** — DAMNATA. **Герметизм** = **«**полноценная** **религиозная** **традиция**»** (**II**–**III** **вв.**)
4. «**Демиург** = **«**просто** **«**создатель**»** — DAMNATA. **Демиург** = **«**творец**»** (**«**создатель** **материального** **мира**»**)
5. «**София** = **«**просто** **«**мудрость**»** — DAMNATA. **София** = **«**падшая** **мудрость**»** (**«**искра** **света** **в** **материи**»**)
6. «**Пистис** **София** — **«**еретический** **текст**»** — DAMNATA. **Пистис** **София** = **«**гностический** **трактат**»** (**IV** **в.**)
7. «**Гностицизм** — **«**искажение** **христианства**»** — DAMNATA. **Гностицизм** = **«**параллельная** **передача**»** (**«**не** **искажение**»**)
