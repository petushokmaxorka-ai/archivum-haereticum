### VI. СТРАТЕГИЯ TRANSMISSIO + NODUS

**Апокрифическая** **стратегия** **TRANSMISSIO** + **NODUS** (**передача**/**узел**):

**Сравнение** **с** **другими** **стратегиями**:
- **Апокрифы** — **TRANSMISSIO** + **NODUS** (**передача** + **узел**)
- **Церковь** **Востока** — **TRANSMISSIO** (**передача**)
- **Балканы** — **NODUS**/**TRANSMISSIO** (**узел**/**передача**)
- **Средиземноморье** — **CONCURRENTIA** (**соперничество**)
- **Славия** — **REPLICATIO** (**копирование**)
- **Эфиопия**/**Семитское** — **ISOLATIO**/**RESISTENTIA**
