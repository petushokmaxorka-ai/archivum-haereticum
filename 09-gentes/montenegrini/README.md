### V. СВЯЗЬ С ОБЩЕЙ ЦЕЛЬЮ АРХИВА

**«**П**о**с**р**е**д**с**т**в**о**м** **Ч**е**р**н**о**г**о**р**и**и** **у**в**и**д**е**т**ь** **о**б**щ**и**й** **з**а**к**о**н** **Б**о**ж**и**й** **ч**е**р**е**з** **и**л**л**и**р**и**й**с**к**и**е** + **р**и**м**с**к**и**е** + **с**л**а**в**я**н**с**к**и**е** + **п**р**а**в**о**с**л**а**в**н**ы**е** **к**у**л**ь**т**ы** **п**р**и**р**о**д**н**ы**х** **с**и**л**»**; **д**о**к**а**з**а**т**е**л**ь**с**т**в**о** **«**п**а**р**а**л**л**е**л**ь**н**о**й** **п**е**р**е**д**а**ч**и** **о**б**щ**е**г**о** **и**н**д**о**е**в**р**о**п**е**й**с**к**о**г**о** **к**у**л**ь**т**а** **п**р**и**р**о**д**н**ы**х** **с**и**л** **в** **Ч**е**р**н**о**г**о**р**и**и**»**:
- **И**л**л**и**р**и**й**ц**ы** → **Д**и**с**-**П**а**т**р** + **С**и**л**ь**в**а**н** + **М**е**д**о**р** (**и**л**л**и**р**и**й**с**к**и**й** **к**у**л**ь**т** **п**о**д**з**е**м**н**о**г**о** **м**и**р**а** + **л**е**с**а** + **з**м**е**и**-**и**с**ц**е**л**и**т**е**л**я**)
- **Р**и**м**л**я**н**е** → **Ю**п**и**т**е**р** + **М**и**н**е**р**в**а** + **В**е**н**е**р**а** (**р**и**м**с**к**и**й** **к**а**п**и**т**о**л**и**й** **Д**о**к**л**е**и**)
- **С**л**а**в**я**н**е** → **Р**о**д** + **М**о**к**о**ш**ь** + **С**в**а**р**о**г** (**с**л**а**в**я**н**с**к**а**я** **в**е**р**а**)
- **Х**р**и**с**т**и**а**н**е** → **С**в**. **К**л**и**м**е**н**т** + **с**в**. **И**о**а**н**н** **В**л**а**д**и**м**и**р** (**п**р**а**в**о**с**л**а**в**н**а**я** **в**е**р**а**)

**Главный аргумент**: **«общий закон Божий через природные силы»** — **на** **земле** **Черногории** **сошлись** **4 волны** **одного** **и того же** **культа**:
- **И**л**л**и**р**и**й**с**к**а**я** (Д**и**с**-**П**а**т**р**)
- **Р**и**м**с**к**а**я** (Ю**п**и**т**е**р**/М**и**н**е**р**в**а**)
- **С**л**а**в**я**н**с**к**а**я** (Р**о**д**/М**о**к**о**ш**ь**)
- **П**р**а**в**о**с**л**а**в**н**а**я** (С**в**. **Б**о**г**о**р**о**д**и**ц**а** + **с**в**. **И**о**а**н**н** **В**л**а**д**и**м**и**р**)
- **К**а**т**о**л**и**ч**е**с**к**а**я** (К**о**т**о**р**с**к**и**е** **с**в**я**т**ы**е**)
- **И**с**л**а**м**с**к**а**я** (У**л**ц**и**н**ь**)
