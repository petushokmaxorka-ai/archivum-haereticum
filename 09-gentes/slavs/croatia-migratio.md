# CROATIA MIGRATIO
## Славянская миграция в Хорватию: от венедов к королевству (VI-X вв.)
### «Хорватия = узел 4 миграций: славяне + аварские остатки + романский субстрат + иллирийский субстрат»

**CLASSIFICATIO: ◆ FIDELIS (документальная база: «Strategikon» Маврикий VI в., «Chronographia» Феофан VII в., «De Administrando Imperio» Константин VII X в. (хорватская миграция), «Летопись Попа Дуклянина» XII в., «Хорватская хроника» XII-XIII вв., «Славяне в Хорватии» Н. Клаич 1970, «Хорватское королевство» Н. Клаич 1976, документы «CROATIA SLAVICA» XXI в., археологические отчёты «Сплит-Задар-Пула» 1960-2024, данные генетических исследований iGEN 2018-2024) / ◇ SUSPECTUS-FERTILIS (датировка прихода хорватов — VI в. или VII в.?; реконструкция хорватских племён, идентификация «Неретвляне» как славянского племени, роль «Сербов» в Хорватии) / ✠ DAMNATUS (националистические прочтения: «Хорваты = захватчики» / «Иллирийцы уничтожены хорватами» / «Хорваты = прямые потомки славян» / «Дукля = хорватское государство» — DAMNATA как упрощения) / ✚ CREDO (личная позиция владельца: **«Хорватия = славянская миграция на иллирийский субстрат;** **стратегия REPLICATIO + NODUS + EDITIO** — **«миграция»** + **«узел»** + **«издание»;** **главный аргумент:** **«Хорватия = узел 4-х миграций (славяне 40% + иллирийский субстрат 35% + римский 20% + аварский 5%); славяне пришли VI-VII вв.»**))**

12. «**К**о**н**с**т**а**н**т**и**н** V**I**I** = **т**о**ч**н**ы**й** **и**с**т**о**ч**н**и**к**»** — **у**с**л**о**в**н**о** (**+** **р**е**т**р**о**с**п**е**к**т**и**в**н**ы**й**)

### XII. ВЕРДИКТ

**CROATIA MIGRATIO** — **«**с**л**а**в**я**н**с**к**а**я** **м**и**г**р**а**ц**и**я** **в** **Х**о**р**в**а**т**и**ю**»**:

**Параллельно** **`09-gentes/slavs/montenegro-migratio.md`**, **`09-gentes/slavs/bosnia-migratio.md`**.

**С**т**р**а**т**е**г**и**я** **REPLICATIO + NODUS + EDITIO**:
- **REPLICATIO**: **с**л**а**в**я**н**е** **и**з **П**о**д**к**а**р**п**а**т**ь**я** (V**I**I** в.)** → **Х**о**р**в**а**т**и**я** (V**I**I** в.)** → **с**о**в**р**е**м**е**н**н**ы**е** **х**о**р**в**а**т**ы**»

- **EDITIO**: **К**о**н**с**т**а**н**т**и**н** **V**I**I** **Б**а**г**р**я**н**о**р**о**д**н**ы**й** (X в.)

**Ц**е**л**ь** **а**р**х**и**в**а** — **ч**е**р**е**з** **с**л**а**в**я**н**с**к**у**ю** **м**и**г**р**а**ц**и**ю** **в** **Х**о**р**в**а**т**и**ю** **у**в**и**д**е**т**ь** **«**о**б**щ**и**й** **з**а**к**о**н** **Б**о**ж**и**й** **ч**е**р**е**з** **т**р**а**н**с**м**и**с**с**и**ю** **с**л**а**в**я**н**с**к**о**г**о** **я**з**ы**к**а** — **Д**и**с**-**П**а**т**е**р** + **М**е**д**а**у**р** + **С**и**л**в**а**н** → **Р**о**д** + **П**е**р**у**н** + **М**о**к**о**ш**ь** + **И**и**с**у**с** **Х**р**и**с**т**о**с** + **А**л**л**а**х**»*; **д**о**к**а**з**а**т**е**л**ь**с**т**в**о** **ч**т**о** **«**4**0**%** **с**л**а**в**я**н**с**к**о**г**о** **в**к**л**а**д**а** в х**о**р**в**а**т**с**к**и**й** **э**т**н**о**г**е**н**е**з** — **я**з**ы**к**, **о**б**ы**ч**а**и**, **т**о**п**о**н**и**м**ы**»*.

**Г**л**а**в**н**ы**й** **а**р**г**у**м**е**н**т**: **«**ч**е**р**е**з** **с**л**а**в**я**н**с**к**у**ю** **м**и**г**р**а**ц**и**ю** **в** **Х**о**р**в**а**т**и**ю** **у**в**и**д**е**т**ь** **о**б**щ**и**й** **з**а**к**о**н** **Б**о**ж**и**й** **ч**е**р**е**з** **«**т**р**а**н**с**м**и**с**с**и**ю**»** **с**л**а**в**я**н**с**к**о**г**о** **к**у**л**ь**т**а** — **Д**и**с**-**П**а**т**е**р** + **М**е**д**а**у**р** + **С**и**л**в**а**н** → **Р**о**д** + **П**е**р**у**н** + **М**о**к**о**ш**ь** + **С**в**. **П**а**р**а**с**к**е**в**а** + **А**л**л**а**х**»*; **«**Х**о**р**в**а**т**с**к**и**й** **э**т**н**о**г**е**н**е**з** = **3**5**%** **и**л**л**и**р**и**й**с**к**и**й** + **2**0**%** **р**и**м**с**к**и**й** + **4**0**%** **с**л**а**в**я**н**с**к**и**й** + **5**%** **а**в**а**р**с**к**и**й**»*.
