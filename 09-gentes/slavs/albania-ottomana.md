### VIII. DAMNATA-ПРОТОКОЛ

**DAMNATA (нельзя писать как факт):**

1. «**А**л**б**а**н**и**я** = **р**а**б**с**т**в**о**»** — DAMNATA. **«**А**л**б**а**н**и**я** = **в**а**с**с**а**л**ь**н**о**е** **к**н**я**ж**е**с**т**в**о** + **а**в**т**о**н**о**м**н**ы**е** **з**а**р**ы**»**
2. «**О**с**м**а**н**ы** = **у**г**н**е**т**а**т**е**л**и**»** — DAMNATA. **«**О**с**м**а**н**ы** = **и**м**п**е**р**а**т**о**р**ы**»**
3. «**А**л**б**а**н**ц**ы** = **г**е**р**о**и**»** — DAMNATA. **«**А**л**б**а**н**ц**ы** = **с**л**о**ж**н**ы**е**»**
4. «**С**к**а**н**д**е**р**б**е**г** = **п**о**б**е**д**и**т**е**л**ь**»** — DAMNATA. **«**С**к**а**н**д**е**р**б**е**г** = **р**е**з**и**с**т**е**н**т**»**
5. «**А**л**б**а**н**ц**ы** = **о**д**н**а** **н**а**ц**и**я**»** — DAMNATA. **«**А**л**б**а**н**ц**ы** = **Г**е**г**ы** + **Т**о**с**к**и**»*
6. «**А**л**и** **П**а**ш**а** **Я**н**и**н**а** = **у**г**н**е**т**а**т**е**л**ь**»** — DAMNATA. **«**А**л**и** **П**а**ш**а** **Я**н**и**н**а** = **а**в**т**о**н**о**м**н**ы**й** **г**у**б**е**р**н**а**т**о**р**»**
7. «**А**л**б**а**н**с**к**а**я** **Л**и**г**а** = **н**е**з**а**в**и**с**и**м**о**с**т**ь**»** — DAMNATA. **«**А**л**б**а**н**с**к**а**я** **Л**и**г**а** = **а**в**т**о**н**о**м**и**я**»**
8. «**А**л**б**а**н**с**к**и**е** **к**л**а**н**ы** = **д**и**к**и**е**»** — DAMNATA. **«**А**л**б**а**н**с**к**и**е** **к**л**а**н**ы** = **с**т**р**у**к**т**у**р**а**»**
9. «**И**с**л**а**м** в А**л**б**а**н**и**и** = **н**а**с**и**л**и**е**»** — DAMNATA. **«**И**с**л**а**м** в А**л**б**а**н**и**и** = **в**ы**б**о**р**»*
10. «**С**к**а**н**д**е**р**б**е**г** = **В**Е**Л**И**К**И**Й**»** — **у**с**л**о**в**н**о** (**+** **к**о**н**т**е**к**с**т X**V** в**.)**»**
11. «**Л**и**г**а** **П**р**и**з**р**е**н**а** = **н**а**ц**и**о**н**а**л**ь**н**о**е** **д**в**и**ж**е**н**и**е**»** — **у**с**л**о**в**н**о** (**+** **с**л**о**ж**н**о**е**)**»**
12. «**Н**е**з**а**в**и**с**и**м**о**с**т**ь** 1**9**1**2** = **В**Е**Л**И**К**А**Я**»** — **у**с**л**о**в**н**о** (**+** **о**г**р**а**н**и**ч**е**н**н**а**я**)**»**
