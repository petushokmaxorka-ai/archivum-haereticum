**Три** **этапа**:
- **I** — **Предмиграционный** (**IV**-**V** **вв.**) — **п**р**е**д**в**и**ж**е**н**и**е** **с**л**а**в**я**н** **к** **Д**у**н**а**ю**

**Источники**:
- **П**р**о**к**о**п**и**й** **К**е**с**а**р**и**й**с**к**и**й** *Войны*, кн. V-VIII (**V**I** **в.**) — *«**С**к**л**а**в**и**н**ы** **и** **а**н**т**ы** **в** **М**а**к**е**д**о**н**и**и**»**

- **А**р**х**е**о**л**о**г**и**я**: **к**е**р**а**м**и**к**а** **П**р**а**г**а**-**К**о**р**ч**а**к** (**с**л**а**в**я**н**с**к**а**я**), **В**е**л**б**у**ж**д**/**К**ю**с**т**е**н**д**и**л** (Б**о**л**г**а**р**и**я**)

### V. ОСЕДАНИЕ В МАКЕДОНИИ (VII-VIII вв.)

**Т**е**р**р**и**т**о**р**и**а**л**ь**н**о**е** **з**а**с**е**л**е**н**и**е** (**VII**-**VIII** **вв.**):

**С**е**в**е**р**н**а**я** **М**а**к**е**д**о**н**и**я** (**С**к**о**п**ь**е**, **К**у**м**а**н**о**в**о**, **Т**е**т**о**в**о**):
- **С**а**г**у**д**а**т**ы** (**«**С**к**о**п**ь**е**»**) — **с**т**а**р**о**с**т** **С**к**о**п**ь**е** **с** **V**II** **в.** (т**о**п**о**н**и**м** **С**к**у**п**и**)
- **Б**р**з**я**к**и** (**«**К**у**м**а**н**о**в**о**»**)
- **Д**р**а**г**у**в**и**т**ы** (**«**Т**е**т**о**в**о**»**)

**З**а**п**а**д**н**а**я** **М**а**к**е**д**о**н**и**я** (**О**х**р**и**д**, **С**т**р**у**г**а**, **Д**е**б**а**р**):
- **В**е**л**е**г**е**з**и**т**ы** (**«**о**к**о**л**о** **В**е**л**е**с**а**»**)
- **Б**р**з**я**к**и** (**«**О**х**р**и**д**»**)
- **С**т**р**у**м**я**н**е** (**«**о**к**о**л**о** **С**т**р**у**м**ы**»**)

**Ю**ж**н**а**я** **М**а**к**е**д**о**н**и**я** (**С**а**л**о**н**и**к**и**, **С**е**р**р**ы**, **К**а**в**а**л**а**):
- **С**т**р**у**м**я**н**е** (**«**С**т**р**у**м**а**»**)
- **Р**и**н**х**и**н**е** (**«**о**к**о**л**о** **С**а**л**о**н**и**к**и**»**)
- **С**а**г**у**д**а**т**ы** (**«**С**е**р**р**ы**»**)

**В**о**с**т**о**ч**н**а**я** **М**а**к**е**д**о**н**и**я** (**С**т**р**у**м**и**ц**а**, **Ш**т**и**п**, **К**о**ч**а**н**и**):
- **С**т**р**у**м**я**н**е** (**«**С**т**р**у**м**и**ц**а**»**)
- **Б**р**з**я**к**и** (**«**Ш**т**и**п**»**)
- **С**а**г**у**д**а**т**ы** (**«**К**о**ч**а**н**и**»**)

**С**л**а**в**я**н**с**к**а**я** **т**о**п**о**н**и**м**и**к**а** **М**а**к**е**д**о**н**и**и** (**с**о**х**р**а**н**е**н**н**а**я** **д**о** **с**е**г**о**д**н**я**):

| **Т**о**п**о**н**и**м** | **Э**т**и**м**о**л**о**г**и**я** | **Э**т**н**и**ч**е**с**к**а**я** **п**р**и**в**я**з**к**а** |
|---|---|---|
| **С**к**о**п**ь**е** | **«**С**к**у**п**и**»** (р**и**м**с**к**о**е**) **→** **«**С**к**о**п**ь**е**»** (с**к**о**п**ь** = **с**т**а**д**о**) | **С**а**г**у**д**а**т**ы** |
| **В**е**л**е**с** | **«**В**е**л**е**с**»** = **б**о**г** **с**к**о**т**а** | **В**е**л**е**г**е**з**и**т**ы** |
| **Б**и**т**о**л**а** | **«**Б**и**т**о**л**а**»** = **м**о**н**а**с**т**ы**р**ь** | **Б**р**з**я**к**и** |
| **О**х**р**и**д** | **«**о**х**р**и**д**»** = **с**к**а**л**а** | **Б**р**з**я**к**и** |
| **Ш**т**и**п** | **«**В**е**л**е**с**»** (**«**Ш**т**и**п**»** = **«**Ш**т**и**п**ь**»**) | **С**т**р**у**м**я**н**е** |
| **К**у**м**а**н**о**в**о** | **«**К**у**м**а**н**о**в**о**»** = **к**у**м**а**н**ы** (**п**о**з**ж**е**)** | **С**а**г**у**д**а**т**ы** |
| **Т**е**т**о**в**о** | **«**т**е**т**о**»** = **с**в**ё**к**о**р** | **Д**р**а**г**у**в**и**т**ы** |
| **С**т**р**у**м**и**ц**а** | **«**С**т**р**у**м**а**»** | **С**т**р**у**м**я**н**е** |
| **П**р**и**л**е**п** | **«**п**р**и**л**и**п**ь**»** = **с**к**л**е**е**н**н**ы**й** | **С**а**г**у**д**а**т**ы** |
| **К**о**ч**а**н**и** | **«**к**о**ч**а**н**»** = **к**а**п**у**с**т**а** | **С**а**г**у**д**а**т**ы** |
| **Р**а**д**о**в**и**ш** | **«**Р**а**д**о**в**и**ш**»** = **в**е**с**ё**л**ы**й** | **С**т**р**у**м**я**н**е** |
| **К**р**и**в**а** **П**а**л**а**н**к**а** | **«**К**р**и**в**а**»** + **«**П**а**л**а**н**к**а**»** = **к**р**и**в**а**я** **у**к**р**е**п**л**е**н**и**е** | **Б**р**з**я**к**и** |
| **С**в**. **Н**и**к**о**л**е** | **«**С**в**е**т**и** **Н**и**к**о**л**а**»** (**п**о**з**ж**е**)** | **С**а**г**у**д**а**т**ы** |
| **М**а**в**р**о**в**о** | **«**М**а**в**р**о**в**о**»** = **ч**ё**р**н**о**е** (**о**з**е**р**о**) | **В**е**л**е**г**е**з**и**т**ы** |

**В**с**е**г**о** **с**л**а**в**я**н**с**к**и**х** **т**о**п**о**н**и**м**о**в** **в** **М**а**к**е**д**о**н**и**и**: **~**4**0**0** (к**а**р**т**а** **Ф**и**л**о**в**а 1952)

### VI. ВЗАИМОДЕЙСТВИЕ С МЕСТНЫМ НАСЕЛЕНИЕМ

**М**е**с**т**н**о**е** **н**а**с**е**л**е**н**и**е** **М**а**к**е**д**о**н**и**и** **в** **V**I** **в.**:
- **Р**о**м**е**и** (**г**р**е**к**о**-**р**и**м**с**к**о**е** **н**а**с**е**л**е**н**и**е**, **г**о**в**о**р**я**т** **н**а** **г**р**е**ч**е**с**к**о**м**)
- **Р**о**м**а**н**и**з**и**р**о**в**а**н**н**ы**е** **ф**р**а**к**и**й**ц**ы**/**и**л**л**и**р**и**й**ц**ы** (**в**л**а**х**и** — **р**о**м**а**н**и**з**а**ц**и**я** **V**-**VI** **вв.**)
- **М**а**л**ы**е** **г**р**е**ч**е**с**к**и**е** **а**н**т**и**ч**н**ы**е** **о**с**т**а**т**к**и**

**Т**р**и** **с**ц**е**н**а**р**и**я** **в**з**а**и**м**о**д**е**й**с**т**в**и**я**:

**С**ц**е**н**а**р**и**й** **1**: **«**С**л**а**в**я**н**с**к**о**е** **з**а**х**в**а**т**»** (**у**с**т**а**р**е**в**ш**и**й**):
- **С**л**а**в**я**н**е** **в**ы**т**е**с**н**и**л**и**/у**н**и**ч**т**о**ж**и**л**и** **м**е**с**т**н**о**е** **н**а**с**е**л**е**н**и**е**
- **П**р**о**б**л**е**м**а**: **а**р**х**е**о**л**о**г**и**я** **п**о**к**а**з**ы**в**а**е**т** **п**р**о**д**о**л**ж**е**н**и**е** **р**о**м**е**й**с**к**и**х** **т**р**а**д**и**ц**и**й**

**С**ц**е**н**а**р**и**й** **2**: **«**С**л**а**в**я**н**с**к**а**я** **и**н**ф**и**л**ь**т**р**а**ц**и**я**»** (**б**о**л**е**е** **в**е**р**о**я**т**н**ы**й**):
- **С**л**а**в**я**н**е** **п**р**и**х**о**д**я**т** **м**а**л**ы**м**и** **г**р**у**п**п**а**м**и** (**с**е**м**ь**я**м**и**, **р**о**д**а**м**и**)
- **О**с**е**д**а**ю**т** **с**р**е**д**и** **р**о**м**е**й**с**к**о**г**о** **н**а**с**е**л**е**н**и**я**
- **С**м**е**ш**и**в**а**ю**т**с**я** **ч**е**р**е**з** **б**р**а**к**и**, **т**о**р**г**о**в**л**ю**, **о**б**щ**у**ю** **ж**и**з**н**ь**

**С**ц**е**н**а**р**и**й** **3**: **«**Д**в**у**х**с**т**о**р**о**н**н**я**я** **а**с**с**и**м**и**л**я**ц**и**я**»** (**с**о**в**р**е**м**е**н**н**ы**й** **к**о**н**с**е**н**с**у**с**):
- **С**л**а**в**я**н**е** **п**р**и**н**и**м**а**ю**т** **р**о**м**е**й**с**к**у**ю** **к**у**л**ь**т**у**р**у** (**п**р**а**в**о**с**л**а**в**и**е**, **з**е**м**л**е**д**е**л**и**е**, **г**о**р**о**д**а**)
- **Р**о**м**е**и** **п**р**и**н**и**м**а**ю**т** **с**л**а**в**я**н**с**к**и**й** **я**з**ы**к** (**о**с**о**б**е**н**н**о** **в** **д**е**р**е**в**н**я**х**)
- **Р**е**з**у**л**ь**т**а**т**: **н**о**в**ы**й** **н**а**р**о**д** (**м**а**к**е**д**о**н**ц**ы**) = **с**л**а**в**я**н**с**к**а**я** **о**с**н**о**в**а** + **р**о**м**е**й**с**к**и**й** **с**у**б**с**т**р**а**т**

**А**р**х**е**о**л**о**г**и**ч**е**с**к**и**е** **д**а**н**н**ы**е**:
- **К**е**р**а**м**и**к**а** **П**р**а**г**а**-**К**о**р**ч**а**к** (**VI**-**VII** **вв.**) — **с**л**а**в**я**н**с**к**а**я**
- **К**е**р**а**м**и**к**а** **В**е**л**б**у**ж**д**/**К**ю**с**т**е**н**д**и**л** (**VII**-**VIII** **вв.**) — **с**л**а**в**я**н**с**к**а**я** + **р**о**м**е**й**с**к**а**я**
- **П**р**о**д**о**л**ж**е**н**и**е** **р**о**м**е**й**с**к**и**х** **т**р**а**д**и**ц**и**й** **в** **г**о**р**о**д**а**х** (С**а**л**о**н**и**к**и**, **С**т**р**у**м**и**ц**а**)
- **С**л**а**в**я**н**с**к**и**е** **д**е**р**е**в**н**и** (**о**с**н**о**в**н**о**й** **т**и**п** **п**о**с**е**л**е**н**и**я** **в** **V**II**-**IX** **вв.**)

### VII. ФОРМИРОВАНИЕ МАКЕДОНСКОЙ ИДЕНТИЧНОСТИ (VIII-X вв.)

**Э**т**н**о**г**е**н**е**т**и**ч**е**с**к**и**е** **п**р**о**ц**е**с**с**ы**:

**1. П**е**р**и**о**д** **«**д**о**с**л**а**в**я**н**с**к**о**г**о** **н**а**с**л**е**д**и**я**»** (**V**I**I**-**IX** **вв.**):
- **С**л**а**в**я**н**е** **с**о**х**р**а**н**я**ю**т** **р**о**м**е**й**с**к**и**е** **и**м**е**н**а** (**С**т**р**у**м**а** = **С**т**р**у**м**о**н**)
- **С**л**а**в**я**н**е** **п**о**к**у**п**а**ю**т**/о**т**н**и**м**а**ю**т** **з**е**м**л**и** **у** **р**о**м**е**е**в**
- **С**л**а**в**я**н**е** **н**е** **у**н**и**ч**т**о**ж**а**ю**т** **ц**е**р**к**в**и** (**С**в**. **Д**и**м**и**т**р**и**й** **С**о**л**у**н**с**к**и**й** = **п**а**т**р**о**н** **Ф**е**с**с**а**л**о**н**и**к**и**)

**2. П**е**р**и**о**д** **«**С**в**. **К**л**и**м**е**н**т**а**»** (**IX**-**X** **вв.**):
- **К**л**и**м**е**н**т** **О**х**р**и**д**с**к**и**й** (**~**8**4**0**-**9**1**6**) — **«**с**л**а**в**я**н**с**к**и**й** **З**л**а**т**о**у**с**т**»**
- **С**о**з**д**а**н**и**е** **с**л**а**в**я**н**с**к**о**й** **п**и**с**ь**м**е**н**н**о**с**т**и** **в** **М**а**к**е**д**о**н**и**и**
- **О**х**р**и**д**с**к**а**я** **ш**к**о**л**а** — **ц**е**н**т**р** **с**л**а**в**я**н**с**к**о**й** **у**ч**ё**н**о**с**т**и**
- **Н**а**у**м** **О**х**р**и**д**с**к**и**й** (**+**9**1**0**) — **«**у**ч**е**н**и**к** **К**и**р**и**л**л**а** + **М**е**ф**о**д**и**я**»**

**3. С**а**м**у**и**л**о**в**о** **ц**а**р**с**т**в**о** (**9**7**6**-**1**0**1**4**)**:
- **С**а**м**у**и**л** (**р**о**м**е**й**/**с**л**а**в**я**н**и**н**/**а**р**м**я**н**и**н** — **с**п**о**р**н**о**)
- **С**т**о**л**и**ц**а** **О**х**р**и**д** — **«**с**л**а**в**я**н**с**к**о**е** **ц**а**р**с**т**в**о** **с** **р**о**м**е**й**с**к**и**м** **и**н**с**т**и**т**у**ц**и**о**н**а**м**и**»**
- **И**с**п**о**л**ь**з**о**в**а**н**и**е** **с**л**а**в**я**н**с**к**о**г**о** **я**з**ы**к**а** **в** **а**д**м**и**н**и**с**т**р**а**ц**и**и**
- **С**о**х**р**а**н**е**н**и**е** **с**т**р**у**к**т**у**р**ы** **р**о**м**е**й**с**к**о**г**о** **п**р**а**в**а**
- **С**о**з**д**а**н**и**е** **п**е**р**в**о**г**о** **«**м**а**к**е**д**о**н**с**к**о**г**о**»** **с**а**м**о**с**о**з**н**а**н**и**я**

**4. П**о**с**л**е** **1**0**1**8** (**а**н**н**е**к**с**и**я** **В**и**з**а**н**т**и**и**):
- **В**и**з**а**н**т**и**й**с**к**а**я** **а**д**м**и**н**и**с**т**р**а**ц**и**я** **в** **М**а**к**е**д**о**н**и**и**
- **С**о**х**р**а**н**е**н**и**е** **с**л**а**в**я**н**с**к**о**г**о** **я**з**ы**к**а** **в** **ц**е**р**к**в**и** (**о**с**н**о**в**н**а**я** **м**а**с**с**а** **н**а**с**е**л**е**н**и**я**)
- **Ф**о**р**м**и**р**о**в**а**н**и**е** **«**м**а**к**е**д**о**н**с**к**о**й**»** **с**л**а**в**я**н**с**к**о**й** **и**д**е**н**т**и**ч**н**о**с**т**и**

**Г**е**н**е**т**и**к**а** **м**а**к**е**д**о**н**ц**е**в**:

- **М**т**ДНК**: **H** (**з**а**п**а**д**н**о**е**в**р**о**п**е**й**с**к**и**й**), **U** (**д**р**е**в**н**и**й** **Е**в**р**о**п**ы**), **K** (**з**а**п**а**д**н**о**е**в**р**о**п**е**й**с**к**и**й**)
- **Narasimhan** **2019** (**Southern** **Arc**): **с**л**а**в**я**н**с**к**а**я** **п**р**и**б**ы**л**ь** **VI**-**VII** **вв.** **п**о**д**т**в**е**р**ж**д**е**н**а**
- **Hellenthal** **2021**: **м**а**к**е**д**о**н**ц**ы** = **с**л**а**в**я**н**с**к**а**я** **п**р**и**б**ы**л**ь** + **б**а**л**к**а**н**с**к**и**й** **с**у**б**с**т**р**а**т**

### X. ТРИ ВЕРСИИ ЭТНОГЕНЕЗА МАКЕДОНЦЕВ

**В**е**р**с**и**я** **1**: **«**Ч**и**с**т**ы**е** **с**л**а**в**я**н**е**»** (**д**о**к**а**з**а**т**е**л**ь**с**т**в**о** — **г**е**н**е**т**и**к**а**):
- **Y-ДНК** **R1a** (**с**л**а**в**я**н**с**к**и**й**) — **с**т**а**т**и**с**т**и**ч**е**с**к**и** **з**н**а**ч**и**м**ы**й**
- **М**а**к**е**д**о**н**с**к**и**й** **я**з**ы**к** = **ю**ж**н**о**с**л**а**в**я**н**с**к**и**й**
- **С**л**а**в**я**н**с**к**а**я** **т**о**п**о**н**и**м**и**к**а** (~**4**0**0** **т**о**п**о**н**и**м**о**в**)

**В**е**р**с**и**я** **2**: **«**С**л**а**в**я**н**с**к**о**-**р**о**м**е**й**с**к**и**й** **с**п**л**а**в**»** (**д**о**к**а**з**а**т**е**л**ь**с**т**в**о** — **и**с**т**о**р**и**я**):
- **Р**о**м**е**й**с**к**о**е** **н**а**с**е**л**е**н**и**е** **н**е** **у**н**и**ч**т**о**ж**е**н**о** (**С**а**л**о**н**и**к**и**, **С**т**р**у**м**и**ц**а**)
- **А**р**х**е**о**л**о**г**и**я** **п**о**к**а**з**ы**в**а**е**т** **п**р**о**д**о**л**ж**е**н**и**е** **р**о**м**е**й**с**к**о**й** **к**у**л**ь**т**у**р**ы**
- **О**х**р**и**д**с**к**а**я** **ш**к**о**л**а** = **с**л**а**в**я**н**с**к**и**й** **я**з**ы**к** + **г**р**е**ч**е**с**к**и**й** **б**о**г**о**с**л**у**ж**е**б**н**ы**й**

**В**е**р**с**и**я** **3**: **«**М**у**л**ь**т**и**э**т**н**и**ч**е**с**к**а**я** **с**м**е**с**ь**»** (**д**о**к**а**з**а**т**е**л**ь**с**т**в**о** — **г**е**н**е**т**и**к**а** + **а**р**х**е**о**л**о**г**и**я**):
