# POLABIA WOLGAST
## Вольгаст (Wolgast) — капище Яровита, столица Вольгастской линии Грифитов
### «Вольгаст (Wolgast / Woligast) = славянский город в Полабии, столица славян-бодричей, главное капище Яровита (Jarovit); разрушен Генрихом Львом 1160-1167 гг.н.э.; столица Вольгастской линии Поморского герцогства; династия Грифитов; современный Вольгаст в Мекленбурге-Передней Померании»

**CLASSIFICATIO: ◆ FIDELIS / ◇ SUSPECTUS-FERTILIS / ✠ DAMNATUS / ✚ CREDO: «Вольгаст = славянский город, столица бодричей, капище Яровита»**
Datum: 112.M3

---

### I. СВОДКА

**Вольгаст (Wolgast / Woligast)** — славянский город в Полабии, столица славян-бодричей, главное капище бога **Яровита** (Jarovit). Расположен на острове, напротив острова Рюген. Разрушен Генрихом Львом в 1160-1167 гг.н.э. Стал столицей Вольгастской линии Поморского герцогства (Грифиты). Современный Вольгаст — земля Мекленбург-Передняя Померания.

**Главные тезисы**:
1. **«Wolgast»** = «Wolg-» (Волк) + «-ast» (город) = *«Волчий город»*
2. **Славянское название** = «Woligast» (Волигаст)
3. **Столица** = славян-бодричей
4. **Капище Яровита** = главное святилище бодричей
5. **1160-1167 н.э.** — разрушено Генрихом Львом
6. **XII-XVII вв. н.э.** — столица Вольгастской линии Грифитов

---

### II. ЭТИМОЛОГИЯ

**«Wolgast»** = **«Wolg-» (Волк) + «-ast» (город)** = *«Волчий город»*
**«Woligast»** = **«Woli-» (Воли) + «-gast» (гаст)** = *«Волигость»*
**«Wolgast»** = славянское название
**«Wolgast»** = «волк» (Wolf) + «город» (Gast)
**«Wolgast»** = «Wolg» (Волк) + «-ast» (гость, город)
**«Вольгаст»** = «Волчий город» (как «Волгоград»)

**2. ОПИСАНИЕ ЯРОВИТА:
- **Яровит** (Jarovit) = «Яр-» (яр) + «-вит» (свет) = *«Яросвет»*

- **Яровит** = «Яр-» (ярость) + «-вит» (свет)
- **Яровит** = изображался с щитом
- **Яровит** = описывался как воин

**3. ДВОЕВЕРИЕ: ЯРОВИТ = СВ. ГЕОРГИЙ:
- **Яровит** = бог войны = Св. Георгий
- **Яровит** = воин = воин

**4. ЦЕРКОВЬ СВ. ГЕОРГИЯ:
