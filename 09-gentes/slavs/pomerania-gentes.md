## Кашубы, словинцы, куявы, полабяне — подплемена поморян (V-XXI вв. н.э.)

### I. СВОДКА

**Поморские подплемена (V-XXI вв.н.э.)** — живые потомки Поморского герцогства, говорящие на поморских диалектах. Главные подплемена: **кашубы** (500 000, самые многочисленные), **словинцы** (2000-5000, самые консервативные), **куявы** (Куявия), **полабяне** (группа кашубов).

**Главные тезисы**:
1. **КАШУБЫ (KASZUBY)** = 500 000, в Кашубии (Гданьск, Гдыня, Вейхерово)
2. **СЛОВИНЦЫ (SŁOWIŃCY)** = 2000-5000, в Словиньце (Слупск, Лемборк)
3. **КУЯВЫ (KUJAWIA)** = 1 000 000, в Куявии (Быдгощ, Торунь, Влоцлавек)
4. **ПОЛАБЯНЕ (POŁABIANIE)** = 200 000, в долине Вислы
5. **4 живых племени** = потомки Поморского герцогства

### II. КАШУБЫ (KASZUBY) — 500 000

**1. ОСНОВНЫЕ СВЕДЕНИЯ:
- **Кашубы** (Kaszuby / Kaszëbi) = «Kašub-» (Кашуб) + «-y» (земля) — *«земля Кашубов»*
- **Кашубы** = крупнейшее поморское подплемя
- **Кашубы** = 500 000 (XXI в.)
- **Кашубы** = живут в Кашубии (Гданьск, Гдыня, Вейхерово, Бытув, Хойнице, Картузы, Косцежина, Щецин)
- **Кашубы** = говорят на кашубском языке (kaszëbsczi jãzëk)
- **Кашубы** = католики (преимущественно)

**2. ЭТИМОЛОГИЯ:
- **«Кашубы»** = **«Kašub-» (Кашуб)** + **«-y» (земля)** — *«земля Кашубов»*
- **«Кашубы»** = **«Kaš-» (Каш)** + **«-ub-» (уба, уб)** — *«Кашуба»*
- **«Кашубы»** = **«Kaš-» (Каш)** + **«-ub-» (Уб)** + **«-y» (земля)** — *«земля Кашуба»*
- **«Кашубы»** = **«Kašub» (имя легендарного предка Кашуба)** + **«-y» (земля)** — *«земля Кашуба»*
- **«Кашуб»** = легендарный предок кашубов (XII в.)

**3. ИСТОРИЯ:
- **V-VI вв. н.э.** — кашубы = славянское племя
- **X-XII вв. н.э.** — кашубы = часть Поморского герцогства
- **XII в. н.э.** — кашубы крещены Отто Бамбергским (1124-1128)
- **XIII в. н.э.** — кашубы под властью Тевтонского ордена (1308-1466)
- **XIII в. н.э.** — кашубы под властью Польши (1466-1772)
- **1772 н.э.** — кашубы = Пруссии (Первый раздел)
- **1920 н.э.** — кашубы = Польше (после WWI)
- **1939-1945 н.э.** — кашубы под немецкой оккупацией
- **1945 н.э.** — кашубы = Польше (после WWII)

**4. ЯЗЫК:
- **Кашубский язык** (kaszëbsczi jãzëk) = «Kašub-» (Кашуб) + «-sczi» (-ский) + «jãzëk» (язык) = *«Кашубский язык»*
- **Кашубский язык** = западнославянский
- **Кашубский язык** = близок к польскому
- **Кашубский язык** = имеет 5 диалектов
- **Кашубский язык** = литературный с XIX в.
- **Кашубский язык** = 500 000 носителей (XXI в.)
- **Кашубский язык** = признан региональным языком в Польше (2005)

**5. КУЛЬТУРА:
- **Кашубская литература** = XIX-XXI вв.н.э.
- **Кашубская музыка** = народная
- **Кашубский фольклор** = сказки, легенды, песни
- **Кашубская кухня** = рыба, картофель, капуста
- **Кашубский костюм** = народный
- **Кашубские праздники** = ежегодные

**6. ГЕОГРАФИЯ:
- **Кашубия** (Kaszuby) = регион в Польше
- **Гданьск** = главный город
- **Гдыня** = крупный порт
- **Вейхерово** = «столица Кашубии»
- **Бытув** = «кашубский Иерусалим»
- **Хойнице** = город
- **Картузы** = город (название от монахов-картузианцев)
- **Косцежина** = «кашубский Версаль»
- **Щецин** = крупный город (частично кашубский)

**7. РЕЛИГИЯ:
- **Кашубы** = католики (95%)
- **Кашубы** = имеют кашубский крест
- **Кашубы** = имеют кашубский святой (Св. Войцех)
- **Кашубы** = имеют кашубский обряд
- **Кашубы** = двоеверие (католицизм + языческие пережитки)

**8. СОВРЕМЕННОСТЬ:
- **Кашубы** = 500 000 носителей кашубского языка
- **Кашубы** = имеют кашубское телевидение
- **Кашубы** = имеют кашубский музей
- **Кашубы** = имеют кашубский парламент (Zrzëszënie Kaszëbskò)
- **Кашубы** = имеют кашубский флаг (белый крест на синем)
- **Кашубы** = имеют кашубскую газету
