# POLABIA LÜBECK & STRALSUND
## Любек (Любеке) + Штральзунд (Стреле): ганзейские столицы полабских славян
### «Любек = славянское «Любеке» (Lybeke = «любимое место»); Штральзунд = славянское «Стреле» (Stréle = «стрелка/пролив»); оба = славянские порты ободритов, ставшие ганзейскими столицами; Любек = «королева Ганзы» (1150-1500); двоеверие до XVIII в.»

**CLASSIFICATIO: ◆ FIDELIS («Славянская хроника» Гельмольда Босау, «Топонимия Балтики» Г. Траутманн 1949, «Славянские топонимы Германии» К. Цёрниг 1991, «Бодричи» Л. Нидерле 1910, «Ганза и славяне» 1990) / ◇ SUSPECTUS-FERTILIS / ✠ DAMNATUS / ✚ CREDO: «Любек + Штральзунд = славянские порты»**
Datum: 080.M3

### I. СВОДКА

**Любек (Lübeck)** — город в федеральной земе Шлезвиг-Гольштейн (~215 000 жителей), «королева Ганзы» (1150-1500). Название — **славянское**, от «любимый» (Lybeke = «любимое место»).

**Штральзунд (Stralsund)** — портовый город в Мекленбурге-Передней Померании (~60 000 жителей). Название — **славянское**, от «стрела» (Stréle = «стрелка/пролив»).

Оба — славянские порты ободритов, ставшие ганзейскими столицами.

### II. ЭТИМОЛОГИЯ

**«Любек» (Lübeck)** = **«Lybeke» (Любеке)** = **«любимое место»**
- **«**L**ü**b**e**c**k**»** = **«**L**y**b**e**k**e**»** (**Л**ю**б**е**к**е**)** — *«**л**ю**б**и**м**о**е** м**е**с**т**о**»*
- **«**L**y**b**e**k**e**»** = **«**л**ю**б**»** (**л**ю**б**и**м**ы**й**)** + **«**e**k**e**»** (**с**у**ф**ф**и**к**с м**е**с**т**а**)** — *«**л**ю**б**и**м**о**е** м**е**с**т**о**»*
- **Г. **Т**р**а**у**т**м**а**н**н** (**1**9**4**9**)** — **«**L**ü**b**e**c**k** = **L**y**b**e**k**e** (л**ю**б**и**м**о**е**)**»** — *«**г**л**а**в**н**о**е**»*

**«Штральзунд» (Stralsund)** = **«Stréle» (Стреле)** = **«стрелка/пролив»**
