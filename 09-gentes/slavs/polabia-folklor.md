# POLABIA FOLKLOR
## Фольклор полабского мира: сказки, легенды, песни, заговоры

**CLASSIFICATIO: ◆ FIDELIS (документальная база: «Wendische Sagen» А. Шнайдера 1862, «Volkssagen der Lausitzer Wenden» В. Шульца 1880, «Sorbische Volksmärchen» Л. Гаупт и Л. Шмидт 1902, «Volksbräuche und Volksglaube der Wenden» В. Матерна 1905, «Sorbische Volkskunde» П. Нойгебауэра 1955, «Sagenbuch der Lausitzer Sorben» Х. Кравчик 1979, «Sorbische Märchen» М. Ланге 1990) / ◇ SUSPECTUS-FERTILIS (точная атрибуция текстов, локализация) / ✠ DAMNATUS (националистические прочтения: «Полабский фольклор = немецкий» / «Лужичане = поляки» — DAMNATA) / ✚ CREDO: «Полабский фольклор = живое наследие вымершего мира; через лужичан он СОХРАНИЛСЯ»**

### I. СВОДКА

**Фольклор полабского мира** — устное наследие полабских славян, пережившее гибель большинства племён в XII в. СОХРАНИЛОСЬ через лужичан (единственный живой полабский народ) в форме: сказок, легенд, песен, заговоров, преданий, обрядовых текстов. Главные источники: записи XIX-XX вв. (Шнайдер 1862, Шульц 1880, Гаупт и Шмидт 1902, Матерна 1905, Нойгебауэр 1955, Кравчик 1979, Ланге 1990).

**Главные тезисы**:
1. **Сказки** — славянские параллели (Krabat, солдат и чёрт, брат и сестра)
2. **Легенды** — о Krbabu (водяной/чудище), о Подземном короле, о Соляном замке
3. **Песни** — колядки, купальские, свадебные, погребальные
4. **Заговоры** — лечебные, хозяйственные, любовные, защитные
5. **Эпос** — «Mikławs z Kórnja» (лужицкий эпос, К. Б. Пфил 1841)
6. **Предания** — о Сербском Рае, о Витке (Witka)

### II. СКАЗКИ ЛУЖИЧАН

**1. КРАБАТ (Krabat) — главная лужицкая легенда:
- **Krabat** (Крабат) = «ворон, воронёнок» (сербск. Krawat)
- **Krabat** = юноша, проданный мельнику-чернокнижнику в Сорбском Шварцкольме
- **Krabat** — параллель с Гансом Хеннингом Яном (Johannes Jänike, XVII в.)
- **Krabat** — роман Отто Ниггемаера (1976) и фильм «Krabat» (2008)
- **Krabat** = славянский параллель «Фауста» (немецкая легенда о Фаусте)
- **Krabat** — в Лужице считали: Крабат «наш» юноша, продавший душу дьяволу

**2. ТИПЫ СКАЗОК (по Аарне-Томпсону):
- **АТ 300** — «Победитель змея» (Серб, победивший змея)
- **АТ 313** — «Чудесное бегство» (Брат и сестра бегут от ведьмы)
- **АТ 327А** — «Ганс и Грета» (славянский вариант — Hanka a Hanka)
- **АТ 400** — «Муж ищет пропавшую жену»
- **АТ 425** — «Поиски исчезнувшей невесты» (Серб ищет невесту)
- **АТ 510** — «Золушка» (славянский вариант)
- **АТ 552** — «Девушка-змея» (Sorbische Dróhka)
- **АТ 706** — «Девушка без рук» (Hanka bźez rucy)
- **АТ 1000-1029** — Звери-мужья (муж-собака, муж-змей)
- **АТ 1300-1439** — Сказки о дураках (Hlupak)
- **АТ 3000-3299** — Сказки о чёрте (Sorbiske čort)

**3. ГЕРОИ СКАЗОК:
- **Hlupak** (Глупак) — типичный герой (аналог русского «Иванушки-дурачка»)
- **Mudra Hana** (Мудрая Хана) — мудрая девушка
- **Kralowna** (Королевна) — царевна
- **Serbja** (Сербы) — типичные герои
- **Krbab** (Крбаб) — отрицательный герой (аналог Кощея)
- **Wódny muž** (Водяной муж) — водяной
- **Lesny muž** (Лесной муж) — леший

### III. ЛЕГЕНДЫ ЛУЖИЧАН

**1. СОЛЯНОЙ ЗАМОК В БАУЦЕНЕ (Ortenburg):
- **Soleny hród** = Соляной замок = Salzausstellung Бауцена
- **Ortenburg** = замок в Бауцене, на горе Schlossberg
- **Легенда**: В подземелье Ортенбурга спрятаны соляные сокровища
- **Легенда**: Каждые 7 лет соляной дух выходит из замка
- **Легенда**: Соляная Дева (Solenka) охраняет сокровища
- **Легенда**: «Кто войдёт в подземелье — найдёт сокровище, но не сможет выйти»
- **Реальность**: В 2006 г. найдены соляные пещеры под Бауценом

**2. ПОДЗЕМНЫЙ КОРОЛЬ (Podzemski Kral):
- **Podzemski Kral** = Подземный король = Unterirdische König
- **Легенда**: Под землёй живёт Подземный Король со своей армией
- **Легенда**: Когда наступит час, он выйдет и победит врагов лужичан
- **Легенда**: Армия Подземного Короля = славянские войны
- **Параллель**: русская легенда о Подземном Короле (в Пскове, Вологде)

**3. КРБАБ (Krbab) — лужицкое чудище:
- **Krbab** = славянское чудище, обитает в воде
- **Krbab** = водяной / болотный дух
- **Krbab** = пожирает людей, особенно детей
- **Krbab** = аналог русского Водяного, украинского Водяника
- **Krbab** = «Krbabische Sagen» (Х. Кравчик 1979)

**4. ВИТКА (Witka) — дева-воительница:
- **Witka** (Витка) = легендарная лужицкая героиня
- **Witka** = дева-воительница, защищавшая лужичан от немцев
- **Witka** = аналог славянских легенд о Валькирии
- **Witka** = может быть, отождествлена с Witiza (Witiges), королём готов
- **Witka** = «Witka-Sagen» (Х. Кравчик 1979)

**5. СЕРБСКИЙ РАЙ (Serbske Nebo / Serbski Raj):
- **Serbski Raj** = Сербский Рай = лужицкий рай
- **Легенда**: На горе Копани (Kopany) есть врата в Сербский Рай
- **Легенда**: Там живут лужичане, ушедшие из мира живых
- **Легенда**: В Сербском Рае — вечная весна, молочные реки, кисельные берега
- **Легенда**: Славянский параллель: Вырий (рус.), Ирий (укр.), Nav (полабск.)

### IV. ПЕСНИ ЛУЖИЧАН

**1. КОЛЯДКИ (Kólesnicy / Wanochowne pěsnje):
- **Коляда** (Kólesnicy) = рождественские песни
- **Коляда** = 25 декабря (по католическому календарю)
- **Коляда** = 7 января (по православному календарю)
- **Лужицкая колядка**: «Hdyž so narodźi Spasowc / na Betlehemskim polu» (Когда родился Спаситель / на Вифлеемском поле)
- **Лужицкая колядка**: «Přinošamy wam nowiny» (Приносим вам новости)
- **Лужицкая колядка**: «Z jasnej hwjezdy» (С ясной звезды)

**2. КУПАЛЬСКИЕ (Kupalne pěsnje / Swjatki):
- **Купала** (Kupala) = летний праздник (7 июля)
- **Купала** = Иван Купала (по православному календарю)
- **Лужицкая купальская**: «Kupajce, kupajce!» (Купайтесь, купайтесь!)
- **Лужицкая купальская**: «Na Kupale» (На Купалу)
- **Лужицкая купальская**: «Swjataja nóc» (Святая ночь)

**3. СВАДЕБНЫЕ (Swajźbne pěsnje):
- **Swajźba** = свадьба
- **Свадебные песни** = 30+ текстов (П. Нойгебауэр 1955)
- **Свадебный пир** = «Serbska swajźba» (Sorbische Hochzeit)
- **Свадебный обряд** = «swajźbny rjad» (свадебный чин)

**4. ПОГРЕБАЛЬНЫЕ (Pogrjebne pěsnje):
- **Pogrjeb** = похороны
- **Погребальные песни** = плачи, причитания
- **«Žalobne pěsnje»** (Жалобные песни)
- **Параллель**: русские «причитания», украинские «голосіння»

**5. ПЕСНИ ЯРИЛО (Jarilske pěsnje):
- **Jaril** = Ярило = бог весны
- **Песни Ярило** = весенние песни
- **«Jarilske pěsnje»** (Песни Ярилы) = 15+ текстов

- **Легенда**: В Раю — молочные реки, кисельные берега

- **Легенда**: В Раю — праотцы лужичан

**2. ЛЕГЕНДА О ВЫРИЕ (Vyrij):

**3. ЛЕГЕНДА О НАВИ (Nav):
- **Nav** = «Навь» = подземный мир
- **Nav** = «мёртвые» (навьи) = «мертвецы»
- **Nav** = славянский подземный мир
- **Nav** = «Navsja» (Навья) = «Навский» = «подземный»

### XI. DAMNATA

1. «Полабский фольклор = немецкий» — DAMNATA. Полабский = славянский
2. «Лужичане = поляки» — DAMNATA. Лужичане = ЗАПАДНЫЕ славяне
3. «Крбаб = немецкое чудище» — DAMNATA. Крбаб = славянское
4. «Krabat = немецкая легенда» — DAMNATA. Krabat = лужицкая (Krabat = Krawat)
5. «Свадебные обряды = немецкие» — DAMNATA. Свадебные = славянские
6. «Двоеверие = синкретизм» — DAMNATA. Двоеверие = особый тип религии
7. «Лужицкий фольклор = исчез» — DAMNATA. Фольклор СОХРАНЯЕТСЯ до XXI в.

### XII. ВЕРДИКТ

**POLABIA FOLKLOR** — **«Полабский фольклор = живое наследие вымершего мира»**:

**Главный аргумент**: «Полабский фольклор — устное наследие полабского мира, СОХРАНИВШЕЕСЯ через лужичан. Сказки (Krabat, брат и сестра, девушка-змея), легенды (о Крбабе, Подземном Короле, Соляном замке, Сербском Рае), песни (колядки, купальские, свадебные, погребальные), заговоры (лечебные, хозяйственные, любовные, защитные), эпос (Mikławs z Kórnja). Собрано в XIX-XX вв. (Барт 1862, Шульц 1880, Гаупт 1902, Нойгебауэр 1955, Кравчик 1979, Ланге 1990). Сохраняется до XXI в. через лужицкие музеи, ансамбли, радио, газеты. Полабский фольклор = живое наследие вымершего мира».
