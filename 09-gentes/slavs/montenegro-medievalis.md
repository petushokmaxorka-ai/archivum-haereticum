### I. СВОДКА

**Монтенегрини** **медиевальны** = **«**с**р**е**д**н**е**в**е**к**о**в**а**я** **Ч**е**р**н**о**г**о**р**и**я**»**:

**Время**: **X**-**X**V** **вв.** (**~**500** **л**е**т**)
**Место**: **Д**у**к**л**ь**а** → **З**е**т**а** → **Ч**е**р**н**а** **Г**о**р**а**

**Т**р**и** **п**е**р**и**о**д**а**:
- **Д**у**к**л**ь**а** (**X**-**X**I**I** **вв.**) — **п**е**р**в**о**е** **ю**ж**н**о**с**л**а**в**я**н**с**к**о**е** **г**о**с**у**д**а**р**с**т**в**о** (**д**и**н**. **В**о**и**с**л**а**в**л**и**ч**е**й**)
- **З**е**т**а** (**X**I**I**-**X**I**V** **вв.**) — **п**о**д** **в**л**а**с**т**ь**ю** **Н**е**м**а**н**и**ч**е**й**, **з**а**т**е**м** **Б**а**л**ш**и**ч**е**й**
- **З**е**т**а**-**Ц**р**н**о**е**в**и**ч**и** (**X**I**V**-**X**V** **вв.**) — **п**о**с**л**е**д**н**я**я** **н**е**з**а**в**и**с**и**м**а**я** **д**и**н**а**с**т**и**я**

**Источники**:
- **«**Л**е**т**о**п**и**с**е**ц** **П**о**п**а** **Д**у**к**л**я**н**и**н**а**»** (**B**a**r** **R**o**m**a**n**i**u**m** **G**r**e**m**i**a** **d**e** **D**u**c**l**i**a**)** — **X**I**I** **в.** (**г**л**а**в**н**ы**й**)
- **«**Л**е**т**о**п**и**с**е**ц** **П**о**п**а** **Д**у**к**л**я**н**и**н**а**»** **Б**а**р**и**з**и**н**и** (**V**a**r**i**n**g**b**e**r**g**) — **X**I**I** **в.**
- **К**о**н**с**т**а**н**т**и**н** **V**I**I** *«**D**e** **A**d**m**i**n**i**s**t**r**a**n**d**o** **I**m**p**e**r**i**o**»** (**X** **в.**)
- **А**н**н**а** **К**о**м**н**и**н**а** *«**А**л**е**к**с**и**а**д**а**»** (**X**I**-**X**I**I** **вв.**) — *«**Д**у**к**л**ь**а** **п**о**к**о**р**е**н**а** **Н**е**м**а**н**е**й**»**
- **С**т**е**ф**а**н** **Н**е**м**а**н**и**ч** *«**Р**о**д**о**с**л**о**в**»** (**X**I**I**I** **в.**)
- **Д**о**м**е**н**т**и**а**н** *«**Ж**и**т**и**е** **С**т**е**ф**а**н**а** **Н**е**м**а**н**и**»** (**X**I**I**I** **в.**)
- **С**ф**р**а**н**д**з**и** *«**Х**р**о**н**и**к**а**»** (**X**I**V**-**X**V** **вв.**)
- **М**а**в**р**о** **О**р**б**и**н**и** *«**И**л**л**и**р**с**к**о**е** **ц**а**р**с**т**в**о**»** (**1**6**0**1**)
- **Н**е**х**а**й** *«**И**с**т**о**р**и**я**»** (**X**V** **в.**)
- **П**а**п**с**к**и**е** **п**и**с**ь**м**а** **Г**р**и**г**о**р**и**я** **V**I**I** (**1**0**7**7**), **И**н**н**о**к**е**н**т**и**я** **I**I**I** (**1**1**9**8**)

**С**т**р**а**т**е**г**и**я** **RESISTENTIA + NODUS + REPLICATIO**:
- **RESISTENTIA**: **Д**у**к**л**ь**а**/**З**е**т**а** = **н**е**з**а**в**и**с**и**м**о**с**т**ь** **о**т** **В**и**з**а**н**т**и**и** + **о**с**м**а**н**
- **NODUS**: **М**о**н**т**е**н**е**г**р**и**н**о** = **у**з**е**л** **и**л**л**и**р**и**й**с**к**о**г**о** + **с**л**а**в**я**н**с**к**о**г**о** + **в**и**з**а**н**т**и**й**с**к**о**г**о** + **к**а**т**о**л**и**ч**е**с**к**о**г**о** + **и**с**л**а**м**с**к**о**г**о**
- **REPLICATIO**: **с**л**а**в**я**н**е** **«**к**о**п**и**р**у**ю**т**»** **в**и**з**а**н**т**и**й**с**к**у**ю** **ц**а**р**с**к**у**ю** **и**д**е**о**л**о**г**и**ю**

**Г**л**а**в**н**ы**е** **т**е**м**ы**:
- **Д**у**к**л**ь**а** **В**о**и**с**л**а**в**л**и**ч**е**й** (**X**-**X**I**I** **вв.**) — **П**е**т**р**и**л**, **С**т**е**ф**а**н** **В**о**и**с**л**а**в**, **К**о**н**с**т**а**н**т**и**н** **Б**о**д**и**н**
- **З**е**т**а** **Б**а**л**ш**и**ч**е**й** (**X**I**V**-**X**V** **вв.**) — **Б**а**л**ш**а** **I**, **Д**ж**у**р**а**дж** **I** **С**т**р**а**ц**и**м**и**р**о**в**и**ч**
- **З**е**т**а** **Ц**р**н**о**е**в**и**ч**е**й** (**X**V**-**X**V**I** **вв.**) — **И**в**а**н** **Ц**р**н**о**е**в**и**ч** (**п**е**р**в**ы**й** **т**и**п**о**г**р**а**ф** **н**а** **Б**а**л**к**а**н**а**х**)
- **К**а**т**у**н**ы** — **п**а**с**т**у**ш**ь**и** **о**б**щ**и**н**ы** **С**т**а**р**о**й** **Ч**е**р**н**о**г**о**р**и**и**
- **Б**и**т**в**а** **н**а** **К**о**с**о**в**о** (**1**3**8**9**) + **З**е**т**с**к**а**я** **д**о**л**и**н**а**
- **Д**в**о**е**в**е**р**и**е**:** **В**е**л**е**с**/**с**в**. **В**л**а**с**и**й** + **М**о**к**о**ш**ь**/**с**в**. **П**а**р**а**с**к**е**в**а**

### II. ДИНАСТИЯ ВОИСЛАВЛЬЕВИЧЕЙ (X-XII вв.)

**1. ПЕТРИЛ (ок. 940-960) — «отец» Дукли:
- **«**П**е**т**р**и**л**»** = **«**П**е**т**р**и**л** = **П**е**т**р**ь**»** (**с**л**а**в**я**н**с**к**о**е**)
- **П**е**р**в**ы**й** **и**с**т**о**р**и**ч**е**с**к**и**й** **к**н**я**з**ь** **Д**у**к**л**ь**а** (п**о** **Л**е**т**о**п**и**с**ц**у**)

- **В**а**с**с**а**л** **В**и**з**а**н**т**и**и**
- **С**в**о**й** Д**у**к**л**ь**а**н**с**к**а**я** **ц**е**р**к**о**в**ь** (п**о**д** **К**о**н**с**т**а**н**т**и**н**о**п**о**л**е**м**)

**2. СТЕФАН ВОИСЛАВ (ок. 1040-1050) — «освободитель»:
- **«**С**т**е**ф**а**н** **В**о**и**с**л**а**в**»** = **«**с**т**е**ф**а**н** + **в**о**и**с**л**а**в** = **в**о**и**н** + **с**л**а**в**а**»**
- **В**о**с**с**т**а**н**и**е** **п**р**о**т**и**в** **В**и**з**а**н**т**и**и** (**1**0**4**0**-**1**0**4**1**)** — *«**А**н**н**а** **К**о**м**н**и**н**а**»* (**«**в**с**е** **с**л**а**в**я**н**е** **п**о**д**н**я**л**и**с**ь**»**)
- **С**о**ю**з** **с** **П**е**т**р**о**м** **Д**е**л**ь**я**н**о**м** (**С**е**р**б**и**я**)** — *«**с**л**а**в**я**н**с**к**а**я** **к**о**н**ф**е**д**е**р**а**ц**и**я**»*
- **1**0**4**0** — **о**с**в**о**б**о**ж**д**е**н**и**е** **С**к**о**п**ь**е**, **О**х**р**и**д**а**, **С**е**р**д**и**к**и**
- **1**0**4**1** — **р**а**з**г**р**о**м** **в**и**з**а**н**т**и**й**ц**а**м**и**; **С**т**е**ф**а**н** **п**л**е**н**ё**н**, **о**с**л**е**п**л**ё**н**
- **П**о**с**л**е** **п**л**е**н**а** — **с**н**о**в**а** **к**н**я**з**ь** (**п**о**с**т**р**и**г** **г**л**а**з** — **ч**а**с**т**и**ч**н**о**)

**3. МИХАИЛО (ок. 1050-1081) — «царь Дукли»:
- **«**М**и**х**а**и**л**о**»** = **«**М**и**х**а**и**л** = **к**т**о** **к**а**к** **Б**о**г**»** (**д**р**е**в**н**е**е**в**р**е**й**с**к**о**е**)
- **С**ы**н** **С**т**е**ф**а**н**а** **В**о**и**с**л**а**в**а**
- **С**о**ю**з** **с** **п**а**п**с**т**в**о**м** (**п**р**о**т**и**в** **В**и**з**а**н**т**и**и**)** — *«**п**о**л**и**т**и**ч**е**с**к**о**е** **р**а**з**д**в**о**е**н**и**е**»*
- **1**0**7**7** — **п**а**п**а** **Г**р**и**г**о**р**и**й** **V**I**I** **п**р**и**з**н**а**ё**т** **к**н**я**з**я** **Д**у**к**л**ь**а** (р**е**к**о**г**н**и**ц**и**я** **с**у**в**е**р**е**н**и**т**е**т**а**)
- **1**0**7**8** — **в**о**й**с**к**о** **Д**у**к**л**ь**а** **н**а** **п**о**м**о**щ**ь** **Н**о**р**м**а**н**н**а**м** (**Р**о**б**е**р**т** **Г**в**и**с**к**а**р**)

**4. КОНСТАНТИН БОДИН (ок. 1081-1101) — «царь славян»:
- **«**К**о**н**с**т**а**н**т**и**н** **Б**о**д**и**н**»** = **«**К**о**н**с**т**а**н**т**и**н** + **Б**о**д**и**н** = **б**о**д** = **б**о**д**р**с**т**в**о**в**а**т**ь**»**
- **В**н**у**к** **С**т**е**ф**а**н**а** **В**о**и**с**л**а**в**а**
- **1**0**8**1** — **о**б**ъ**е**д**и**н**и**т**е**л**ь**н**ы**й** **с**о**б**о**р** **Д**у**к**л**ь**а** + **С**к**у**т**а**р**и**
- **1**0**8**2** — **о**с**в**о**б**о**ж**д**е**н**и**е** **о**т** **в**и**з**а**н**т**и**й**с**к**о**г**о** **и**г**а**
- **1**0**8**9** — **п**о**х**о**д** **п**р**о**т**и**в** **А**л**ь**п**о**в** (**п**о**м**о**щ**ь** **в**и**з**а**н**т**и**й**ц**а**м** **п**р**о**т**и**в** **П**е**ч**е**н**е**г**о**в**)
- **1**0**9**7** — **у**ч**а**с**т**и**е** **в** **П**е**р**в**о**м** **К**р**е**с**т**о**в**о**м** **п**о**х**о**д**е** (Б**о**д**и**н** — **«**ц**а**р**ь** **С**л**а**в**и**н**и**и**»**)
- **1**1**0**1** — **с**м**е**р**т**ь** **Б**о**д**и**н**а**; **Д**у**к**л**ь**а** **с**н**о**в**а** **к** **В**и**з**а**н**т**и**и**

**5. ДОБРОСЛАВ II (ок. 1101-1118):
- **«**Д**о**б**р**о**с**л**а**в**»** = **«**д**о**б**р**о** + **с**л**а**в**а**»**
- **1**1**0**1**-**1**1**1**8** — **к**н**я**з**ь** **Д**у**к**л**ь**а** **п**о**д** **в**и**з**а**н**т**и**й**с**к**и**м** **с**ю**з**е**р**е**н**и**т**е**т**о**м**

**6. МИХАИЛО II (1118-1130) + ГРУЕША/БОГДАН (1130-1146):
