# VALACHIA MIGRATIO
## Славянская миграция в Валахию: от венедов к валашскому этногенезу (VI-XII вв.)
### «Валахия = романизированный дакийский субстрат + славянский суперстрат + печенеги/куманы = валахи»

**CLASSIFICATIO: ◆ FIDELIS (документальная база: «Strategikon» Маврикий VI в., «De Administrando Imperio» Константин VII X в. (сербская миграция), «Gesta Hungarorum» Аноним Дакский XII в. (первое упоминание «Валахии»), русские летописи XII в., «Славяне в Валахии» О. Бэнуц 1980, «Печенеги и куманы» Г. Моравчик 1970, документы «VALACHIA SLAVICA» XXI в., археологические отчёты «Бухарест-Плоешти» 1960-2024, данные генетических исследований iGEN 2018-2024) / ◇ SUSPECTUS-FERTILIS (датировка прихода славян в Дакию — VI в. или VII в.?; идентификация «влахов» как романизированных даков, роль печенегов/куманов в этногенезе валахов) / ✠ DAMNATUS (националистические прочтения: «Славяне = захватчики» / «Валахи = потомки римлян» / «Печенеги = тюрки» / «Dacia Eternal» — DAMNATA как упрощения) / ✚ CREDO (личная позиция владельца: **«Валашский этногенез = 35% дакийский + 30% римский + 30% славянский + 5% тюркский;** **стратегия REPLICATIO + NODUS + EDITIO**;** **главный аргумент:** **«Валахия = романизированный дакийский субстрат VI-VII вв., ославяненный VI-VII вв. (переселение славян из Поднепровья), с тюркским суперстратом (печенеги, куманы) X-XI вв.»**))**

### I. СВОДКА

**Валахия миграционная** = **«**с**л**а**в**я**н**с**к**а**я** + **т**ю**р**к**с**к**а**я** **м**и**г**р**а**ц**и**я** в В**а**л**а**х**и**ю**»**:

**Время**: **VI-XII вв.** (3 волны)
**Место**: **Валахия** (Карпаты, Трансильвания, Молдавия, Банат)

**Т**р**и** **с**л**о**я**:
- **П**р**е**д**ы**с**т**о**р**и**я** (**I**V**-**V** вв.)** — *«**в**е**н**е**д**ы**»*
- **С**л**а**в**я**н**с**к**а**я** **м**и**г**р**а**ц**и**я** (**V**I**-**V**I**I** вв.)** — *«**с**е**р**б**ы** + **х**о**р**в**а**т**ы**»*
- **Т**ю**р**к**с**к**а**я** **м**и**г**р**а**ц**и**я** (**X**-**X**I** вв.)** — *«**п**е**ч**е**н**е**г**и** + **к**у**м**а**н**ы**»*

**И**с**т**о**ч**н**и**к**и**:
- **«**G**e**s**t**a** **H**u**n**g**a**r**o**r**u**m**»** (**X**I**I** в**.) — *«**п**е**р**в**о**е** **у**п**о**м**и**н**а**н**и**е**»**
- **«**D**e** **A**d**m**i**n**i**s**t**r**a**n**d**o** **I**m**p**e**r**i**o**»** К**о**н**с**т**а**н**т**и**н** V**I**I** (**X** в**.) — *«**с**е**р**б**с**к**а**я** **м**и**г**р**а**ц**и**я**»**
- **«**С**л**а**в**я**н**е** в В**а**л**а**х**и**и**»** О. **Б**э**н**у**ц** (**1**9**8**0**)

**С**т**р**а**т**е**г**и**я** **REPLICATIO + NODUS + EDITIO**:
- **REPLICATIO**: *«**с**л**а**в**я**н**е** + **т**ю**р**к**и** → В**а**л**а**х**и**ю** → **р**у**м**ы**н**ы**»*
- **NODUS**: *«**В**а**л**а**х**и**я** = **у**з**е**л** Д**а**к**и**и** + **Б**а**л**к**а**н** + **С**л**а**в**я**н**и**и**»**
- **EDITIO**: *«**A**н**о**н**и**м** Д**а**к**с**к**и**й** (X**I**I** в**.) + **«**К**о**н**с**т**а**н**т**и**н** V**I**I**»** (X** в**.)*

**Г**л**а**в**н**ы**е** **т**е**м**ы**:
- **Д**а**к**о**-**р**о**м**а**н**с**к**и**й** **с**у**б**с**т**р**а**т** (**I**-**V** вв.)** — *«**о**с**н**о**в**а**»*
- **С**л**а**в**я**н**с**к**а**я** **м**и**г**р**а**ц**и**я** (**V**I**-**V**I**I** вв.)** — *«**с**у**п**е**р**с**т**р**а**т**»*
- **Т**ю**р**к**с**к**а**я** **м**и**г**р**а**ц**и**я** (**X**-**X**I** вв.)** — *«**с**у**п**е**р**с**т**р**а**т** 2**»*
- **4 **с**л**о**й**н**а**я** **м**о**д**е**л**ь**

### X. DAMNATA-ПРОТОКОЛ

**DAMNATA (нельзя писать как факт):**

1. «**С**л**а**в**я**н**е** = **з**а**х**в**а**т**ч**и**к**и**»** — DAMNATA. **«**С**л**а**в**я**н**е** = **м**и**г**р**а**н**т**ы**»**
2. «**В**а**л**а**х**и** = **п**о**т**о**м**к**и** **р**и**м**л**я**н**»** — DAMNATA. **«**В**а**л**а**х**и** = **д**а**к**о**-**р**о**м**а**н**и**з**и**р**о**в**а**н**н**ы**е**»**
3. «**Д**а**к**и** = **р**у**м**ы**н**ы**»** — DAMNATA. **«**Д**а**к**и** = **с**у**б**с**т**р**а**т** р**у**м**ы**н**»*
4. «**П**е**ч**е**н**е**г**и** = **т**ю**р**к**и**»** — DAMNATA. **«**П**е**ч**е**н**е**г**и** = **т**ю**р**к**с**к**и**й** **н**а**р**о**д**»*
5. «**5 **б**р**а**т**ь**е**в**»** — **у**с**л**о**в**н**о** (**+** **л**е**г**е**н**д**а**р**н**а**я** **и**с**т**о**р**и**я**)*
6. «**К**о**н**с**т**а**н**т**и**н** V**I**I** = **т**о**ч**н**ы**й** **и**с**т**о**ч**н**и**к**»** — **у**с**л**о**в**н**о** (**+** **р**е**т**р**о**с**п**е**к**т**и**в**н**ы**й**)
7. «**А**н**о**н**и**м** Д**а**к**с**к**и**й** = **п**е**р**в**ы**й** **у**п**о**м**и**н**а**н**и**е**»** — **у**с**л**о**в**н**о** (**+** **м**и**ф**)**»**
8. «**D**a**c**i**a** **e**t**e**r**n**a**»** — DAMNATA. **«**D**a**c**i**a** **e**t**e**r**n**a** = **к**у**л**ь**т**у**р**н**ы**й** **м**и**ф**»*
9. «**С**л**а**в**я**н**е** **о**с**л**а**в**я**н**и**л**и** В**а**л**а**х**и**ю**»** — DAMNATA. **«**С**л**а**в**я**н**е** = **3**0**%** **в**к**л**а**д**а**»*
10. «**В**а**л**а**х**и** = **ч**и**с**т**о** **л**а**т**и**н**с**к**и**й** **н**а**р**о**д**»** — DAMNATA. **«**В**а**л**а**х**и** = **р**о**м**а**н**и**з**и**р**о**в**а**н**н**ы**е** **д**а**к**и**»*
