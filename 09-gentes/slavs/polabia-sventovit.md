### I. СВОДКА

**Свентовит (Sventovit / Svantevit / Святовит / Sventovith / Звентовит)** — верховный бог полабского мира (V-XII вв.н.э.). Почитался во всех полабских племенах (лютичей, древлян, бодричей, поморян, руян, гаволян, стодоран, лужичан). Описан Саксоном Грамматиком (XIII в.н.э.) как «бог всех славян». Главные святилища: **Аркона** (на Рюгене) и **Ретра** (в Мекленбурге). Разрушен 16 мая 1168 г.н.э. в Арконе датским королём **Вальдемаром I**.

**Главные тезисы**:
1. **Этимология**: «Свентовит» = «Свент-» (Святой) + «-вит» (Свет) = «Святой Свет»
2. **Иконография**: 4-головая статуя, 3 м высотой, с 4 мечами
3. **Священные предметы**: белый конь, рог, 4 меча
4. **Святилища**: Аркона (главное), Ретра, Гавельберг, Вольгаст
5. **Саксон Грамматик**: «Свентовит = бог всех славян»
6. **Двоеверие**: Свентовит = Св. Витий (христианский святой)

### II. ЭТИМОЛОГИЯ

**«Свентовит» (Sventovit / Svantevit / Святовит)** = **«Святой Свет»**

- **«Свентовит»** = **«Свент-» (святой)** + **«-вит» (свет)** — *«Святой Свет»*
- **«Sventovit»** = **«Svent-» (sventъ = святой)** + **«-ovit» (-ovitъ = свет)** — *«Святой Свет»*
- **«Svantevit»** = **«Svant-» (svantъ = святой)** + **«-evit» (-evitъ = свет)** — *«Святой Свет»*
- **«Святовит»** = **«Свят-» (святой)** + **«-овит» (свет)** — *«Святой Свет»*
- **«Zvantevit»** = **«Zvant-» (святой)** + **«-evit» (свет)** — *«Святой Свет»*

**Сравнить**:
- **Свентовит** = «Святой Свет»
- **Даждьбог** = «Дажь-» (дай) + «-бог» (бог) = «Дающий Бог»
- **Хорс** = «Хорс-» (солнце) = «Солнце»
- **Сварог** = «Свар-» (сварка, кузнец) = «Кузнец»

**В. В. Седов (1954)**: «Свентовит = Святой Свет = верховный бог полабского мира».
**Й. Херрман (1990)**: «Свентовит = Свет + Святой = бог света и добра».
**Г. Траутманн (1949)**: «Свентовит = Sventovit = от svęt- (святой) + vit- (свет)».

### IV. СВЯЩЕННЫЕ ПРЕДМЕТЫ СВЕНТОВИТА

**1. 4-ГОЛОВАЯ СТАТУЯ:
- **Статуя Свентовита** = деревянная, 3 м высотой
- **4 головы** = смотрят на 4 стороны света (Север, Юг, Запад, Восток)
- **4 бороды** = у каждой головы
- **4 венца** = у каждой головы
- **4 меча** = в руках (по 2 в каждой руке)
- **Священный рог** = в руках (или рядом)
- **Волчья шкура** = у ног (символ)

**2. БЕЛЫЙ КОНЬ (СВЯЩЕННЫЙ):
- **Белый конь** = священный, для гадания
- **Гадание** = жрец ведёт коня через 2 копья
- **Первый шаг** = направо = «к своей земле» (успех)
- **Первый шаг** = налево = «к чужой земле» (поражение)
- **Конь = символ** = победы, плодородия, Солнца

**3. СВЯЩЕННЫЙ РОГ:
- **Рог Свентовита** = рог быка, украшенный
- **Вместимость** = «один человек пьёт» (большой)
- **Использование** = только жрец пьёт
- **После обряда** = рог выливается на землю
- **Символ** = плодородия, изобилия

**4. 4 СВЯЩЕННЫХ ВОИНА (СТРАЖИ):
- **4 стража** = священные воины-витязи
- **4 стража** = живут в капище
- **4 стража** = погибают в бою
- **4 стража** = символ воинской доблести

**5. 4 МЕЧА (В РУКАХ СТАТУИ):
- **4 меча** = в 4 руках статуи
- **2 меча** = в передних руках
- **2 меча** = в задних руках
- **Мечи** = символ войны и защиты

6. «Свентовит = Тор» — DAMNATA. Свентовит = независимый бог (хотя индоевропейский общий)
7. «4 головы = суеверие» — DAMNATA. 4 головы = символ 4 сторон света

---

### XIII. ВЕРДИКТ

**POLABIA SVENTOVIT** — **«Свентовит = Святой Свет = верховный бог полабского мира»**:

**Главный аргумент**: «Свентовит (Sventovit) = верховный бог полабского мира (V-XII вв.н.э.). Почитался во всех полабских племенах (лютичей, древлян, бодричей, поморян, руян, гаволян, стодоран, лужичан). Описан Саксоном Грамматиком (XIII в.н.э.) как «бог всех славян». 4-головая статуя (3 м) с 4 мечами. Белый конь для гадания. Священный рог. Главные святилища: Аркона (главное, Рюген) и Ретра (второе, Мекленбург). Двоеверие: Свентовит = Св. Витий (христианский святой). Разрушен 16 мая 1168 г.н.э. в Арконе датским королём Вальдемаром I, 1180 г.н.э. в Ретре Саксонским герцогом Бернардом. Свентовит = индоевропейский верховный бог-громовержец (как Зевс, Юпитер, Тор, Перун, Индра) с особенностью 4 голов. Этимология: Свентовит = Святосвет = Святой Свет».
