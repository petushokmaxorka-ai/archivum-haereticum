### I. СВОДКА

**Двоеверие (Dvoeverie)** — особый тип религии у полабских славян (и всех восточных/западных славян в X-XVIII вв. н.э.), при котором христианские и языческие верования СОСУЩЕСТВУЮТ, не сливаясь. Главные отождествления:
- **Свентовит = Св. Витий** (Sankt Veit, 15 июня)
- **Яровит = Св. Георгий** (Sankt Georg, 23 апреля)
- **Триглав = Три царя** (Heilige Drei Könige, 6 января)
- **Сива/Жива = Дева Мария** (Sankta Maria)
- **Чернобог = Св. Николай** (Sankt Nikolaus, 6 декабря)
- **Радегаст = Св. Маврикий** (Sankt Mauritius)
- **Припегала = Параскева Пятница** (Sankta Paraskewa)
- **Поревит = Св. Пантелеймон** (Sankt Panteleimon)

**Главные тезисы**:
1. **Двоеверие ≠ синкретизм** — это особый тип религии (сосуществование)
2. **8 богов = 8 христианских святых** (полное отождествление)
3. **Языческие праздники = христианские праздники** (наложение)
4. **Языческие обряды = христианские обряды** (смешение)
5. **Двоеверие сохранялось до XVIII-XXI вв.** (особенно в Шпреевальде)

### II. ОПРЕДЕЛЕНИЕ ДВОЕВЕРИЯ

**1. ЧТО ТАКОЕ ДВОЕВЕРИЕ:
- **Двоеверие** = Dvoeverie (от «две веры»)
- **Двоеверие** = сочетание христианства и язычества
- **Двоеверие** = особый тип религии, НЕ синкретизм
- **Двоеверие** = христианский и языческий пласты СОСУЩЕСТВУЮТ
- **Двоеверие** = типично для всех славян X-XVIII вв. н.э.
- **Двоеверие** = описано в «Стоглаве» (1551) как «смешение вер»

**2. ДВОЕВЕРИЕ ≠ СИНКРЕТИЗМ:
- **Синкретизм** = слияние двух религий в ОДНУ новую религию
- **Двоеверие** = сосуществование двух религий, каждая со своими обрядами
- **Синкретизм** = возникает в крупных городах (например, Антиохия)
- **Двоеверие** = характерно для деревень (полабский мир — деревни)
- **Синкретизм** = «новый бог» (например, Христос-Будда)
- **Двоеверие** = «два бога» (Свентовит = Св. Витий, НЕ СЛИЯНИЕ, а ОТОЖДЕСТВЛЕНИЕ)

**3. ДВОЕВЕРИЕ У ЛУЖИЧАН:
- **Двоеверие** = особый тип религии лужичан
- **Двоеверие** = сохранялось до XVIII-XXI вв. (Шпреевальд)
- **Двоеверие** = «Лужичане — дважды верующие»
- **Двоеверие** = «Лужичане — в церкви, и в лесу»

### III. 8 БОГОВ = 8 СВЯТЫХ (полное отождествление)

**1. СВЕНТОВИТ = СВ. ВИТИЙ (SANKT VEIT):
- **Свентовит** (Sventovit) = «Святой Свет» (верховный бог)
- **Св. Витий** (Sankt Veit) = раннехристианский мученик, святой
- **День Св. Вития**: 15 июня
- **Отождествление**: Свентовит = Св. Витий
- **Место**: остров Рюген → церковь Св. Вития в Арконе (после 1168)
- **Сохранение**: праздник Св. Вития в Лужице до XXI в.

**2. ЯРОВИТ = СВ. ГЕОРГИЙ (SANKT GEORG):
- **Яровит** (Jarovit) = «Яростный Свет» (бог войны и весны)
- **Св. Георгий** (Sankt Georg) = победитель змея (змееборец)
- **День Св. Георгия**: 23 апреля
- **Отождествление**: Яровит = Св. Георгий
- **Место**: Вольгаст → церковь Св. Георгия в Вольгасте (после 1160)
- **Сохранение**: праздник Св. Георгия в Лужице = Ярило

**3. ТРИГЛАВ = ТРИ ЦАРЯ (HEILIGE DREI KÖNIGE):
- **Триглав** (Triglav) = «Три Главы» (трёхглавый бог)
- **Три царя** (Caspar, Melchior, Balthasar) = волхвы, принёсшие дары Христу
- **День Трёх царей**: 6 января (Богоявление)
- **Отождествление**: Триглав = Три царя
- **Место**: Щецин и Волин → церкви Трёх царей (после 1128)
- **Сохранение**: праздник Трёх царей в Лужице

**4. СИВА/ЖИВА = ДЕВА МАРИЯ (SANKTA MARIA):
- **Сива/Жива** (Siva) = «Живая» (богиня-мать)
- **Дева Мария** (Sankta Maria) = Богородица
- **День Девы Марии**: 15 августа (Успение)
- **Отождествление**: Сива/Жива = Дева Мария
- **Место**: восточно-славянское → лужицкое
- **Сохранение**: «Sorbische Marienverehrung» (Лужицкое почитание Марии)

**5. ЧЕРНОБОГ = СВ. НИКОЛАЙ (SANKT NIKOLAUS):
- **Чернобог** (Černobog) = «Чёрный Бог» (тёмный бог)
- **Св. Николай** (Sankt Nikolaus) = святой, даритель подарков
- **День Св. Николая**: 6 декабря
- **Отождествление**: Чернобог = Св. Николай (с негативной коннотацией)
- **Место**: Лужица
- **Сохранение**: праздник Св. Николая в Лужице

**6. РАДЕГАСТ = СВ. МАВРИКИЙ (SANKT MAURITIUS):
- **Радегаст** (Radegast) = «Гость Радости» (бог войны и гостеприимства)
- **Св. Маврикий** (Sankt Mauritius) = раннехристианский мученик, святой
- **День Св. Маврикия**: 22 сентября
- **Отождествление**: Радегаст = Св. Маврикий
- **Место**: Бранибор (Берлин) → церковь Св. Маврикия
- **Сохранение**: праздник Св. Маврикия в Лужице

**7. ПРИПЕГАЛА = ПАРАСКЕВА ПЯТНИЦА (SANKTA PARASKEWA):
- **Припегала** (Pripegala) = богиня весны и плодородия
- **Параскева Пятница** (Sankta Paraskewa) = христианская святая
- **День Параскевы**: 14 октября (Покров) / 10 ноября (день Параскевы)
- **Отождествление**: Припегала = Параскева Пятница
- **Место**: восточно-славянское → лужицкое
- **Сохранение**: пятница — особый день (Мокошь = Параскева)

**8. ПОРЕВИТ = СВ. ПАНТЕЛЕЙМОН (SANKT PANTELEIMON):
- **Поревит** (Porevit) = «Полевой Свет» (бог плодородия)
- **Св. Пантелеймон** (Sankt Panteleimon) = целитель, врач
- **День Св. Пантелеймона**: 9 августа
- **Отождествление**: Поревит = Св. Пантелеймон
- **Место**: Шверин → церковь Св. Пантелеймона
- **Сохранение**: праздник Св. Пантелеймона в Лужице

### V. ЯЗЫЧЕСКИЕ ОБРЯДЫ = ХРИСТИАНСКИЕ ОБРЯДЫ

**1. КРЕСТНОЕ ЗНАМЕНИЕ vs КАПИЩНЫЙ КРЕСТ:
- **Крещение** = погружение в воду (двоеверие — очищение)
- **Крещение** = изначально языческий обряд (крещение Святовитом = причащение у Свентовита)
- **Крещение** = «крст» = «крест» (солярный символ — крест в круге)

**2. СВЕЧИ В ЦЕРКВИ vs СВЕЧИ НА КАПИЩЕ:
- **Свечи в церкви** = типичный христианский обряд
- **Свечи на капище** = типичный языческий обряд
- **Свечи = огонь** = символ очищения
- **Свечи = связь** = между мирами (земным и небесным)

**3. ПОМИНОВЕНИЕ УСОПШИХ vs ДЗЯДЫ:
- **Поминовение усопших** = христианский обряд (1-2 ноября)
- **Дзяды** (Dźady) = славянский обряд (поминовение предков)
- **Дзяды = «деды»** = предки
- **Дзяды** = «ходить в гости к предкам» (поминки)

**4. КОЛЯДОВАНИЕ vs КОЛЯДА:
- **Колядование** = обход домов с песнями
- **Колядование** = типичный языческий обряд
- **Колядование** = сохранён в христианстве (рождественские колядки)
- **Колядование** = «кто не колядует — тому чёрт» (двоеверие)

**5. КУПАНИЕ vs КРЕЩЕНИЕ:
- **Купание на Купалу** = типичный языческий обряд
- **Купание в проруби** = типичный христианский обряд (Крещение Господне)
- **Купание = очищение** = символ духовного обновления

### X. DAMNATA

1. «Двоеверие = синкретизм» — DAMNATA. Двоеверие = сосуществование, не слияние
2. «Двоеверие = суеверие» — DAMNATA. Двоеверие = особый тип религии
3. «Двоеверие = смешение вер» — DAMNATA. Двоеверие = параллельное существование
4. «Двоеверие = результат слабости язычества» — DAMNATA. Двоеверие = результат силы обеих религий
5. «Свентовит = Св. Витий = слияние» — DAMNATA. Свентовит = Св. Витий = отождествление
6. «Двоеверие исчезло» — DAMNATA. Двоеверие СОХРАНЯЕТСЯ до XXI в.

### XI. ВЕРДИКТ

**POLABIA DVOEVERIE** — **«Двоеверие = особый тип религии, не синкретизм»**:

**Главный аргумент**: «Двоеверие (Dvoeverie) = особый тип религии у полабских славян, при котором христианские и языческие верования СОСУЩЕСТВУЮТ, не сливаясь. 8 богов = 8 христианских святых (Свентовит = Св. Витий, Яровит = Св. Георгий, Триглав = Три царя, Сива/Жива = Дева Мария, Чернобог = Св. Николай, Радегаст = Св. Маврикий, Припегала = Параскева, Поревит = Св. Пантелеймон). Языческие праздники = христианские праздники (наложение). Капища → церкви. Священные рощи + церкви. Священные источники + купели. Сохранялось до XVIII-XXI вв. в Шпреевальде. Описано в «Стоглаве» (1551), «Просветителе» Иосифа Волоцкого (1499), Саксоне Грамматике (XIII в.), Гельмольде Босау (XII в.)».
