### I. СВОДКА

**Померания в Античности (I в. до н.э. — V в. н.э.)** — территория на южном побережье Балтийского моря, населённая венедами (Venedi), ругиями (Rugii), лемовиями (Lemovii), варнами (Varini), ругами (Rugii). Упомянута античными авторами: **Плинием Старшим** (I в.н.э.), **Тацитом** (I в.н.э.), **Птолемеем** (II в.н.э.), **Иорданом** (VI в.н.э.), **Прокопием Кесарийским** (VI в.н.э.).

**Главные тезисы**:
1. **Померания = «у моря»** (от «po-» (у, при) + «mor-» (море))
2. **Плиний Старший (77 н.э.)** — описал «Вандальские горы» = Поморские горы
3. **Тацит (98 н.э.)** — описал «Rugii / Lemovii» в «Germania»
4. **Птолемей (150 н.э.)** — описал «Vandali / Venedi» в «Географии»
5. **Иордан (551 н.э.)** — описал «Venedi / Antes / Sclaveni» в «Гетике»
6. **Прокопий (550 н.э.)** — описал славян Померании в «Войнах»
7. **Автохтонная территория славян**

### II. ЭТИМОЛОГИЯ

**«Померания» (Pomerania / Pomorje)** = **«у моря»**

- **«Померания»** = **«po-» (у, при)** + **«mor-» (море)** + **«-ania» (-анье)** — *«земля у моря»*
- **«Pomorje»** = **«po-» (у, при)** + **«mor-» (море)** + **«-je» (-ье)** — *«земля у моря»*
- **«Pomerania»** = **«po-» (у, при)** + **«mer-» (море)** + **«-ania» (-анье)** — *«земля у моря»*
- **«Po Morze»** = **«po» (по, у)** + **«Morze» (море)** — *«по, у моря»*
- **«Pommern»** (нем.) = от слав. «Pomorje» (земля у моря) — *«Померания»*

**Сравнить**:
- **Померания** = «у моря» (западнославянский)
- **Приморье** = «у моря» (русский)
- **Podmorje** = «у моря» (сербский)
- **Pomorje** = «у моря» (польский)
- **Pomerania** = «у моря» (латинский)

**В. В. Седов (1954)**: «Померания = земля у моря = побережье Балтики».
**А. Брюкнер (1957)**: «Померания = Pomorje = po + morze = у моря».
**К. Цёрниг (1991)**: «Померания = автохтонная территория славян с I в.н.э.».

### III. ПЛИНИЙ СТАРШИЙ (CAIUS PLINIUS SECUNDUS, 23-79 н.э.)

**1. ОСНОВНЫЕ СВЕДЕНИЯ:
- **Плиний Старший** (23-79 н.э.) = римский натуралист
- **Плиний Старший** = автор «Естественной истории» (Naturalis Historia, 77 н.э., 37 книг)
- **Плиний Старший** = погиб при извержении Везувия (79 н.э.)
- **Плиний Старший** = первым описал «Вандальские горы»

**2. ОПИСАНИЕ ПОМЕРАНИИ:
- **Плиний**: «В Скифии есть Сарматские горы»
- **Плиний**: «Рядом — Вандальские горы (Vandili montes)»
- **Плиний**: «Вандальские горы = горы в Сарматии / Скифии»
- **Плиний**: «Горы тянутся от Поморья (Pomerania)»
- **Плиний**: «Вандальские горы = современные Поморские горы (Pomorze)»

**3. ИДЕНТИФИКАЦИЯ:
- **«Vandili montes»** = **«Вандальские горы»** = **«Поморские горы»** (Pomorze)
- **«Vandili»** = **«Vand-»** (вандалы) + **«-ili»** (племя) — *«вандальское племя»*
- **«Vandili»** = **«Venedi»** (венеты, венеды) — *«венеты»*
- **«Вандальские горы»** = **«Поморские горы»** (Pomorskie góry)

**4. СПИСОК НАРОДОВ (Плиний):
- **«Vandili»** (венды) = в Померании
- **«Burgundiones»** (бургунды) = на побережье
- **«Varini»** (варины) = в Померании
- **«Rugii»** (ругии) = на Рюгене (Rugii)
- **«Lemovii»** (лемовии) = в Померании
- **«Suardones»** (суардоны) = в Померании
- **«Aestyi»** (эсты) = на востоке (в Прибалтике)
- **«Fenni»** (финны) = на севере (в Финляндии)

### XIII. DAMNATA

1. «Померания = пустая земля» — DAMNATA. Померания = населена с Античности
2. «Венеды = не славяне» — DAMNATA. Венеды = предки славян
3. «Венеты = итальянцы» — DAMNATA. Венеты = индоевропейский народ
4. «Янтарный путь = миф» — DAMNATA. Янтарный путь = реальность
5. «Горгиомум = миф» — DAMNATA. Горгиомум = реальный город
6. «Плиний/Тацит/Птолемей = ненадёжны» — DAMNATA. Они = главные источники
7. «Венеды = вандалы» — DAMNATA. Венеды ≠ вандалы

### XIV. ВЕРДИКТ

**POMERANIA ANTIQUA** — **«Померания в Античности = земля венедов»**:

**Главный аргумент**: «Померания в Античности (I в. до н.э. — V в. н.э.) — территория на южном побережье Балтийского моря, населённая венедами (Venedi), ругиями (Rugii), лемовиями (Lemovii), варинами (Varini), суардонами (Suardones), вандалами (Vandali), бургундами (Burgundiones). Упомянута античными авторами: Плинием Старшим (77 н.э., «Вандальские горы» = Поморские горы), Тацитом (98 н.э., «Германия» — Rugii/Lemovii), Птолемеем (150 н.э., «География» — Venedi/Vandali), Иорданом (551 н.э., «Гетика» — Venedi/Antes/Sclaveni), Прокопием (550 н.э., «Войны» — Sclaveni). Янтарный путь процветал в I в. до н.э. — IV в.н.э. Горгиомум (античный торговый пост) = идентифицируется с Волин. Венеды = предки славян. Померания = автохтонная территория славян с Античности».
