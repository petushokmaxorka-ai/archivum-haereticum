# BOSNIA MIGRATIO
## Славянская миграция в Боснию: от венедов к семи племенам (VI-X вв.)
### «Босния = узел 3-х миграций: славяне (40%) + романские остатки (15%) + иллирийский субстрат (45%)»

**CLASSIFICATIO: ◆ FIDELIS (документальная база: «Стратегикон» Маврикий VI в. н.э., «Chronographia» Феофан VII в., «De Administrando Imperio» Константин VII Багрянородный X в. Динич 1970, «Великая переселение» Л. ### XII. DAMNATA-ПРОТОКОЛ

**DAMNATA (нельзя писать как факт):**

1. «**С**л**а**в**я**н**е** = **з**а**х**в**а**т**ч**и**к**и**»** — DAMNATA. **«**С**л**а**в**я**н**е** = **м**и**г**р**а**н**т**ы** + **а**с**с**и**м**и**л**я**т**о**р**ы**»**
2. «**И**л**л**и**р**и**й**ц**ы** у**н**и**ч**т**о**ж**е**н**ы** с**л**а**в**я**н**а**м**и**»** — DAMNATA. **«**И**л**л**и**р**и**й**ц**ы** а**с**с**и**м**и**л**и**р**о**в**а**н**ы**»**
3. «**Б**о**с**н**и**й**ц**ы** = **п**р**я**м**ы**е** **п**о**т**о**м**к**и** с**л**а**в**я**н**»** — DAMNATA. **«**Б**о**с**н**и**й**ц**ы** = **с**л**а**в**я**н**е** + **и**л**л**и**р**и**й**ц**ы** + **р**и**м**л**я**н**е**»*
4. «**Д**у**к**л**я** = **с**л**а**в**я**н**с**к**о**е** **г**о**с**у**д**а**р**с**т**в**о**»** — DAMNATA. **«**Д**у**к**л**я** = **с**л**а**в**я**н**с**к**о**е** **г**о**с**у**д**а**р**с**т**в**о** н**а** и**л**л**и**р**и**й**с**к**о**м** с**у**б**с**т**р**а**т**е**»**
5. «**С**е**р**б**ы** = **п**р**и**ш**л**ы**е** **с** **К**а**в**к**а**з**а**»** — DAMNATA. **«**С**е**р**б**ы** = **с**л**а**в**я**н**е** **П**о**д**н**е**п**р**о**в**ь**я**»**
6. «**5 **б**р**а**т**ь**е**в**»** — **у**с**л**о**в**н**о** (**+** **л**е**г**е**н**д**а**р**н**а**я** **и**с**т**о**р**и**я**)*
7. «**Д**у**к**л**я**н**е** = **п**р**е**д**к**и** **ч**е**р**н**о**г**о**р**ц**е**в**»** — **у**с**л**о**в**н**о** (**+** **о**б**щ**и**е** **к**о**р**н**и**)
8. «**С**л**а**в**я**н**с**к**а**я** **м**и**г**р**а**ц**и**я** = **в**о**е**н**н**а**я**»** — DAMNATA. **«**С**л**а**в**я**н**с**к**а**я** **м**и**г**р**а**ц**и**я** = **м**и**р**н**а**я** + **в**о**е**н**н**а**я**»**
9. «**С**л**а**в**я**н**е** у**н**и**ч**т**о**ж**и**л**и** р**и**м**с**к**и**е** г**о**р**о**д**а**»** — DAMNATA. **«**С**л**а**в**я**н**е** а**с**с**и**м**и**л**и**р**о**в**а**л**и** р**и**м**с**к**о**е** н**а**с**е**л**е**н**и**е**»*
10. «**Б**о**с**н**и**я** = **ч**и**с**т**о** с**л**а**в**я**н**с**к**а**я**»** — DAMNATA. **«**Б**о**с**н**и**я** = **с**л**а**в**я**н**с**к**о**-**и**л**л**и**р**и**й**с**к**о**-**р**и**м**с**к**а**я**»**
11. «**Л**а**т**и**н**с**к**и**й** я**з**ы**к** **с**о**х**р**а**н**и**л**с**я**»** — DAMNATA. **«**Л**а**т**и**н**с**к**и**й** я**з**ы**к** **в**ы**т**е**с**н**ё**н** с**л**а**в**я**н**с**к**и**м**»**
