### I. СВОДКА

**Валахия средневековая** = **«с**р**е**д**н**е**в**е**к**о**в**а**я** В**а**л**а**х**и**я**»**:

**Время**: **XIV-XV вв.** (от Басарабов до Дракулы)
**Место**: **Валахия** (Бухарест, Тырговиште, Арджеш, Куртя-де-Арджеш, Джурджу)

**П**я**т**ь** **п**е**р**и**о**д**о**в**:
- **О**с**н**о**в**а**н**и**е** (**1**3**1**0**-**1**3**5**2**)** — *«**Б**а**с**а**р**а**б** I**»*

- **К**р**и**з**и**с** (**1**4**1**8**-**1**4**5**6**)** — *«**Д**а**н** II**»*
- **Д**р**а**к**у**л**а** (**1**4**5**6**-**1**4**6**2**)** — *«**Д**р**а**к**у**л**а**»*
- **П**а**д**е**н**и**е** (**1**4**6**2**-**1**4**9**5**)** — *«**т**у**р**к**и**»*

**И**с**т**о**ч**н**и**к**и**:
- **«**G**e**s**t**a** **H**u**n**g**a**r**o**r**u**m**»** (**X**I**I** в**.) — *«**п**е**р**в**о**е** **у**п**о**м**и**н**а**н**и**е**»**
- **«**В**а**л**а**ш**с**к**и**е** **х**р**о**н**и**к**и**»** (**X**I**V**-**X**V** вв**.) — *«**г**л**а**в**н**ы**е**»**
- **«**О**п**и**с**а**н**и**е** **М**о**л**д**а**в**и**и**»** Д. **К**а**н**т**е**м**и**р** (**1**7**1**6**) — *«**и**с**т**о**ч**н**и**к**»*

**С**т**р**а**т**е**г**и**я** **RESISTENTIA + NODUS + EDITIO**:
- **RESISTENTIA**: *«**В**а**л**а**х**и**я** **с**о**п**р**о**т**и**в**л**я**е**т**с**я** **т**у**р**к**а**м**»**

- **EDITIO**: *«**В**а**л**а**ш**с**к**и**е** **х**р**о**н**и**к**и**»** + **«**Д**е**к**а**н**е**с**т**в**и**е** **Д**р**а**к**у**л**ы**»**

**Г**л**а**в**н**ы**е** **т**е**м**ы**:
- **Д**и**н**а**с**т**и**я** **Б**а**с**а**р**а**б**о**в** (**1**3**1**0**-**1**5**0**0**)**
- **В**л**а**д** **Ц**е**п**е**ш** (**Д**р**а**к**у**л**а**)

### X. DAMNATA-ПРОТОКОЛ

**DAMNATA (нельзя писать как факт):**

1. «**В**а**л**а**х**и**я** = **в**е**ч**н**о** **с**л**а**в**я**н**с**к**а**я**»** — DAMNATA. **«**В**а**л**а**х**и**я** = **д**а**к**о**-**р**о**м**а**н**с**к**о**-**с**л**а**в**я**н**с**к**а**я**»**
2. «**Д**р**а**к**у**л**а** = **г**е**р**о**й**»** — **у**с**л**о**в**н**о** (**+** **ж**е**с**т**о**к**и**й** **п**р**а**в**о**с**л**а**в**н**ы**й** **к**н**я**з**ь**)**
3. «**Б**а**с**а**р**а**б**и** = **р**у**м**ы**н**ы**»** — **у**с**л**о**в**н**о** (**+** **т**е**р**м**и**н**о**л**о**г**и**я** — **«**б**а**с**а**р**а**б**»** **т**ю**р**к**с**к**о**е**)**»**
4. «**В**а**л**а**ш**с**к**о**е** **к**н**я**ж**е**с**т**в**о** = **п**е**р**в**о**е** **р**у**м**ы**н**с**к**о**е** **г**о**с**у**д**а**р**с**т**в**о**»** — **у**с**л**о**в**н**о** (**+** **к**у**м**а**н**с**к**о**е** **в**л**и**я**н**и**е**)**»**
5. «**М**и**р**ч**а** **С**т**а**р**ы**й** = **в**е**л**и**к**и**й**»** — **у**с**л**о**в**н**о** (**+** **п**а**т**р**и**о**т**)**»**
6. «**А**р**д**ж**е**ш** = **д**р**е**в**н**я**я** **с**т**о**л**и**ц**а**»** — DAMNATA. **«**А**р**д**ж**е**ш** = **с**р**е**д**н**е**в**е**к**о**в**а**я** **с**т**о**л**и**ц**а**»*
7. «**К**о**с**о**в**о**в**о** = **В**а**л**а**х**с**к**а**я** **п**о**б**е**д**а**»** — **у**с**л**о**в**н**о** (**+** **с**е**р**б**с**к**а**я** **п**о**б**е**д**а**)**»**
8. «**Д**р**а**к**у**л**а** = **В**а**л**а**х**с**к**и**й** **г**е**р**о**й**»** — **у**с**л**о**в**н**о** (**+** **м**и**ф**)**»**
9. «**В**а**л**а**х**и**я** = **с**а**м**о**б**ы**т**н**а**я** **к**у**л**ь**т**у**р**а**»** — DAMNATA. **«**В**а**л**а**х**и**я** = **с**м**е**с**ь** **д**а**к**о**-**р**о**м**а**н**с**к**о**г**о** + **с**л**а**в**я**н**с**к**о**г**о**»**
10. «**К**у**м**а**н**ы** = **ч**у**ж**и**е**»** — DAMNATA. **«**К**у**м**а**н**ы** = **5**%** **в**к**л**а**д**а**»*
11. «**С**в**. **Н**и**к**о**л**а** = **п**а**т**р**о**н**»** — **у**с**л**о**в**н**о** (**+** **к**у**л**ь**т**)**»**
12. «**В**а**л**а**х**и**я** = **н**а**ц**и**о**н**а**л**ь**н**о**е** **г**о**с**у**д**а**р**с**т**в**о**»** — **у**с**л**о**в**н**о** (**+** **к**у**м**а**н**с**к**о**е** **в**л**и**я**н**и**е**)**»**
