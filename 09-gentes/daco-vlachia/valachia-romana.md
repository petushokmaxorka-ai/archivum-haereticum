### I. СВОДКА

### IX. DAMNATA-ПРОТОКОЛ

**DAMNATA (нельзя писать как факт):**

1. «**Д**а**к**и**я** = **р**и**м**с**к**а**я** **к**о**л**о**н**и**я**»** — DAMNATA. **«**Д**а**к**и**я** = **р**и**м**с**к**а**я** **п**р**о**в**и**н**ц**и**я**»*
2. «**Р**у**м**ы**н**ы** = **п**р**я**м**ы**е** **р**и**м**л**я**н**е**»** — DAMNATA. **«**Р**у**м**ы**н**ы** = **р**о**м**а**н**и**з**и**р**о**в**а**н**н**ы**е** **д**а**к**и**»*
3. «**С**л**а**в**я**н**е** = **п**р**и**ш**л**ы**е** **з**а**х**в**а**т**ч**и**к**и**»** — DAMNATA. **«**С**л**а**в**я**н**е** = **ч**а**с**т**ь** **э**т**н**о**г**е**н**е**з**а** (3**0**%)**
4. «**К**о**л**о**н**н**а** Т**р**а**я**н**а** = **у**н**и**в**е**р**с**а**л**ь**н**ы**й** **и**с**т**о**ч**н**и**к**»** — **у**с**л**о**в**н**о** (**+** **п**р**о**п**а**г**а**н**д**а**)**»**
5. «**Д**е**ц**е**б**а**л** = **г**е**р**о**й**»** — **у**с**л**о**в**н**о** (**+** **м**и**ф**)**»**
6. «**Р**у**м**ы**н**ы** = **п**р**я**м**ы**е** **п**о**т**о**м**к**и** **р**и**м**л**я**н**»** — DAMNATA. **«**Р**у**м**ы**н**ы** = **р**о**м**а**н**и**з**и**р**о**в**а**н**н**ы**е** **д**а**к**и**»*
7. «**D**a**c**i**a** **F**e**l**i**x** = **р**а**й**»** — **у**с**л**о**в**н**о** (**+** **п**а**т**р**и**о**т**и**ч**е**с**к**о**е**)**»**
8. «**О**т**с**т**у**п**л**е**н**и**е** А**в**р**е**л**и**а**н**а** = **у**х**о**д** **р**и**м**л**я**н**»** — DAMNATA. **«**О**т**с**т**у**п**л**е**н**и**е** А**в**р**е**л**и**а**н**а** = **ч**а**с**т**и**ч**н**ы**й** **у**х**о**д** **л**е**г**и**о**н**о**в**»*
9. «**А**в**р**е**л**и**а**н** = **г**е**р**о**й**»** — DAMNATA. **«**А**в**р**е**л**и**а**н** = **и**м**п**е**р**а**т**о**р**-**с**т**р**а**т**е**г**»*
10. «**В**л**а**х**и** = **т**о**л**ь**к**о** **п**а**с**т**у**х**и**»** — DAMNATA. **«**В**л**а**х**и** = **р**о**м**а**н**и**з**и**р**о**в**а**н**н**о**е** **н**а**с**е**л**е**н**и**е**»*
11. «**С**и**р**м**и**й**с**к**а**я** **а**р**х**и**д**и**о**ц**е**з**а** = **в**и**з**а**н**т**и**й**с**к**а**я**»** — DAMNATA. **«**С**и**р**м**и**й**с**к**а**я** **а**р**х**и**д**и**о**ц**е**з**а** = **а**р**х**и**д**и**о**ц**е**з**а** **п**о**д** **в**и**з**а**н**т**и**й**с**к**и**м** **к**о**н**т**р**о**л**е**м**»*
12. «**D**a**c**i**a** **e**t**e**r**n**a** = **н**а**у**ч**н**а**я** **т**е**о**р**и**я**»** — DAMNATA. **«**D**a**c**i**a** **e**t**e**r**n**a** = **к**у**л**ь**т**у**р**н**ы**й** **м**и**ф**»*
