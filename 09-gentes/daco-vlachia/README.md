### V. ОТГОННОЕ ЖИВОТНОВОДСТВО (VALACHIA TRANSHUMANTIA)

**Трансгумация (Transhumance)**:
- **«Отгонное скотоводство»** (от латинского *trans* + *humus* = «через землю»)
- **«Сезонные перегоны скота»** (горы ↔ равнины)
- **«Балкано-Дунайские перегоны»** (Балканы ↔ Фракия ↔ Эгейское море)
- **«Карпато-Дунайские перегоны»** (Карпаты ↔ Дунай ↔ Валахия)
- **«Римская основа»** (римские латифундии + византийская преемственность)
- **«Славянский пласт»** (южнославянское крестьянство)
- **«Тюркский пласт»** (булгарский/печенежский/татарский)
- **«Романский пласт»** (италийский)
- **«Доказательство слоистости этноса»** (даки + римляне + славяне + тюрки + греки)

### VII. СТРАТЕГИЯ NODUS

**Дако-влашская стратегия NODUS (узел)**:

> «Дако-влахи — «узел» между Дакией, Римом, Славией, Византией, Тюркией; доказательство слоистости этноса; DAMNATA к националистическим прочтениям — «влахи = истинные римляне»/«истинные славяне»/«истинные даки»; это NODUS + двоеверие».

**Сравнение с другими стратегиями**:
- **Дако-влахи** — NODUS (узел)
- **Скифы** — NODUS (узел)
- **Валашское язычество** — NODUS (узел)
- **Балканы** — NODUS/TRANSMISSIO (узел/передача)
- **Средиземноморье** — CONCURRENTIA (соперничество)
- **Славия** — REPLICATIO (копирование)
- **Эфиопия/Семитское** — ISOLATIO/RESISTENTIA

### IX. DAMNATA-ПРОТОКОЛ

**DAMNATA (нельзя писать как факт):**

1. «Влахи = «истинные римляне» — DAMNATA. Влахи = «слоистый этнос» (даки + римляне + славяне + тюрки + греки)
2. «Влахи = «истинные славяне» — DAMNATA. Влахи = «слоистый этнос» (даки + римляне + славяне + тюрки + греки)
3. «Влахи = «истинные даки» — DAMNATA. Влахи = «слоистый этнос» (даки + римляне + славяне + тюрки + греки)
4. «Дракула = вампир» — DAMNATA. Дракула = «исторический господарь Валахии», «прототип литературного вампира»
5. «Влахи = румыны» — DAMNATA. Влахи = «общее название» пастухов и романоязычных, румыны = «национальное самоназвание» XIX-XXI вв.
6. «Басарабы = румынская династия» — DAMNATA. Басарабы = «общая династия» Валахии, Молдавии, Трансильвании (сложная история)
7. «Трансгумация = пастушество» — DAMNATA. Трансгумация = «механизм романизации» + «механизм сохранения этноса»
8. «Объединение 1859 = «создание Румынии» — DAMNATA. Объединение 1859 = «объединение Молдавии и Валахии» (без Трансильвании, Буковины, Бессарабии)
