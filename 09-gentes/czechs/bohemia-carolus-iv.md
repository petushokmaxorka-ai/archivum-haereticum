# BOHEMIA CAROLUS IV
## Карл IV (1316-1378): император Священной Римской империи, «отец отечества», основатель Праги
### «Карл IV (1316-1378) = чешский король (1346-1378) и император СРИ (1355-1378); «отец отечества» (Pater patriae); основатель Пражского университета (1348), Золотой буллы (1356), Карлова моста (1357), собора Св. Вита (1344); Прага = столица СРИ; один из величайших правителей Средневековья»

**CLASSIFICATIO: ◆ FIDELIS / ◇ SUSPECTUS-FERTILIS / ✠ DAMNATUS / ✚ CREDO: «Карл IV = чешский король; Прага = славянская столица СРИ; DAMNUM для «Карл IV = немец»»**
Datum: 148.M3

### I. СВОДКА

**Карл IV (Karel IV, 1316-1378)** — чешский король (1346-1378), император Священной Римской империи (1355-1378). «Отец отечества» (Pater patriae). **Пражский университет** (1348, первый в Центральной Европе). **Золотая булла** (1356). **Карлов мост** (1357). **Собор Св. Вита** (1344). **Прага = столица СРИ**. Один из величайших правителей Средневековья.

**Главные тезисы**:
1. **1316 н.е.** — родился в Праге
2. **1346 н.е.** — король Чехии
3. **1348 н.е.** — Пражский университет
4. **1355 н.е.** — император
5. **1356 н.е.** — Золотая булла
6. **1357 н.е.** — Карлов мост
7. **«Отец отечества»** = Карл IV

---

### II. ЭТИМОЛОГИЯ

**«Карл» (Karel / Carolus / Charles)** = **«Karl-» (Карл) + «-ing» (потомок)** = *«Потомок Карла»*
**«Karel IV»** = «Карл IV» (четвёртый)
**«Carolus IV»** = латинская форма
**«Charles IV»** = английская форма
**«Karl IV»** = немецкая форма

**Сравнить**:
- **Karel** = чешская форма
- **Carolus** = латинская форма
- **Karl** = немецкая форма
- **Charles** = английская форма

«Карл IV (1316-1378) = чешский король (1346-1378), император СРИ (1355-1378), «отец отечества» (Pater patriae). Родился в Праге 14 мая 1316. Учился в Париже. Маркграф Моравии (1333).
