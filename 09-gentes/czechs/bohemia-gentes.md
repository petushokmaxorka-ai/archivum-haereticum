# BOHEMIA GENTES
## Чешский этнос и его подгруппы: чехи, ханека, мoравa, влахи, поляки, немцы
### «Чешский этнос (Češi) = 10,5 млн живых чехов; жители Чешской Республики; говорят на чешском языке (čeština); подгруппы: ханека (чешск.) (X-XI вв.), поляки (горные), словаки (Моравская Словакия), немцы (Судеты), влахи (румыны), цыгане; чешский = западнославянский, ближе к словацкому, чем к польскому»

**CLASSIFICATIO: ◆ FIDELIS / ◇ SUSPECTUS-FERTILIS / ✠ DAMNATUS / ✚ CREDO: «Чехи = отдельный народ 10,5 млн; DAMNUM для «чехи = словаки» / «чехи = словаки = один»»**
Datum: 144.M3

---

### I. СВОДКА

**Чешский этнос (Češi) = 10,5 млн живых чехов**:
- **Жители**: Чешская Республика
- **Язык**: чешский (čeština) = 10,5 млн носителей
- **Подгруппы**: ханека, мораване, силезцы (отдельные), влахи, поляки, словаки, немцы (Судеты)
- **Чешский = западнославянский** (ближе к словацкому, чем к польскому)

**Главные тезисы**:
1. **10,5 млн чехов**
2. **Чешский язык** = отдельный
3. **Ханека** = в Чехии
4. **Мораване** = отдельный народ (3 млн)
5. **Силезцы** = отдельный народ (4-5 млн)
6. **Судеты** = DAMNUM (1945)
7. **Влахи** = в Чехии

---

### II. ЭТИМОЛОГИЯ

**«Чехи» (Češi / Čechové)** = **«Čech-» (Чех) + «-ové» (люди)** = *«Люди Чеха»*
**«Чехия»** = от легендарного **Чеха** (праотца)
**«Češi»** = чешский самоназвание
**«Bohemi»** = латинское название
**«Böhmen»** = немецкое название
**«Чехи»** = русское название

**Гипотезы**:
1. **«Чех»** = от славянского «čech-» (межевой знак)
2. **«Чех»** = имя мифического предка
3. **«Bohemia»** = от кельтского «Boii»
