# MORAVIA RELIGIO
## Моравская религия: синтез язычества, христианства и двоеверия
### «Моравская религия (V-XXI вв.н.э.) — синтез язычества (V-IX вв.), христианства (863-1948), двоеверия (863-XIX вв.), безверия (1948-1989), возрождения (1990-2026); 8 богов = 8 святых; 12 святых мораван; 8 обрядов; 7 праздников; 5 мифов; 3 сакральных места»

**CLASSIFICATIO: ◆ FIDELIS / ◇ SUSPECTUS-FERTILIS / ✠ DAMNATUS / ✚ CREDO: «Моравская религия = синтез язычества, христианства, двоеверия; DAMNUM для «мораване = чехи (без своего)»»**
Datum: 132.M3

---

### I. СВОДКА

**Моравская религия (V-XXI вв.н.э.)** — синтез:
1. **Язычество (V-IX вв.)** — славянское, Перун, Велес, Мокошь
2. **Христианство (863-1948)** — Кирилл и Мефодий, Бенеш (Гуситы)
3. **Двоеверие (863-XIX вв.)** — славянские обряды + христианские
4. **Безверие (1948-1989)** — социализм, атеизм
5. **Возрождение (1990-2026)** — моравский национализм, MNH

**8 богов = 8 святых** (по аналогии с полабской двоеверием):
- **Перун** = Св. Илья (громовержец)
- **Велес** = Св. Власий (скот)
- **Мокошь** = Параскева Пятница (женщины)
- **Сварог** = Св. Кузьма (кузнец)
- **Даждьбог** = Св. Харлампий (солнце)
- **Лада** = Дева Мария (красота)
- **Хорс** = Св. Спиридон (солнце)
- **Стрибог** = Св. Николай (ветры)

**Главные тезисы**:
1. **5 периодов** моравской религии
2. **8 богов = 8 святых**
3. **12 святых мораван**
4. **8 обрядов**
5. **7 праздников**
6. **5 мифов**
7. **3 сакральных места**
