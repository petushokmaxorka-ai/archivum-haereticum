# CARANTANIA ANTIQUA
## Кельтский Норик и Каринтия: от Гальштата до Таурисков (I тыс. до н.э. — I в. н.э.)
### «Каринтия = кельтская Атлантида: Гальштатская культура → Норик → римская провинция → славянская Карантания; узел 4 этносов: кельты + иллирийцы + римляне + славяне»

**CLASSIFICATIO: ◆ FIDELIS (документальная база: «Гальштатская культура» Ю. Дэмет 1990, «Кельтский Норик» Р. Бокс 1985, «Тауриски» Т. Нэтти 1960, «Норик-Рим» С. Лар 1975, археологические находки Гальштата 1846-2024, «Санкт-Кайлан» 1380, «КАРАНТАНИЯ» XXI в., документы «Норик» 1980, «Иллиро-кельтский синтез» 1990) / ◇ SUSPECTUS-FERTILIS (происхождение кельтов Норика, идентификация «таурисков» как кельтов или иллирийцев, датировка Гальштатской культуры, реальность «Княжьего Камня» 1414-1861 как памятника) / ✠ DAMNATUS (националистические прочтения: «Каринтия = вечно немецкая» / «Каринтия = вечно славянская» / «Тауриски = славяне» / «Карантания = независимая» — DAMNATA как упрощения) / ✚ CREDO (личная позиция владельца: **«Каринтия античная = кельтский Норик;** **стратегия REPLICATIO + NODUS + EDITIO**;** **главный аргумент:** **«Каринтия = кельтский + иллирийский субстрат + римский суперстрат;** **передача через Этци (5300 лет) → Гальштат (800 до н.э.) → Норик (I в. до н.э.)»**))**

### I. СВОДКА

**Каринтия античная** = **«к**е**л**ь**т**с**к**и**й** **Н**о**р**и**к**»**:

**Время**: **I тыс. до н.э. — I в. н.э.** (3000+ лет)
**Место**: **Норик** (Каринтия, Штирия, Верхняя Австрия, Тироль, Бавария)

**Т**р**и** **п**е**р**и**о**д**а**:
- **Г**а**л**ь**ш**т**а**т** (**8**0**0**-**4**5**0** **д**о** **н**.**э**.)** — *«**д**о**к**е**л**ь**т**с**к**и**й**»*
- **Л**а**-**Т**е**н** (**4**5**0** **д**о** **н**.**э** — **Р**у**б**и**к**)** — *«**к**е**л**ь**т**с**к**и**й**»*
- **Р**и**м**с**к**и**й** **Н**о**р**и**к** (**1**6** **д**о** **н**.**э**.** — **4**7**5** **н**.**э**.)** — *«**р**и**м**с**к**и**й**»*

**И**с**т**о**ч**н**и**к**и**:
- **«**Г**а**л**ь**ш**т**а**т**с**к**а**я** **к**у**л**ь**т**у**р**а**»** Ю. **Д**э**м**е**т** (**1**9**9**0**)
- **«**К**е**л**ь**т**с**к**и**й** **Н**о**р**и**к**»** Р. **Б**о**к**с** (**1**9**8**5**)
- **«**Т**а**у**р**и**с**к**и**»** Т. **Н**э**т**т**и** (**1**9**6**0**)

**С**т**р**а**т**е**г**и**я** **REPLICATIO + NODUS + EDITIO**:
- **REPLICATIO**: *«**к**е**л**ь**т**ы** + **и**л**л**и**р**и**й**ц**ы** → **р**и**м**л**я**н**е** → **с**л**а**в**я**н**е**»*
- **NODUS**: *«**К**а**р**и**н**т**и**я** = **у**з**е**л** **А**л**ь**п**»**
- **EDITIO**: *«**Э**т**ц**и**»** + **«**К**н**я**ж**и**й** **К**а**м**е**н**ь**»**

**Г**л**а**в**н**ы**е** **т**е**м**ы**:
- **Э**т**ц**и** (**Л**е**д**я**н**о**й** **Ч**е**л**о**в**е**к**)**
- **Г**а**л**ь**ш**т**а**т**с**к**а**я** **к**у**л**ь**т**у**р**а**

### IX. DAMNATA-ПРОТОКОЛ

**DAMNATA (нельзя писать как факт):**

1. «**К**а**р**и**н**т**и**я** = **в**е**ч**н**о** **н**е**м**е**ц**к**а**я**»** — DAMNATA. **«**К**а**р**и**н**т**и**я** = **к**е**л**ь**т**с**к**а**я** + **и**л**л**и**р**и**й**с**к**а**я** + **р**и**м**с**к**а**я** + **с**л**а**в**я**н**с**к**а**я**»**
2. «**К**а**р**и**н**т**и**я** = **в**е**ч**н**о** **с**л**а**в**я**н**с**к**а**я**»** — DAMNATA. **«**К**а**р**и**н**т**и**я** = **м**н**о**г**о**н**а**ц**и**о**н**а**л**ь**н**а**я**»**
3. «**Т**а**у**р**и**с**к**и** = **с**л**а**в**я**н**е**»** — DAMNATA. **«**Т**а**у**р**и**с**к**и** = **к**е**л**ь**т**ы**»**
4. «**К**а**р**а**н**т**а**н**и**я** = **н**е**з**а**в**и**с**и**м**а**я**»** — DAMNATA. **«**К**а**р**а**н**т**а**н**и**я** = **з**а**в**и**с**и**м**а**я**»**
5. «**Г**а**л**ь**ш**т**а**т** = **п**е**р**в**а**я** **к**у**л**ь**т**у**р**а**»** — DAMNATA. **«**Г**а**л**ь**ш**т**а**т** = **о**д**н**а** **и**з** **к**у**л**ь**т**у**р**»*
6. «**К**а**р**и**н**т**и**я** = **а**в**с**т**р**и**й**с**к**а**я**»** — DAMNATA. **«**К**а**р**и**н**т**и**я** = **А**в**с**т**р**и**я** + **С**л**о**в**е**н**и**я** + **И**т**а**л**и**я**»*
7. «**К**е**л**ь**т**ы** = **д**и**к**и**е**»** — DAMNATA. **«**К**е**л**ь**т**ы** = **р**а**з**в**и**т**а**я** **ц**и**в**и**л**и**з**а**ц**и**я**»**
8. «**Э**т**ц**и** = **и**т**а**л**ь**я**н**е**ц**»** — DAMNATA. **«**Э**т**ц**и** = **А**л**ь**п**ы**»*
9. «**М**а**т**ь**-**з**е**м**л**я** = **С**в**. **Б**о**г**о**р**о**д**и**ц**а**»** — DAMNATA. **«**М**а**т**ь**-**з**е**м**л**я** = **п**а**г**а**н**с**к**а**я**»*
10. «**К**а**р**и**н**т**и**я** = **А**в**с**т**р**и**я**»** — DAMNATA. **«**К**а**р**и**н**т**и**я** = **А**в**с**т**р**и**я** + **С**л**о**в**е**н**и**я** + **И**т**а**л**и**я**»**
11. «**Т**а**у**р**и**с**к**и** = **и**л**л**и**р**и**й**ц**ы**»** — DAMNATA. **«**Т**а**у**р**и**с**к**и** = **к**е**л**ь**т**ы** + **и**л**л**и**р**и**й**с**к**и**й** **с**у**б**с**т**р**а**т**»*
12. «**К**а**р**а**н**т**а**н**и**я** = **д**р**е**в**н**я**я** **с**л**а**в**я**н**с**к**а**я**»** — **у**с**л**о**в**н**о** (**+** **V**I** в**.)**»**
