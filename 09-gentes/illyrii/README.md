### II. ГЛАВНЫЕ ВОПРОСЫ

**Вопрос 1: Кто такие иллирийцы?**
- **Иллирийский язык** — **нерасшифрован**; относится к **индоевропейским** языкам (возможно, балто-славянская или фракийско-иллирийская ветвь)
- **Генетика** — **Y-ДНК J2** (средиземноморская) + **R1b** (западноевропейская) + **I2** (балканская)
- **Археология** — **поля погребений** (Хумка, Гласинац), **мегалитические** сооружения, **крепости** (Доклеа, Ризон)
- **Страбон**: «Иллирийцы живут от Адриатики до Шар-Планины и Южного Дуная»

**Вопрос 2: Иллирийцы = албанцы?**
- **Албанская гипотеза (50%)**: иллирийский → албанский (неславянский язык на Балканах, отличный от румынского)
- **Фракийская гипотеза (30%)**: иллирийский → фракийский → дако-мизийский → албанский
- **Славянская гипотеза (10%)**: иллирийский = предок сербо-хорватского
- **Гипотеза множественности (10%)**: иллирийский = общий термин для нескольких разных этносов

**Вопрос 3: Доклеаты = предки черногорцев?**
- **Доклеа** = «Дукля» = «Дукльа» = «место у озера» (древнее название Подгорицы/Скадарского озера)
- **Дукля** (Доклеа) = первое славянское государство на Балканах (X-XII вв.)
- **Дукля → Зета → Черногория** (национальная преемственность)
- **Дукля = «Дукльаны»** в «Хронике Попа Дуклянина» (XII в.)

### V. СВЯЗЬ С ОБЩЕЙ ЦЕЛЬЮ АРХИВА

**«**П**о**с**р**е**д**с**т**в**о**м** **и**л**л**и**р**и**й**ц**е**в** **у**в**и**д**е**т**ь** **о**б**щ**и**й** **з**а**к**о**н** **Б**о**ж**и**й** **ч**е**р**е**з** **и**л**л**и**р**и**й**с**к**и**е** **к**у**л**ь**т**ы** **п**р**и**р**о**д**н**ы**х** **с**и**л**»**:
- **И**л**л**и**р**и**й**ц**ы** → **Д**и**с**-**П**а**т**р** + **С**и**л**ь**в**а**н** + **М**е**д**о**р** (**и**л**л**и**р**и**й**с**к**и**й** **к**у**л**ь**т** **п**о**д**з**е**м**н**о**г**о** **м**и**р**а** + **л**е**с**а** + **з**м**е**и**-**и**с**ц**е**л**и**т**е**л**я**)
- **Г**р**е**к**и** → **З**е**в**с** + **А**ф**и**н**а** + **А**п**о**л**л**о**н** (**г**р**е**ч**е**с**к**и**й** **п**а**н**т**е**о**н**)
- **Р**и**м**л**я**н**е** → **Ю**п**и**т**е**р** + **М**и**н**е**р**в**а** + **В**е**н**е**р**а** (**р**и**м**с**к**и**й** **п**а**н**т**е**о**н**)
- **С**л**а**в**я**н**е** → **Р**о**д** + **М**о**к**о**ш**ь** + **С**в**а**р**о**г** (**с**л**а**в**я**н**с**к**и**й** **п**а**н**т**е**о**н**)
- **П**р**а**в**о**с**л**а**в**н**ы**е** → **С**в**. **К**л**и**м**е**н**т** **О**х**р**и**д**с**к**и**й** + **С**в**. **И**о**а**н**н** **В**л**а**д**и**м**и**р** (**п**р**а**в**о**с**л**а**в**н**ы**й** **п**а**н**т**е**о**н**)
- **К**а**т**о**л**и**к**и** → **С**в**. **Т**р**и**ф**у**н** **К**о**т**о**р**с**к**и**й** (**к**а**т**о**л**и**ч**е**с**к**и**й** **п**а**н**т**е**о**н**)
