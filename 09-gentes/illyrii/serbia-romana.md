# SERBIA ROMANA
## Римская Сербия: Верхняя Мезия, Дакия, Прибрежная Дакия (I в. до н.э. — IV в. н.э.)
### «Сербия была ОДНОЙ ИЗ ВАЖНЕЙШИХ ПРОВИНЦИЙ РИМСКОЙ ИМПЕРИИ: через неё шёл Дунай (limes); отсюда пришли 17 римских императоров (Константин Великий, Юстиниан, Максимин Фракиец и др.); Виминаций = крупнейший город Балкан; Сердика = "маленький Рим"; христианизация в IV в. = одна из первых в империи»

Петровича 1971) / ◇ SUSPECTUS-FERTILIS (реконструкция римской религиозной жизни, точное число легионов, соотношение романизированного и местного населения, идентификация племен) / ✠ DAMNATUS (националистические прочтения: «римляне = оккупанты» / «римляне = братья по крови» — обе DAMNATA) / ✚ CREDO (личная позиция владельца: **«Римская Мезия = "Великая Сербия до сербов": 17 римских императоров; 200+ латинских надписей; главный плавильный котёл балканского этногенеза»;** **стратегия EDITIO + TRANSMISSIO + NODUS**;** **главный аргумент:** **«Сербия = "римский близнец" Балкан: после распада Рима Верхняя Мезия стала Сербией, как Галлия = Франция»**))**

### I. СВОДКА

**Римская Сербия (I в. до н.э. — IV в. н.э.) = 500 лет**:

**Три фазы**:
- **Завоевание** (I в. до н.э. — I в. н.э.) — *«Август → Траян»*
- **Расцвет** (I-III вв. н.э.) — *«limes Danubianus»*
- **Кризис и распад** (III-IV вв. н.э.) — *«великое переселение»*

**Ключевые провинции**:
- **Верхняя Мезия** (Moesia Superior) — *«ядро Сербии»*
- **Дакия** (Dacia) — *«восток Сербии»*
- **Прибрежная Дакия** (Dacia Ripensis) — *«юг»*
- **Внутренняя Дакия** (Dacia Mediterranea) — *«центр»*

**17 римских императоров**:
- **Константин Великий** (306-337) — *«родился в Наиссе»*
- **Юстиниан I** (527-565) — *«родился в Таврисиуме»*

- **Клавдий II Готский** (268-270) — *«родился в Иллирии»*
- **Аврелиан** (270-275) — *«родился в Сирмии (Сремска Митровица)»*
- **Проб** (276-282) — *«родился в Сирмии»*
- **Максимиан** (286-305) — *«родился в Сирмии»*
- **Галерий** (305-311) — *«родился в Сирмии»*
- **Лициний** (308-324) — *«родился в Мезии»*
- **Констанций II** (337-361) — *«родился в Сирмии»*
- **Юлиан Отступник** (361-363) — *«родился в Константинополе (но мать из Мезии)»*
- **Грациан** (367-383) — *«родился в Сирмии»*
- **Валент** (364-378) — *«родился в Паннонии»*

**Параллельно** **`09-gentes/illyrii/bosnia-romana.md`**, **`09-gentes/illyrii/montenegro-romana.md`**, **`09-gentes/illyrii/croatia-romana.md`**, **`09-gentes/illyrii/albania-romana.md`**, **`09-gentes/illyrii/carantania-romana.md`**.

**С**т**р**а**т**е**г**и**я** **EDITIO + TRANSMISSIO + NODUS**:
- **EDITIO**: **р**и**м**с**к**и**е** **и**с**т**о**ч**н**и**к**и** (**Т**а**ц**и**т**, **С**в**е**т**о**н**и**й**, **А**м**м**и**а**н** М**а**р**ц**е**л**л**и**н**, **П**р**о**к**о**п**и**й**)
- **TRANSMISSIO**: **З**а**л**м**о**л**к**с**и**с** + **М**а**р**с** + **Ю**п**и**т**е**р** → **С**в**. **Г**е**о**р**г**и**й** + **И**и**с**у**с** **Х**р**и**с**т**о**с**
- **NODUS**: **С**е**р**б**и**я** = **у**з**е**л** **Б**а**л**к**а**н** (**1**7** и**м**п**е**р**а**т**о**р**о**в**)**

**Главный аргумент**: «**Сербия = "Великая Сербия до сербов": 17 римских императоров из Мезии (Константин, Юстиниан, Максимин, Клавдий II, Аврелиан, Проб, Максимиан, Галерий и др.); 200+ латинских надписей; Виминаций = крупнейший город Балкан; христианизация в IV в. = одна из первых в империи; главный плавильный котёл балканского этногенеза; Мезианская диалектная основа восточно-романского языка (румынский, арумынский) = прямое наследие**».

**Главный аргумент-сравнение**: «**Сербия похожа на Россию по православию, кириллице, славянской основе, НО по крови Сербия = "Великая Сербия до сербов" = 17 римских императоров + римские легионы + мезийская культура, в то время как Россия = славянская + финно-угорская**».
