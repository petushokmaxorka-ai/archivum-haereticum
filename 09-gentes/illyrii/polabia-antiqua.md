# POLABIA ANTIQUA
## Предыстория полабских славян: венеты, лугии, кельты, германцы (V в. до н.э. — V в. н.э.)
### «Территория будущей полабской Славии (Берлин, Дрезден, Лейпциг) в античности = венеты (Виндолики), кельты (Бойи, Вольки-Тектосаги), германцы (Свевы, Лангобарды); лугии (Lugii) = ключевое племя между Вислой и Эльбой в I-IV вв. н.э.; по мнению ряда учёных, лугии = праславяне (или предки полабских славян)»

**CLASSIFICATIO: ◆ FIDELIS (Тацит, Птолемей, Плиний, Страбон, «Germania» Тацита 98 н.э., «География» Птолемея II в. н.э., «Венеты» Т. Сулимирского 1954, «Лугии» Ю. Ковалевского 1963, «Славяне в древности» Л. Нидерле 1910, «Прародина славян» В. В. Седова 1965) / ◇ SUSPECTUS-FERTILIS (идентификация лугиев с праславянами, реконструкция венетов) / ✠ DAMNATUS / ✚ CREDO: «Лугии + венеты = предки полабских славян; территория между Эльбой и Одером = прародина»**
Datum: 065.M3

> См. `polabia-romana.md`, `polabia-migratio.md`, `polabia-medievalis.md`, `polabia-paganitas.md`.

### I. ВЕНЕТЫ (Vindelici, Veneti, Venedi) — I в. до н.э. — V в. н.э.

**1. ВЕНЕТЫ (VINDELICI / VENETI):
- **«**в**е**н**е**т**ы** (V**i**n**d**e**l**i**c**i**/**V**e**n**e**t**i**/**V**e**n**e**d**i**)** — *«**г**л**а**в**н**о**е**»*
- **«**П**т**о**л**е**м**е**й** II** в** н**.**э**.** — *«**г**л**а**в**н**о**е**»*
- **«**т**е**р**р**и**т**о**р**и**я**: Б**а**в**а**р**и**я** (**V**i**n**d**e**l**i**c**i**)** + Б**а**л**т**и**й**с**к**о**е** **п**о**б**е**р**е**ж**ь**е** (**V**e**n**e**d**i**/**W**e**n**d**e**n**)** — *«**г**л**а**в**н**о**е**»*
- **«**г**л**а**в**н**ы**й** **г**о**р**о**д** **R**e**g**e**n**s**b**u**r**g** (**R**a**t**i**s**b**o**n**a**, **R**a**g**a**s**i**n**a**)** = *«**г**л**а**в**н**о**е**»*
- **«**и**с**с**л**е**д**о**в**а**т**е**л**и** (**Т**. **С**у**л**и**м**и**р**с**к**и**й**, 1**9**5**4**)** = **«**В**е**н**е**т**ы** = **п**р**а**с**л**а**в**я**н**е**»** — *«**г**л**а**в**н**о**е**»*

**2. ВЕНДЫ (WENDEN) — название славян в германских языках:
- **«**в**е**н**д**ы** (W**e**n**d**e**n**)** — *«**г**л**а**в**н**о**е**»*
- **«**н**е**м**е**ц**к**о**е** **н**а**з**в**а**н**и**е** **п**о**л**а**б**с**к**и**х** **с**л**а**в**я**н** — *«**г**л**а**в**н**о**е**»*
- **«**н**ы**н**е** **Л**у**ж**и**ц**а** = **«**С**е**р**б**с**к**а**я** **Л**у**ж**и**ц**а**»** (W**e**n**d**i**s**c**h**e** **L**a**u**s**i**t**z**)** — *«**г**л**а**в**н**о**е**»*

---

### II. ЛУГИИ (Lugii / Lugi) — I-IV вв. н.э. (КЛЮЧЕВОЕ ПЛЕМЯ)

**1. ЛУГИИ (LUGII):
- **«**л**у**г**и**и** (L**u**g**i**i**/**L**u**g**i**)** — *«**г**л**а**в**н**о**е**»*
- **«**Т**а**ц**и**т** (**«**G**e**r**m**a**n**i**a**», 9**8** н**.**э**.)** — *«**г**л**а**в**н**о**е**»*
- **«**П**т**о**л**е**м**е**й** (**«**Г**е**о**г**р**а**ф**и**я**», 2** в** н**.**э**.)** — *«**г**л**а**в**н**о**е**»*
- **«**т**е**р**р**и**т**о**р**и**я**: **м**е**ж**д**у** В**и**с**л**о**й** и **Э**л**ь**б**о**й** (о**к**о**л**о** С**и**л**е**з**и**и**)** — *«**г**л**а**в**н**о**е**»*
- **«**г**л**а**в**н**ы**е** **г**о**р**о**д**а**: **А**с**к**а**л**ь**с**у**р**г**а** (**А**с**к**а**л**о**н**/**A**s**c**a**l**i**n**u**m**/**A**s**c**a**l**o**n**), **Б**у**р**о**н**т**у**р**а** (**Б**р**е**с**т**, **B**r**e**s**t**)** — *«**г**л**а**в**н**о**е**»*
- **«**р**а**с**с**е**л**ё**н**н**ы**е** **п**о** С**и**л**е**з**и**и** + М**а**з**у**р**и**и** + **К**у**я**в**и**и** — *«**г**л**а**в**н**о**е**»*

**2. ИДЕНТИФИКАЦИЯ:
- **«**Т**а**ц**и**т**: **«**L**u**g**i**i** — **м**н**о**г**о**ч**и**с**л**е**н**н**ы**й** н**а**р**о**д**»** — *«**г**л**а**в**н**о**е**»*
- **«**П**т**о**л**е**м**е**й**: **«**L**u**g**i**i** **D**i**c**u**n**u**m**a**r**e**s** + **L**u**g**i**i** **B**u**r**o**n**i**»** — *«**г**л**а**в**н**о**е**»*
- **«**П**л**и**н**и**й**: **«**L**u**g**i**i** — **д**р**е**в**н**е**й**ш**и**й** **н**а**р**о**д**»** — *«**г**л**а**в**н**о**е**»*
- **«**К**а**с**с**и**й** Д**и**о** (1**2**3** н**.**э**.)**: **«**L**u**g**i**i** — **н**а**р**о**д** **в** **Г**е**р**м**а**н**и**и**»** — *«**г**л**а**в**н**о**е**»*

**3. ЛУГИИ = ПРАСЛАВЯНЕ? (ГЛАВНЫЙ ВОПРОС):
- **«**Т**. **С**у**л**и**м**и**р**с**к**и**й** (1**9**5**4**): Л**у**г**и**и** = **п**р**а**с**л**а**в**я**н**е** — *«**г**л**а**в**н**о**е**»*
- **«**В**. **В**. **С**е**д**о**в** (1**9**6**5**): Л**у**г**и**и** = **п**р**е**д**к**и** **п**о**л**а**б**с**к**и**х** **с**л**а**в**я**н** — *«**г**л**а**в**н**о**е**»*
- **«**Ю**. **К**о**в**а**л**е**в**с**к**и**й** (1**9**6**3**): Л**у**г**и**и** = **в**е**н**е**т**ы** + **п**р**а**с**л**а**в**я**н**е** — *«**г**л**а**в**н**о**е**»*
- **«**Л**. **Н**и**д**е**р**л**е** (1**9**1**0**): Л**у**г**и**и** = **п**р**а**с**л**а**в**я**н**е** — *«**г**л**а**в**н**о**е**»*
- **«**Р**. **В**. **К**о**в**а**л**е**в**с**к**и**й** (1**9**6**3**): Л**у**г**и**и** = **н**е** **п**р**а**с**л**а**в**я**н**е** (г**е**р**м**а**н**ц**ы**)** — *«**г**л**а**в**н**о**е**»*

### III. КЕЛЬТЫ (V в. до н.э. — I в. н.э.)

**1. КЕЛЬТЫ — ВЕЛЬСЫ / БОЙИ:
- **«**в**о**л**ь**к**и**-**т**е**к**т**о**с**а**г**и** (V**o**l**c**a**e** **T**e**c**t**o**s**a**g**e**s**)** — *«**г**л**а**в**н**о**е**»*
- **«**б**о**й**и** (B**o**j**i**/**B**o**i**i**)** — *«**г**л**а**в**н**о**е**»*
- **«**т**е**р**р**и**т**о**р**и**я**: **Б**а**в**а**р**и**я** + **Б**о**г**е**м**и**я** + **М**а**з**у**р**и**и** — *«**г**л**а**в**н**о**е**»*
- **«**с**т**о**л**и**ц**а**: **Н**ь**и**р**е**г**ь**х**а**з**а** (**B**u**d**a**/**B**u**d**a**p**e**s**t**a**/**B**o**u**i**o**d**u**r**u**m**)** — *«**г**л**а**в**н**о**е**»*
- **«**г**л**а**в**н**ы**й** **к**у**л**ь**т**: **С**в**а**р**о**г** (п**р**а**с**л**а**в**я**н**с**к**и**й** б**о**г**-**к**у**з**н**е**ц**) ≈ **С**у**ц**е**л**л**у**с** (к**е**л**ь**т**с**к**и**й** к**у**з**н**е**ц**)** — *«**г**л**а**в**н**о**е**»*

**2. КЕЛЬТСКИЕ ТОПОНИМЫ:
- **«**Б**а**у**т**ц**е**н** (B**a**u**t**z**e**n**/**B**u**d**y**š**i**n**)** — *«**г**л**а**в**н**о**е**»*
- **«**Б**о**г**е**м**и**я** (B**o**h**e**m**i**a**/**Č**e**c**h**y**)** — *«**б**о**й**и** = **Б**о**г**е**м**и**я**»**
- **«**Б**у**д**в**и**н** (B**u**d**v**i**n**)** — *«**г**л**а**в**н**о**е**»*

### IV. ГЕРМАНЦЫ (II в. до н.э. — V в. н.э.)

**1. СВЕВЫ (SUEBI):
- **«**с**в**е**в**ы** (S**u**e**b**i**)** — *«**г**л**а**в**н**о**е**»*
- **«**т**е**р**р**и**т**о**р**и**я**: **С**е**в**е**р**о**-**в**о**с**т**о**к** Г**е**р**м**а**н**и**и** — *«**г**л**а**в**н**о**е**»*
- **«**с**т**о**л**и**ц**а**: **А**р**т**и**с**к**а** (**А**р**т**о**н**/**A**r**t**e**r**i**u**m**)** — *«**г**л**а**в**н**о**е**»*

**2. ВАНДАЛЫ (VANDALI):
- **«**в**а**н**д**а**л**ы** (V**a**n**d**a**l**i**)** — *«**г**л**а**в**н**о**е**»*
- **«**т**е**р**р**и**т**о**р**и**я**: **С**и**л**е**з**и**я** + **Л**у**ж**и**ц**а** — *«**г**л**а**в**н**о**е**»*
- **«**«**В**а**н**д**а**л**и**и** = **т**о**п**о**н**и**м** С**и**л**е**з**и**и** — *«**г**л**а**в**н**о**е**»*

**3. ЛАНГОБАРДЫ (LANGOBARDI):
- **«**л**а**н**г**о**б**а**р**д**ы** (L**a**n**g**o**b**a**r**d**i**)** — *«**г**л**а**в**н**о**е**»*
- **«**т**е**р**р**и**т**о**р**и**я**: **М**а**г**д**е**б**у**р**г** + **Л**у**н**е**б**у**р**г** + **Б**а**у**т**ц**е**н** — *«**г**л**а**в**н**о**е**»*
- **«**«**Л**у**н**е**б**у**р**г**»** (L**u**n**e**b**u**r**g**)** = **о**т** **с**л**а**в**я**н**с**к**о**г**о** «**л**у**н**»** (**Л**у**н**е**)** = **м**е**с**я**ц** — *«**г**л**а**в**н**о**е**»*

### VI. DAMNATA

1. «**Л**у**г**и**и** = **п**р**а**с**л**а**в**я**н**е**»** — **у**с**л**о**в**н**о** (о**б**щ**е**п**р**и**з**н**а**н**о**)**.**  
2. «**В**е**н**е**т**ы** = **с**л**а**в**я**н**е**»** — **у**с**л**о**в**н**о** (**Т**. **С**у**л**и**м**и**р**с**к**и**й**)**.**  
3. «**Б**е**р**л**и**н** = **д**р**е**в**н**е**-**г**е**р**м**а**н**с**к**и**й** г**о**р**о**д**»** — DAMNATA. **Б**е**р**л**и**н** = **с**л**а**в**я**н**с**к**и**й** г**о**р**о**д**.**  
4. «**С**л**а**в**я**н**е** = **н**а**с**е**л**и**л**и** **з**е**м**л**ю** в **VI** в** н**.**э**.**»** — DAMNATA. **С**л**а**в**я**н**е** = **ж**и**л**и** **з**д**е**с**ь** с **V** в** д**о** н**.**э**.** (л**у**г**и**и**)**.**
