### I. ЧТО ЭТО

Полный машинный корпус священных текстов ветвей Архива: **весь текст целиком** (не выдержки) в оригинале — с параллельным русским переводом, с указанием издания и лицензии каждого слоя. Собирается автоматически из открытых достоверных источников; сайт генерируется и публикуется сам (GitHub Actions → GitHub Pages).

### II. ANTI-CONSPIRATIO — ХАРТИЯ ФАЛЬСИФИЦИРУЕМОСТИ

1. **Каждый слой несёт источник и печать.** Нет источника — нет слоя.
2. **Протокол опровержения публикуется** (см. `05-probatio/damnata.md`).
3. **Граница двух корпусов явна.** *Исторический* (миграция и редактура текстов) — доказуем, текстология. *Метафизический* (единый Автор) — нефальсифицируем, теология, печать SUSPECTUS навсегда. Свод документирует, как люди собирали книги, приписываемые Богу.
4. **Закрытый вопрос запрещён.**
