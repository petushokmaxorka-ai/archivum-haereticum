### IV. СТРАТЕГИЯ CONCURRENTIA + NODUS

**Исламская** **стратегия** **CONCURRENTIA** + **NODUS** (**соперничество**/**узел**):

> **«**Ислам** — **«**соперник**»** **христианства** **+** **«**узел**»** **между** **Азией** **и** **Европой**; **двоеверие** **с** **христианством**/**иудаизмом**/**язычеством**; **«**Аллах**/**Мухаммад**/**Коран** = **«**общий** **закон**»**»** — **это** **CONCURRENTIA** **+** **NODUS**.

**Сравнение** **с** **другими** **стратегиями**:
- **Ислам** — **CONCURRENTIA** + **NODUS** (**соперничество** + **узел**)
- **Средиземноморье** — **CONCURRENTIA** (**соперничество**)
- **Буддизм**/**Индуизм**/**Африканское** — **NODUS** (**узел**)
- **Славия** — **REPLICATIO** (**копирование**)
- **Эфиопия**/**Семитское** — **ISOLATIO**/**RESISTENTIA**

### VI. DAMNATA-ПРОТОКОЛ

**DAMNATA (нельзя писать как факт):**

1. «**Ислам** — **«**ересь**»** — DAMNATA. **Ислам** = **«**полноценная** **религия**»** (**«**1.8** **миллиарда** **последователей**»**)

3. «**Мусульмане** — **«**варвары**»** — DAMNATA. **Мусульмане** = **«**полноценная** **цивилизация**»** (**«**Кордова**, **Багдад**, **Каир**»**)
4. «**Аллах** = **«**другой** **бог**»** — DAMNATA. **Аллах** = **«**Ал**-**Илах**»** = **«**Бог**»** (**«**тот** **же** **Бог**»**, **«**Авраамический**»**)
5. «**Мухаммад** = **«**лжепророк**»** — DAMNATA. **Мухаммад** = **«**пророк** **для** **1.8** **миллиарда** **мусульман**»**
6. «**Коран** = **«**подделка**»** — DAMNATA. **Коран** = **«**священная** **книга**»** (**«**VII** **в.**»**)
7. «**Суфизм** = **«**искажение** **ислама**»** — DAMNATA. **Суфизм** = **«**мистическая** **традиция** **ислама**»**
8. «**Халифат** = **«**теократия**»** — DAMNATA. **Халифат** = **«**форма** **правления** **в** **исламе**»** (**«**не** **только** **теократия**»**)
