### IV. СТРАТЕГИЯ REPLICATIO + NODUS

**Восточноевропейская** **стратегия** **REPLICATIO** + **NODUS** (**копирование**/**узел**):

> **«**Славянская** **церковь** — **«**копия**»** **византийской**/**греческой** **+** **«**своя** **азбука**»** **+** **«**свои** **миссионеры**»**; **«**узел**»** **между** **Балтийским** **морем**, **Средиземным** **морем**, **Чёрным** **морем**; **двоеверие** **с** **язычеством**/**католицизмом**/**исламом**; **«**Кирилл**/**Мефодий**/**Стефан** = **«**общий** **закон** **восточноевропейского** **свода**»**»** — **это** **REPLICATIO** **+** **NODUS**.

**Сравнение** **с** **другими** **стратегиями**:
- **Восточноевропейские** — **REPLICATIO** + **NODUS** (**копирование** + **узел**)
- **Скандинавия** — **REPLICATIO** + **NODUS** (**копирование** + **узел**)
- **Средиземноморье** — **CONCURRENTIA** (**соперничество**)
- **Эфиопия**/**Семитское** — **ISOLATIO**/**RESISTENTIA**

### VI. DAMNATA-ПРОТОКОЛ

**DAMNATA (нельзя писать как факт):**

1. «**Славянское** **православие** — **«**варварское**»** — DAMNATA. **Славянское** **православие** = **«**полноценная** **христианская** **традиция**»** (**«**IX** **в.**»**)
2. «**Валахи** — **«**исконные** **римляне**»** — DAMNATA. **Валахи** = **«**слоистый** **этнос**»** (**даки** **+** **римляне** **+** **славяне** **+** **тюрки** **+** **греки**)
3. «**Славянское** **православие** — **«**греческое**»** — DAMNATA. **Славянское** **православие** = **«**своя** **традиция**»** (**«**своя** **азбука**»**, **«**свои** **миссионеры**»**, **«**свои** **святые**»**)
4. «**Кирилл** **и** **Мефодий** — **«**изобретатели**»** — DAMNATA. **Кирилл** **и** **Мефодий** = **«**миссионеры** **и** **переводчики**»** (**«**не** **изобретатели**»**)
5. «**Стефан** **Великий** — **«**румын**»** — DAMNATA. **Стефан** **Великий** = **«**молдавский** **господарь**»** (**«**не** **только** **румын**»**, **«**и** **не** **только** **молдаванин**»**)
6. «**Путна** — **«**монастырь** **Молдавии**»** — DAMNATA. **Путна** = **«**монастырь** **Стефана** **Великого**»** (**«**с** **XIV** **в.**»**)
7. «**Иларион** **Киевский** — **«**первый** **русский** **митрополит**»** — DAMNATA. **Иларион** **Киевский** = **«**миссионер** **и** **просветитель**»** (**«**не** **только** **русский**»**, **«**и** **не** **только** **митрополит**»**)
