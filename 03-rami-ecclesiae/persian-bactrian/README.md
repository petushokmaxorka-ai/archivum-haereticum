### IV. СТРАТЕГИЯ EDITIO + NODUS

**Персидско-**бактрийская** **стратегия** **EDITIO** + **NODUS** (**редакция**/**узел**):

> **«**Зороастризм** — **«**редакция**»** **Авесты** (**«**издание** **священного** **текста**»**); **«**Буддизм** **Гандхары** — **«**узел**»** **между** **Индией**, **Ираном**, **Средней** **Азией**, **Китаем**; **двоеверие** **с** **индуизмом**/**буддизмом**/**христианством**/**исламом**; **«**Ахура**-**Мазда**/**Заратуштра**/**Будда** = **«**общий** **закон**»**»** — **это** **EDITIO** **+** **NODUS**.

**Сравнение** **с** **другими** **стратегиями**:
- **Зороастризм** — **EDITIO** (**редакция**)
- **Буддизм** **Гандхары** — **NODUS** (**узел**)
- **Средиземноморье** — **CONCURRENTIA** (**соперничество**)
- **Славия** — **REPLICATIO** (**копирование**)
