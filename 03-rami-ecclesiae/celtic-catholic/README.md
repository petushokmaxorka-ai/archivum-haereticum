### III. СТРАТЕГИЯ REDITUS + REPLICATIO

**Кельтская** **стратегия** **REDITUS** + **REPLICATIO** (**возврат**/**копирование**):

> **«**Кельтская** **церковь** — **«**возврат**»** **к** **истокам** **христианства** (**«**палестинский** **источник** **через** **Британ**»**); **«**копия**»** **римской**/**палестинской** **традиции**; **двоеверие** **с** **кельтским** **язычеством** (**«**Ту**ат**а** **Де** **Дан**а**нн**»**/**«**Сиды**»**/**«**Священные** **рощи**»**/**«**Священные** **источники**»**); **«**Патрик**/**Брендан**/**Колумба** = **«**общий** **закон** **кельтского** **свода**»**»** — **это** **REDITUS** **+** **REPLICATIO**.

**Сравнение** **с** **другими** **стратегиями**:
- **Гиберния** — **REDITUS** + **REPLICATIO** (**возврат** + **копирование**)
- **Скандинавия** — **REPLICATIO** + **NODUS** (**копирование** + **узел**)
- **Средиземноморье** — **CONCURRENTIA** (**соперничество**)
- **Славия** — **REPLICATIO** (**копирование**)

### V. DAMNATA-ПРОТОКОЛ

**DAMNATA (нельзя писать как факт):**

1. «**Кельтское** **христианство** — **«**ересь**»** — DAMNATA. **Кельтское** **христианство** = **«**полноценная** **христианская** **традиция**»**
2. «**Ирландия** = **«**отсталая** **страна**»** — DAMNATA. **Ирландия** = **«**полноценная** **цивилизация**»** (**«**Книга** **Келлов**»**, **«**Книга** **Диммы**»**)
3. «**Кельтское** **христианство** — **«**примитивное**»** — DAMNATA. **Кельтское** **христианство** = **«**развитая** **монастырская** **традиция**»**
4. «**Святой** **Патрик** = **«**легенда**»** — DAMNATA. **Святой** **Патрик** = **«**исторический** **деятель**»** (**«**V** **в.**»**)
5. «**«**Священные** **рощи**»** — **«**языческое** **суеверие**»** — DAMNATA. **«**Священные** **рощи**»** = **«**общий** **индо**-**европейский** **архетип**»** (**«**друидизм**»**)
6. «**Кельтский** **календарь** — **«**примитивный**»** — DAMNATA. **Кельтский** **календарь** = **«**высокая** **космология**»**
7. «**Брендан** — **«**выдумка**»** — DAMNATA. **Брендан** = **«**историческое** **лицо**»** (**«**V**–**VI** **вв.**»**)
8. «**Колумба** — **«**шотландец**»** — DAMNATA. **Колумба** = **«**ирландец**»** (**«**миссия** **в** **Шотландию**»**)
