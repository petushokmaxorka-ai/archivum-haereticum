Völundarkviða (др.-исл. «Песнь о Вёлунде») — героическая поэма о Вёлунде (Веланде), сыне великана, искусном кузнеце. Вместе с двумя братьями он похищает дочерей короля финнов. Когда братья уезжают, король Нидуд захватывает Вёлунда, перерезает ему сухожилия и оставляет на острове, заставляя ковать для него сокровища. Вёлунд мстит страшной местью: убивает сыновей Нидуда, делает из их черепов кубки, из глаз — драгоценности, из зубов — ожерелье. Затем улетает на волшебном крылатом плаще.
