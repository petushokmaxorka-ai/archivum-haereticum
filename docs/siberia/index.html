Tengris Törösi (Тенгрианство, Небесная Вера) — древнейшая религия тюркских, монгольских и сибирских народов, охватывающая Алтай, Туву, Якутию, Бурятию, Калмыкию, Хакасию. Священное место: гора Белуха (Алтай), гора Бурхан-Халдун (Монголия). Источники:

1. Орхонские надписи (732-735 гг. н.э.) — самые древние тюркские памятники. Бильге-каган и Кюль-тегин, Гёктюркский каганат. Написаны РУНИЦЕЙ (38 знаков). Открыты в 1889 г.

2. Олонхо (Якутский эпос, XIX-XX вв.) — самый длинный эпос в мире (до 200 000 строк). Нюргун Боотур. П.А. Ойунский (1893-1939).

3. Книга Гэсэра (Бурят/Монгол XIII в.) — героический эпос от тибетского происхождения.

4. Кудык-Тэнгри / Верхний мир — пантеон: Тенгри (небо), Умай (земля), Эрлик (подземный мир), Кынгыр (Огонь-хозяйка).

Связь с архивом: Q35 — ЖИВАЯ религия, исповедуется до сих пор (Тува, Якутия, Бурятия, Алтай). Тенгри = Дый/Кök Türk = Индра (Q6) = Зевс (Q33) = Юпитер (Q29) = Один (Q22) = Кок-Болтек (Q21). Эрлик = Аид (Q33) = Яма (Q6) = Хель (Q22) = Осирис-подземный (Q20). Умай (богиня-мать, покровительница рожениц и детей) = Деметра (Q33) = Исида (Q20) = Мария (Q2). Тенгрианство — МАТРИЦА для всех индоевропейских религий, уходящая корнями в общий палеоазиатский/прототуранский пласт.
