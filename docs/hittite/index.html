Hatti / Hittite Empire — древнеанатолийская империя (1600-1178 до н.э.), центр в Хаттусе (совр. Богазкой, Турция). Письменность: КЛИНОПИСЬ (адаптация месопотамской), иероглифический лувийский. Языки: Несили (хеттский, анатолийский диалект индоевропейской семьи), Лувийский, Хурритский (внутрихеттский).

Главные тексты:

1. Цикл Кумарби (CTH 345, 346, 348) — Отец Богов Кумарби проглатывает Ану (Небо), рождает Тешуба (Грозу). = ПРЯМОЙ ИСТОЧНИК греческого Уран → Кронос → Зевс!

2. Миф о Телепину (CTH 324) — Бог земледелия исчезает, всё на земле умирает. = ПРОТОТИП умирающего и воскресающего бога.

3. Миф об Иллуянке (CTH 321) — Борьба Бога Грозы с драконом-змеем.

Главные боги: Тешуб (𒀭𕂁𒌓 Бог Грозы), Хебат (𒀭𒄯𒁍 Солнечная Богиня), Кумарби (𒀭𒈗𒋼 Отец Богов), Телепину (𒀭𒋼𒉌𒁲 Бог Земледелия), Иллуянка (Змей-Дракон), Шаушка (𒀭𒊩 Хурритская богиня).

Связь с архивом: Q40 = НЕДОСТАЮЩЕЕ ЗВЕНО МЕЖДУ ШУМЕРОМ И ГРЕЦИЕЙ! Цикл Кумарби (XIV в. до н.э.) = связующее звено между шумерской Теогонией (Q25, Enuma Elish, ~1100 до н.э.) и греческой Теогонией (Q33, Гесиод, VIII в. до н.э.). Схема передачи: Хурриты (XVII в. до н.э.) → Хетты (XIV в. до н.э.) → Финикия (XIII в. до н.э.) → Греция (Гесиод, VIII в. до н.э.). Это САМЫЙ ВАЖНЫЙ текст для доказательства ПРИМОРДИАЛЬНОЙ ТРАДИЦИИ.
