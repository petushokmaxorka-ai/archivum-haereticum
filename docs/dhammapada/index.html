Дхаммапада (пали: Dhammapada, धम्मपद) — одна из самых известных книг буддийского Канона (Типитаки), входящая в раздел Кхуддака Никая («Малая коллекция») Сутта Питаки. Содержит 423 строфы (стиха), сгруппированных в 26 глав (вагг). Это собрание изречений Будды Шакьямуни.
