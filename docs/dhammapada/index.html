Дхаммапада (пали: धम्मपद, Dharmmapada, «Стопа Закона Будды») — один из самых известных текстов Тхеравады (раннего буддизма). Часть Сутта-питаки (Khuddaka Nikāya), собрание изречений Будды Шакьямуни, сгруппированных по темам.

26 вагг (глав) по темам: парные (1), о бдительности (2), ум (3), цветы (4), глупцы (5), мудрецы (6), мудрецы-просветлённые (7), тысячи (8), зло (9), наказание (10), старость (11), сам (12), мир (13), Будда (14), счастье (15), любимое (16), гнев (17), нечистое (18), правильный (19), путь (20), разное (21), ад (22), слон (23), жажда (24), монах (25), брахман (26).
