### IV. СТРАТЕГИЯ TRANSMISSIO

**Православно-легендарная** **стратегия** **TRANSMISSIO** (**передача**):

> **«**Conversio** — **«**передача**»** **римской**/**баварской** **миссионерской** **традиции** **в** **Карантию**; **двоеверие** **с** **язычеством** (**«**До**-**христианские** **культы** **Славин**»**); **«**Христос** = **«**исполнение** **Закона**»** — **общая** **миссионерская** **логика** **для** **всех** **народов**»** — **это** **TRANSMISSIO**.

**Сравнение** **с** **другими** **стратегиями**:
- **Conversio** — **TRANSMISSIO** (**передача**)
- **Церковь** **Востока** — **TRANSMISSIO** (**передача**)
- **Гностицизм** — **TRANSMISSIO** + **NODUS** (**передача** + **узел**)
- **Апокрифы** — **TRANSMISSIO** + **NODUS** (**передача** + **узел**)
- **Средиземноморье** — **CONCURRENTIA** (**соперничество**)
- **Славия** — **REPLICATIO** (**копирование**)
- **Эфиопия**/**Семитское** — **ISOLATIO**/**RESISTENTIA**

### VI. DAMNATA-ПРОТОКОЛ

**DAMNATA (нельзя писать как факт):**

1. «**Conversio** — **«**латинская** **подделка**»** — DAMNATA. **Conversio** = **«**политический** **трактат**»** (**«**не** **подделка**»**)
2. «**Христианизация** **Славин** **была** **насильственной**»** — DAMNATA. **Христианизация** **Славин** = **«**политический** **и** **культурный** **процесс**»** (**«**не** **только** **насилие**»**)
3. «**Баварцы** = **«**колонизаторы**»** — DAMNATA. **Баварцы** = **«**миссионеры**»** (**«**не** **только** **колонизаторы**»**)
4. «**Князь** **Борут** = **«**предатель**»** — DAMNATA. **Князь** **Борут** = **«**политик**»** (**«**двоеверие**»**)
5. «**Карантия** = **«**пассивный** **объект**»** — DAMNATA. **Карантия** = **«**активный** **участник**»** (**«**двоеверие**»**)
6. «**Крес** = **«**просто** **крещение**»** — DAMNATA. **Крес** = **«**Божественный** **закон**»** (**«**Славянский** **свод**»**)
