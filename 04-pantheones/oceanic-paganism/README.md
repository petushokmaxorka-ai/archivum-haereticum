# OCEANIC-PAGANISM
## Океанийское язычество: айны, анито (филиппинские)
### «Духи анито и медвежий праздник» — от Хоккайдо до Филиппин

**CLASSIFICATIO: ◆ FIDELIS (документальная база: Киндаити Кёсукэ «Айнские предания», Леккас 1934, Эрлих А. «Анито» (Манила 1978), айно-мосинский язык, эпос Бирагорон) / ◇ SUSPECTUS-FERTILIS (айнский миф = палеоазиатский субстрат vs. уральский субстрат — рабочие гипотезы) / ✠ DAMNATUS («Айны — это «отсталое племя» / «Анито — это «примитивный дух» / «Айно-мосинский язык — это «изолированный» — DAMNATA как упрощения) / ✚ CREDO (личная позиция владельца: **стратегия NODUS + ISOLATIO** — айны «изолировались» в Хоккайдо/Курилах/Сахалине/Камчатке, анито «узел» Филиппин и Индонезии; **«закон Божий через Камуй» (айн.) / Anito (филипп.)** — Камуй = «бог медведей»/Творец, анито = «священные предки»; **двоеверие** — синкретизм с буддизмом/синтоизмом/христианством)**
Datum: 022.M3

> *Заявление владельца: **океанийское** **язычество** — **«**Духи** **анито** **и** **медвежий** **праздник**»** (**палеолит** — **XXI** **в.**). **2** **досье**: **ainu** (**айны** — **Хоккайдо**/**Курилы**/**Сахалин**/**Камчатка**), **anito** (**филиппинские** **анито**). **Помним** **цель**: **«**через** **океанийское** **язычество** **увидеть** **«**общий** **закон** **Божий** **через** **Камуй**/**анито**»**.*

### IV. СТРАТЕГИЯ NODUS/ISOLATIO

**Океанийская** **стратегия** **NODUS**/**ISOLATIO** (**узел**/**изоляция**):

> **«**Айны** **изолировались** **в** **Хоккайдо**/**Курилах**/**Сахалине**/**Камчатке**; **анито** **—** **«**узел**»** **Филиппин**, **Индонезии**, **Малайзии**; **оба** **сохранили** **«**общий** **закон** **Божий**»** **через** **«**Камуй**»**/**«**Анито**»** **+** **«**медвежий** **праздник**»** **+** **«**тотемизм**»**»** — **это** **NODUS**/**ISOLATIO**.

- **Шаманизм**/**Буддизм**/**Индуизм**/**Дальневосточное** — **NODUS** (**узел**)

### VI. DAMNATA-ПРОТОКОЛ

**DAMNATA (нельзя писать как факт):**

1. «**Айны** — **«**отсталое** **племя**»** — DAMNATA. **Айны** = **«**полноценная** **культура**»** (**с** **пантеоном**, **мифологией**, **обрядами**)
2. «**Анито** — **«**примитивный** **дух**»** — DAMNATA. **Анито** = **«**священные** **предки**»** (**«**добрые** **духи**»**)
3. «**Айно-**мосинский** **язык** — **«**изолированный**»** — DAMNATA. **Айно-**мосинский** **язык** = **«**палеоазиатский**»** (**гипотеза**)
4. «**Медвежий** **праздник** — **«**ужасный** **языческий** **обряд**»** — DAMNATA. **«**Медвежий** **праздник**»** = **«**священная** **жертва** **+** **причастие**»** (**«**послать** **душу** **медведя** **на** **небо**»**)
5. «**Бахала** — **«**просто** **«**бог**»** — DAMNATA. **Бахала** = **«**Верховный** **бог**»** **с** **«**олитическим** **пантеоном**»** (**Дивата**, **Кабуниан**, **Лакапати**)
6. «**Филиппинское** **язычество** — **«**просто** **«**анимизм**»** — DAMNATA. **Филиппинское** **язычество** = **«**полноценная** **религия**»** (**с** **Бахала**/**Анито**/**Дивата**/**тотемизм**)
