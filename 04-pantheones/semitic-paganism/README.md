# SEMITIC-PAGANISM
## Семитское язычество: самаритяне, караимы, бета-Исраэль, мандеи, друзы, езиды
### «Семитский параллелизм» — от Месопотамии до Эфиопии, от Ханаана до Тигра

**CLASSIFICATIO: ◆ FIDELIS (документальная база: Тора (МТ), Танах, Караимский комментарий, «Гинзе» Самаритянина, «Гаврона» (молитва) Мандеи, «Книга о Нашурах» Друзов, «Книга Откровения» Езидов) / ◇ SUSPECTUS-FERTILIS (реконструкция общесемитского субстрата — рабочая гипотеза) / ✠ DAMNATUS («Иудейская традиция — это просто иудаизм» / «Езиды — это «дьяволопоклонники» / «Друзы — это «еретики-измаилиты» — DAMNATA как упрощения) / ✚ CREDO (личная позиция владельца: **стратегия ISOLATIO + RESISTENTIA** — семитские гетеродоксии «изолировались» (Эфиопия) и «сопротивлялись» (Армения); **«закон Божий через Тетраграмматон»** — YHWH = «общее имя Бога»; **двоеверие** — синкретизм иудаизма с зороастризмом, манихейством, исламом, христианством)**
Datum: 022.M3

> *Заявление владельца: **семитское** **язычество** **в** **архиве** — **это** **«**гетеродоксальные** **ветви** **авраамического** **ствола**»** (**самаритяне**, **караимы**, **бета-**Исраэль**, **мандеи**, **друзы**, **езиды**). **Стратегия** **ISOLATIO**/**RESISTENTIA** (**как** **Эфиопия**/**Армения**). **6** **досье**. **Помним** **цель**: **«**через** **семитское** **язычество** **увидеть** **«**общий** **закон** **Божий** **через** **Тетраграмматон**»**.*

### III. СТРАТЕГИЯ ISOLATIO/RESISTENTIA

**Семитская** **стратегия** **ISOLATIO**/**RESISTENTIA** (**изоляция**/**сопротивление**) — **как** **Эфиопия**/**Армения**:

> **«**Семитские** **гетеродоксии** **«**изолировались**»** (**в** **горах**, **пустынях**, **диаспоре**)** **и**/**или** **«**сопротивлялись**»** (**преследования** **со** **стороны** **ортодоксии**)**; **все** **6** **ветвей** **сохранили** **«**общий** **закон** **Божий**»** **через** **Тетраграмматон** **YHWH** **или** **его** **эквиваленты**»** — **это** **ISOLATIO**/**RESISTENTIA**.

**Сравнение** **с** **другими** **стратегиями**:
- **Самаритяне** — **ISOLATIO** (**изоляция**)
- **Бета-**Исраэль** — **ISOLATIO** (**изоляция**)
- **Езиды** — **ISOLATIO** (**изоляция**)
- **Мандеи** — **RESISTENTIA** (**сопротивление**)
- **Друзы** — **RESISTENTIA** (**сопротивление**)
- **Эфиопия** — **ISOLATIO** (**изоляция**)
- **Армения** — **RESISTENTIA** (**сопротивление**)
- **Средиземноморье** — **CONCURRENTIA** (**соперничество**)
- **Славия** — **REPLICATIO** (**копирование**)
- **Индуизм**/**Буддизм**/**Дальневосточное** — **NODUS** (**узел**)

### V. DAMNATA-ПРОТОКОЛ

**DAMNATA (нельзя писать как факт):**

1. «**Иудейская** **традиция** — **это** **просто** **иудаизм**»** — DAMNATA. **Семитские** **гетеродоксии** — **«**другие** **ветви** **авраамического** **ствола**»**
2. «**Езиды** — **«**дьяволопоклонники**»** — DAMNATA. **Езиды** **почитают** **«**Мелек**-**Тавус**»** (**«**Павлина**-**Ангела**»**), **не** **«**дьявола**»**
3. «**Друзы** — **«**еретики-**измаилиты**»** — DAMNATA. **Друзы** **—** **«**отдельная** **религия**»** (**XI** **в.**, **Хак**-**Дин**)
4. «**Мандеи** — **«**гностики**»** — DAMNATA. **Мандеи** **—** **«**отдельная** **религия**»** (**III** **в.**, **«**Назорейство**»**)
5. «**Самаритяне** — **«**просто** **«**секта**»** — DAMNATA. **Самаритяне** **—** **«**древняя** **традиция**»** (**VIII** **в.** **до** **н.** **э.**, **«**Пятикнижие** **Самаритянина**»**)
6. «**Караимы** — **«**просто** **иудейская** **секта**»** — DAMNATA. **Караимы** **—** **«**отдельная** **теология**»** (**VIII** **в.**, **«**Только** **Тора**»**)
