**Сравнение** **с** **другими** **стратегиями**:
- **Американское** — **NODUS** (**узел**)
- **Буддизм**/**Индуизм**/**Дальневосточное**/**Шаманизм**/**Африканское** — **NODUS** (**узел**)

### VII. DAMNATA-ПРОТОКОЛ

**DAMNATA (нельзя писать как факт):**

1. «**Ацтеки** — **«**примитивное** **племя**»** — DAMNATA. **Ацтеки** = **«**высокоразвитая** **цивилизация**»** (**календарь**, **архитектура**, **астрономия**)
2. «**Аяуаска** — **«**наркотик**»** — DAMNATA. **Аяуаска** = **«**шаманский** **ритуал**»** (**«**церковь** **священной** **лианы**»**)
3. «**Тихоокеанские** **мифы** — **«**фольклор**»** — DAMNATA. **Тихоокеанские** **мифы** = **«**полноценная** **мифология**»** (**«**Тангароа**»**, **«**Мауи**»**)
4. «**«**Пять** **Солнц**»** — **«**примитивная** **космология**»** — DAMNATA. **«**Пять** **Солнц**»** = **«**глубокая** **космология**»** (**каждый** **цикл** = **«**гибель** **и** **возрождение**»**)
5. «**Человеческие** **жертвы** — **«**дикость**»** — DAMNATA. **Человеческие** **жертвы** = **«**ритуал** **жертвоприношения**»** (**как** **тризна** **у** **славян**)
6. «**Кетцалькоатль** — **«**историческое** **лицо**»** — DAMNATA. **Кетцалькоатль** = **«**культурный** **герой**»**/**«**Ушедший** **бог**»** (**как** **Ярило**/**Христос**)
7. «**«**Землекоп**»** (**Нанабожо**) = **«**просто** **индейский** **миф**»** — DAMNATA. **«**Землекоп**»** = **«**ареальный** **мотив** **Сибирь**→**Америка**»** (**concordantia.md** **слот 16**)
8. «**Виракоча** — **«**просто** **творец**»** — DAMNATA. **Виракоча** = **«**Ушедший** **бог**»** (**«**странник**»**), **«**обещал** **вернуться**»** (**concordantia.md** **слот 10** **DEUS** **ABSCEDENS**)
