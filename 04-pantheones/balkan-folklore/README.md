# BALKAN-FOLKLORE
## Балканский фольклор: страгои, вампиры, русалии
### «Страшный уголок Балкан» — от Трансильвании до Греции, от вампиров до вил

**CLASSIFICATIO: ◆ FIDELIS (документальная база: «Дом лесного мертвеца» Flückiger 1862, Folklorist magazine 1890, Frazer «The Golden Bough» 1890, манускрипт Калоянноса 1733 о «Случае вампира Петра Благоева» в Сербии) / ◇ SUSPECTUS-FERTILIS (византийский vs. славянский vs. фракийский vs. дакийский субстрат — рабочие гипотезы) / ✠ DAMNATUS («Страгои = вампир Дракулы» / «Вампиры — это выдумка» / «Балканский фольклор — это просто суеверие» — DAMNATA как упрощения) / ✚ CREDO (личная позиция владельца: **стратегия NODUS + TRANSMISSIO** — Балканы = «узел» между Востоком и Западом, «передача» через Церковь Востока; **«закон Божий через души предков»** — страгои = «злой предок», вампир = «неупокоенный предок», вилы = «духи предков»; **двоеверие** — синкретизм язычества с православием/католицизмом/исламом)**
Datum: 022.M3

### III. СТРАТЕГИЯ NODUS/TRANSMISSIO

**Балканская** **стратегия** **NODUS**/**TRANSMISSIO** (**узел**/**передача**):

> **«**Балканы** **были** **«**узлом**»** **между** **Византией** **и** **Славянией**, **«**передавая**»** **через** **Церковь** **Востока** **и** **Богомильство**; **страгои**/**вампиры**/**русалии** **=** **«**общий** **некромантический** **субстрат**»** **Балкан**; **двоеверие** **с** **православием**/**католицизмом**/**исламом** **сохранило** **языческий** **субстрат** **до** **XXI** **в.**»** — **это** **NODUS**/**TRANSMISSIO**.

**Сравнение** **с** **другими** **стратегиями**:
- **Балканы** — **NODUS**/**TRANSMISSIO** (**узел**/**передача**)
- **Церковь** **Востока** — **TRANSMISSIO** (**передача**)
- **Бактрия**/**Буддизм**/**Индуизм**/**Дальневосточное**/**Шаманизм**/**Африканское**/**Американское**/**Океанийское**/**Новые** **религии** — **NODUS** (**узел**)
- **Средиземноморье** — **CONCURRENTIA** (**соперничество**)
- **Славия** — **REPLICATIO** (**копирование**)
- **Эфиопия**/**Семитское** — **ISOLATIO**/**RESISTENTIA**

### V. DAMNATA-ПРОТОКОЛ

**DAMNATA (нельзя писать как факт):**

1. «**Страгои** = **«**вампир** **Дракулы**»** — DAMNATA. **Страгои** = **«**злой** **предок**»**, **вампир** **=** **«**неупокоенный** **мёртвый**»**; **Дракула** **=** **«**фольклорный** **персонаж**»** (роман** **Стокера** **1897**)
2. «**Вампиры** — **«**выдумка**»** — DAMNATA. **Вампиры** = **«**общий** **балканский** **архетип**»** (страгои**/**упирь**/**стрыга**/**морои**)
3. «**Балканский** **фольклор** — **«**просто** **суеверие**»** — DAMNATA. **Балканский** **фольклор** = **«**глубокая** **архаика**»** (**сохранившая** **«**пра**-**индоевропейский** **субстрат**»**)
4. «**Вилы** = **«**русалки**»** — DAMNATA. **Вилы** = **«**священные** **духи** **предков**»** (**«**не** **русалки**»**)
5. «**Морa** = **«**Морена**»** — DAMNATA. **Морa** = **«**злая** **богиня** **смерти**»** (балканский** **вариант** **Морены**)
6. «**Души** **предков** = **«**языческое** **суеверие**»** — DAMNATA. **Души** **предков** = **«**общий** **архетип**»** (христианский** **«**поминовение** **усопших**»**)
