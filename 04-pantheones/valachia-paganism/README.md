### VIII. СТРАТЕГИЯ NODUS

**Валашская** **стратегия** **NODUS** (**узел**):

> **«**Валашское** **язычество** **—** **«**узел**»** **между** **Дакией**, **Римом**, **Славией**, **Византией**, **Тюркией**; **доказательство** **слоистости** **этноса**; **DAMNATA** **к** **националистическим** **прочтениям** **—** **«**валахи** **=** **истинные** **римляне**»**/**«**истинные** **славяне**»**/**«**истинные** **даки**»**; **это** **NODUS** **+** **двоеверие**»**.

**Сравнение** **с** **другими** **стратегиями**:
- **Валахия** — **NODUS** (**узел**)
- **Балканы** — **NODUS**/**TRANSMISSIO** (**узел**/**передача**)
- **Средиземноморье** — **CONCURRENTIA** (**соперничество**)
- **Славия** — **REPLICATIO** (**копирование**)
- **Эфиопия**/**Семитское** — **ISOLATIO**/**RESISTENTIA**

### X. DAMNATA-ПРОТОКОЛ

**DAMNATA (нельзя писать как факт):**

1. «**Валахи** = **«**истинные** **римляне**»** — DAMNATA. **Валахи** = **«**слоистый** **этнос**»** (**даки** + **римляне** + **славяне** + **тюрки** + **греки**)
2. «**Валахи** = **«**истинные** **славяне**»** — DAMNATA. **Валахи** = **«**слоистый** **этнос**»** (**даки** + **римляне** + **славяне** + **тюрки** + **греки**)
3. «**Валахи** = **«**истинные** **даки**»** — DAMNATA. **Валахи** = **«**слоистый** **этнос**»** (**даки** + **римляне** + **славяне** + **тюрки** + **греки**)
4. «**Уреке** = **«**исконно** **румынский** **историк**»** — DAMNATA. **Уреке** = **«**молдавский** **князь**-**историк** **с** **церковнославянскими** **вкраплениями**»**
5. «**Костин** = **«**чисто** **румынский** **историк**»** — DAMNATA. **Костин** = **«**молдавский** **боярин**-**историк** **с** **признанием** **славянского** **пласта**»**
6. «**Zmeu** = **«**румынский** **дракон**»** — DAMNATA. **Zmeu** = **«**общий** **индоевропейский** **змей**»** (**ср.** **«**индийский** **Нага**»**, **«**греческий** **Пифон**»**, **«**славянский** **Змей**»**)
