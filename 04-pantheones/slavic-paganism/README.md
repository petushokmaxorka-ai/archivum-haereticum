### II. СТРАТЕГИЯ REPLICATIO

**Славянская** **стратегия** **REPLICATIO** (копирование) — **см.** **`00-manuscriptum-principale/concordantia.md`** **§V** **«Tabula** **Strategiarum»**:

> **«**Славяне** **берут** **византийскую** **модель** **(Кирилл** **и** **Мефодий**)** **и** **копируют** **её** **в** **свой** **язык** **(глаголица** → **кириллица**)**, **затем** **распространяют** **дальше** **(Болгария** → **Сербия** → **Русь**)**, **а** **через** **Богомилов** **(болгарская** **ересь** **X** **в.**)** **распространяют** **в** **Европу** **(Катары** → **Альбигойцы**)**»** — **это** **REPLICATIO** (копирование).

**Ветви** **REPLICATIO**:
- **Славия** (кириллица, Богомилы)
- **Церковь** **Востока** (**TRANSMISSIO** — **«**передача**»**)
- **Гиберния** (**REDITUS** — **«**возврат**»**)

**Сравнение** **с** **другими** **стратегиями**:
- **ISOLATIO** (Эфиопия) — **«**изоляция**»** (**геэз** **как** **священный** **язык**)
- **RESISTENTIA** (Армения) — **«**сопротивление**»** (**армянский** **алфавит** **Месропа** **Маштоца**)
- **EDITIO** (Зороастризм) — **«**издание**»** (**Маздакиты** **V** **в.**)
- **NODUS** (Бактрия) — **«**узел**»** (**Христианство** **Гандхары**, **пакт** **411**)
- **CONCURRENTIA** (Аль-Андалус) — **«**соперничество**»** (**Тондражийцы**)
