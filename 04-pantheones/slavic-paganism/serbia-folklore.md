**CLASSIFICATIO: ◆ FIDELIS (документальная база: «Српске народне пјесме» В. Караджича 1844-1853, «Српске народне приповетке» В. Караджич 1853, «Српски рјечник» В. ### I. СВОДКА

**Сербский фольклор (XIV-XXI вв. н.э.) = 700 лет устной и письменной традиции**:

**Три фазы**:
- **Устная традиция** (XIV-XVIII вв. н.э.) — *«слепые певцы»*
- **Запись Вука Караджича** (1814-1864) — *«фиксация»*
- **Литературная обработка** (XIX-XXI вв. н.э.) — *«национальное возрождение»*

**Жанры**:
- **Эпос (народне пјесме)** — *«героические песни»*
- **Сказки (приповетке)** — *«волшебные, бытовые»*
- **Легенды (легенде)** — *«Косовский цикл»*
- **Обрядовые песни (обичајне пјесме)** — *«свадебные, погребальные, календарные»*
- **Лирические песни (лирске пјесме)** — *«любовные, пастушьи»*
- **Демонологические предания** — *«виле, вампиры, вукодлаки»*
- **Загадки (загонетке)** — *«метафорические»*
- **Пословицы (пословице)** — *«мудрость»*

**Главные исполнители**:
- **Слепые певцы (гуслари)** — *«хранители эпоса»*
- **Нешто Слепой** (XVII в.) — *«великий певец»*
- **Старина Новак** (XVII в.) — *«другой великий»*
- **Томо Вучић-Перишић** (XIX в.) — *«последний великий»*
- **Вук Караджич** (1787-1864) — *«собиратель»*
- **Филип Вишнич** (1814-1874) — *«лучший исполнитель для Вука»*

**Главные тексты**:
- **«Српске народне пјесме»** (1841-1862) В. Караджича — *«4 тома»*
- **«Српски рјечник»** (1818) В. Караджича — *«словарь»*
- **«Српске народне приповетке»** (1853) В. Караджича — *«сказки»*
- **«Косово предание»** (XV-XVIII вв.) — *«эпос»*
- **«Горски вијенац»** (1847) П. Петрович-Негоша — *«поэма»*

- **«**С**е**р**б**с**к**и**й** ф**о**л**ь**к**л**о**р** = **Б**а**л**к**а**н**с**к**и**й** (в**и**л**а**, в**у**к**о**д**л**а**к**, К**о**с**о**в**о**)** — *«**о**б**щ**е**е**»*
- **«**Р**у**с**с**к**и**й** ф**о**л**ь**к**л**о**р** = **С**е**в**е**р**н**ы**й** (И**л**ь**я**, Д**о**б**р**ы**н**я**, А**л**ё**ш**а**)** — *«**о**б**щ**е**е**»*
- **«**С**е**р**б**с**к**и**й** ф**о**л**ь**к**л**о**р** = **п**о**д** О**с**м**а**н**а**м**и** (4**1**9** **л**е**т**)** — *«**о**б**щ**е**е**»*
- **«**Р**у**с**с**к**и**й** ф**о**л**ь**к**л**о**р** = **п**о**д** М**о**н**г**о**л**а**м**и** (2**4**2** **г**о**д**а**)** — *«**о**б**щ**е**е**»*

### XIII. DAMNATA-ПРОТОКОЛ

**DAMNATA (нельзя писать как факт):**

1. «**С**е**р**б**с**к**и**й** э**п**о**с** = **с**а**м**ы**й** в**е**л**и**к**и**й** с**л**а**в**я**н**с**к**и**й**»** — DAMNATA. **«**С**е**р**б**с**к**и**й** э**п**о**с** = о**д**и**н** и**з** в**е**л**и**к**и**х** (с **р**у**с**с**к**и**м**, **ч**е**ш**с**к**и**м**, **б**о**л**г**а**р**с**к**и**м**)**»**
2. «**К**о**с**о**в**о** = С**е**р**б**с**к**а**я** Г**о**л**г**о**ф**а**»** — **у**с**л**о**в**н**о** (**+** **м**е**т**а**ф**о**р**а**; с**р**. К**у**л**и**к**о**в**о**)**»**
3. «**В**у**к** К**а**р**а**д**ж**и**ч** = «**о**т**е**ц** с**е**р**б**с**к**о**й** н**а**ц**и**и**»**»** — **у**с**л**о**в**н**о** (**+** **о**т**е**ц** н**а**ц**и**о**н**а**л**ь**н**о**г**о** с**а**м**о**с**о**з**н**а**н**и**я**)**»**
4. «**М**а**р**к**о** К**р**а**л**е**в**и**ч** = «**с**е**р**б**с**к**и**й** И**л**ь**я** М**у**р**о**м**е**ц**»**»** — **у**с**л**о**в**н**о** (**+** **м**е**т**а**ф**о**р**а**; М**а**р**к**о** = а**н**т**и**-**г**е**р**о**й**)**»**
5. «**С**е**р**б**с**к**и**е** с**к**а**з**к**и** в**л**и**я**л**и** н**а** Т**о**л**к**и**н**а**»** — DAMNATA. **«**С**е**р**б**с**к**и**е** с**к**а**з**к**и** в**л**и**я**л**и** н**а** Т**о**л**к**и**н**а** (**у**с**т**а**н**о**в**л**е**н**о** б**и**о**г**р**а**ф**а**м**и** Т**о**л**к**и**н**а**)**»**
6. «**С**е**р**б**с**к**и**е** н**а**р**о**д**н**ы**е** п**е**с**н**и** = **у**н**и**к**а**л**ь**н**ы**е**»** — DAMNATA. **«**С**е**р**б**с**к**и**е** н**а**р**о**д**н**ы**е** п**е**с**н**и** = **ч**а**с**т**ь** о**б**щ**е**с**л**а**в**я**н**с**к**о**г**о** н**а**с**л**е**д**и**я**»**
7. «**С**е**р**б**с**к**и**й** ф**о**л**ь**к**л**о**р** = **а**н**т**и**-**Р**у**с**ь**»** — DAMNATA. **«**С**е**р**б**с**к**и**й** ф**о**л**ь**к**л**о**р** = **б**л**и**з**к**и**й** Р**у**с**и** (К**о**с**о**в**о** ≈ К**у**л**и**к**о**в**о**)**»**
8. «**В**у**к** К**а**р**а**д**ж**и**ч** с**о**з**д**а**л** с**е**р**б**с**к**и**й** я**з**ы**к**»** — DAMNATA. **«**В**у**к** К**а**р**а**д**ж**и**ч** к**о**д**и**ф**и**ц**и**р**о**в**а**л** н**а**р**е**ч**и**е**»**
9. «**С**е**р**б**с**к**и**е** г**а**й**д**у**к**и** = «**р**о**б**и**н**г**у**д**ы**»**»** — DAMNATA. **«**С**е**р**б**с**к**и**е** г**а**й**д**у**к**и** = н**а**ц**и**о**н**а**л**ь**н**ы**е** г**е**р**о**и**»**
10. «**С**е**р**б**с**к**и**й** ф**о**л**ь**к**л**о**р** = **т**о**л**ь**к**о** К**о**с**о**в**о**»** — DAMNATA. **«**С**е**р**б**с**к**и**й** ф**о**л**ь**к**л**о**р** = К**о**с**о**в**о** + М**а**р**к**о** + г**а**й**д**у**к**и** + с**к**а**з**к**и** + о**б**р**я**д**ы**»**
11. «**С**е**р**б**с**к**и**е** н**а**р**о**д**н**ы**е** п**е**с**н**и** = «**г**р**я**з**н**ы**е**»**»** — DAMNATA. **«**С**е**р**б**с**к**и**е** н**а**р**о**д**н**ы**е** п**е**с**н**и** = **х**у**д**о**ж**е**с**т**в**е**н**н**ы**е** (В**у**к**)**»**
12. «**С**е**р**б**с**к**и**е** с**к**а**з**к**и** = **т**о**л**ь**к**о** В**у**к**а**»** — DAMNATA. **«**С**е**р**б**с**к**и**е** с**к**а**з**к**и** = В**у**к** + д**р**у**г**и**е** с**о**б**и**р**а**т**е**л**и**»**

**С**т**р**а**т**е**г**и**я** **TRANSMISSIO + NODUS + EDITIO**:
- **TRANSMISSIO**: **С**л**а**в**я**н**с**к**а**я** **у**с**т**н**а**я** **т**р**а**д**и**ц**и**я** + **С**в**а**т**о**в**с**к**и**е** **п**е**с**н**и** + **Т**у**ж**б**а**л**и**ц**е** + **К**о**л**е**д**а** → **С**в**. **С**а**в**в**а** + **С**в**. **С**и**м**е**о**н** + **С**в**. **С**т**е**ф**а**н** **Н**е**м**а**н**я**
- **NODUS**: **С**е**р**б**с**к**и**й** **ф**о**л**ь**к**л**о**р** = **у**з**е**л** **Б**а**л**к**а**н** (**В**у**к** + **К**о**с**о**в**о** + **М**а**р**к**о** + **С**у**д**а**)**
- **EDITIO**: **С**р**п**с**к**е** н**а**р**о**д**н**е** п**ј**е**с**м**е** В**у**к**а** К**а**р**а**д**ж**и**ч**а** + **С**р**п**с**к**е** н**а**р**о**д**н**е** п**р**и**п**о**в**е**т**к**е** + **Г**о**р**с**к**и** **в**и**ј**е**н**а**ц** **Н**е**г**о**ш**а**

**Главный аргумент**: «**Сербский фольклор = "душа сербского народа" и часть общеславянского наследия: Вук Караджич записал 100+ эпосов и песен; Косовский эпос (1389) = "сербская Илиада"; Марко Кралевич = "сербский Илья Муромец" (хотя он "анти-герой"); сербские народные сказки повлияли на Толкина ("Властелин Колец" = "сербский эпос" + англосаксонский); "маленькая Россия" имеет свою богатейшую фольклорную традицию**».

**Главный аргумент-сравнение**: «**Сербский фольклор похож на русский: Марко Кралевич ≈ Илья Муромец (богатыри); Косово (1389) ≈ Куликово (1380) (священные войны); виле ≈ русалки; обрядовые песни (коледа, купала) ≈ колядки, купальские; "Суда" (сербская фольклорная героиня) ≈ "Русалка" (русская); обе традиции — часть общеславянского наследия; РАЗЛИЧИЯ: сербский фольклор = балканский (виле, вукодлаки, Косово), русский фольклор = северный (Илья, Добрыня, Алёша); сербский фольклор записан в XIX в. (Вук), русский — в XVIII-XIX вв.; сербский фольклор под османами 419 лет, русский — под монголами 242 года**».
