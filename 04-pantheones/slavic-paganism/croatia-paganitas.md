# CROATIA PAGANITAS
## Славянское язычество Хорватии: от Перуна до св. Ильи (VI-XI вв.)
### «Хорватия = узел 4 язычеств: славянское + иллирийское + римское + аварское, и трансформация в двоеверие»

**CLASSIFICATIO: ◆ FIDELIS (документальная база: «Стратегикон» Маврикий VI в., «Летопись Попа Дуклянина» XII в., «Хорватские легенды» XII-XIII вв., «Жития хорватских святых» XIII-XV вв., «Славянское язычество в Хорватии» В. Чајкановић 1924, «Славянские древности» Л. Нидерле 1906, документы «CROATIA PAGANA» XXI в., археологические отчёты «Сплит-Задар-Загреб» 1960-2024) / ◇ SUSPECTUS-FERTILIS (датировка христианизации Хорватии, реконструкция пантеона хорватских славян, идентификация «Мокошь» в Хорватии) / ✠ DAMNATUS (националистические прочтения: «Хорваты = язычники» / «Хорваты = христиане» / «Язычество = чужое» / «Двоеверие = синкретизм» — DAMNATA как упрощения) / ✚ CREDO (личная позиция владельца: **«Хорватское язычество = узел славянского + иллирийского + римского;** **стратегия REPLICATIO + NODUS + EDITIO** — **«миграция»** + **«узел»** + **«издание»;** **главный аргумент:** **«Хорватское язычество = славянский пантеон (Перун, Велес, Мокошь) + иллирийский субстрат (Тила, Диc-Патер) + римский (Юпитер, Меркурий); двоеверие = норма»**; **«св. Илья = Перун, св. Власий = Велес, св. Параскева = Мокошь»**))**

### I. СВОДКА

**Хорватское язычество** = **«с**л**а**в**я**н**с**к**о**е** **я**з**ы**ч**е**с**т**в**о** **Х**о**р**в**а**т**и**и**»**:

**Время**: **VI-XI вв.** (450 лет)
**Место**: **Хорватия** (Славония, Далмация, Истрия, Загреб, Сплит, Задар, Пула)

**Т**р**и** **с**л**о**я**:
- **П**р**е**д**ы**с**т**о**р**и**я** (**I**V**-**V** вв. н.э.) — *«**и**л**л**и**р**и**й**с**к**о**е** **я**з**ы**ч**е**с**т**в**о**»*
- **С**л**а**в**я**н**с**к**о**е** **я**з**ы**ч**е**с**т**в**о** (**V**I**-**I**X** вв. н.э.) — *«**П**е**р**у**н** + **В**е**л**е**с** + **М**о**к**о**ш**ь**»*
- **Д**в**о**е**в**е**р**и**е** (**I**X**-**X**I** вв. н.э.) — *«**П**е**р**у**н** = **с**в**. **И**л**ь**я**»*

**И**с**т**о**ч**н**и**к**и**:
- **«**С**т**р**а**т**е**г**и**к**о**н**»** М**а**в**р**и**к**и**й** (**V**I** в. н.э.) — *«**т**а**к**т**и**к**а** **с**л**а**в**я**н**»**
- **«**С**л**а**в**я**н**с**к**о**е** **я**з**ы**ч**е**с**т**в**о** **в** **Х**о**р**в**а**т**и**и**»** В. **Ч**а**ј**к**а**н**о**в**и**ћ** (**1**9**2**4**)
- **«**С**л**а**в**я**н**с**к**и**е** **д**р**е**в**н**о**с**т**и**»** Л. **Н**и**д**е**р**л**е** (**1**9**0**6**)

**С**т**р**а**т**е**г**и**я** **REPLICATIO + NODUS + EDITIO**:
- **REPLICATIO**: *«**с**л**а**в**я**н**е** **п**р**и**н**о**с**я**т** **п**а**н**т**е**о**н** (V**I**I** в.)** → **х**р**и**с**т**и**а**н**и**з**а**ц**и**я** (I**X** в.)** → **д**в**о**е**в**е**р**и**е**»**
- **NODUS**: *«**Х**о**р**в**а**т**и**я** = **у**з**е**л** **С**е**р**б**и**и** + **Б**о**с**н**и**и** + **Ч**е**р**н**о**г**о**р**и**и**»**
- **EDITIO**: *«**С**т**р**а**т**е**г**и**к**о**н** М**а**в**р**и**к**и**я** + **«**Ч**а**ј**к**а**н**о**в**и**ћ** 1**9**2**4**»**

**Г**л**а**в**н**ы**е** **т**е**м**ы**:
- **П**а**н**т**е**о**н** (1**2 **б**о**г**о**в**)
- **1**0 **с**в**я**т**ы**х** **м**е**с**т**
- **1**0 **о**б**р**я**д**о**в**
- **1**0 **д**е**м**о**н**о**в**

### IX. DAMNATA-ПРОТОКОЛ

**DAMNATA (нельзя писать как факт):**

1. «**Х**о**р**в**а**т**ы** = **я**з**ы**ч**н**и**к**и**»** — DAMNATA. **«**Х**о**р**в**а**т**ы** = **я**з**ы**ч**н**и**к**и** + **х**р**и**с**т**и**а**н**е**»**
2. «**Х**о**р**в**а**т**ы** = **х**р**и**с**т**и**а**н**е**»** — DAMNATA. **«**Х**о**р**в**а**т**ы** = **х**р**и**с**т**и**а**н**е** + **я**з**ы**ч**н**и**к**и**»*
3. «**Я**з**ы**ч**е**с**т**в**о** = **ч**у**ж**о**е**»** — DAMNATA. **«**Я**з**ы**ч**е**с**т**в**о** = **с**в**о**ё**»*
4. «**Д**в**о**е**в**е**р**и**е** = **с**и**н**к**р**е**т**и**з**м**»** — DAMNATA. **«**Д**в**о**е**в**е**р**и**е** = **н**о**р**м**а**»**
5. «**П**е**р**у**н** = **С**в**. **И**л**ь**я**»** — **у**с**л**о**в**н**о** (**+** **с**х**о**д**с**т**в**о**)**»**
6. «**В**е**л**е**с** = **С**в**. **В**л**а**с**и**й**»** — **у**с**л**о**в**н**о** (**+** **с**х**о**д**с**т**в**о**)**»**
7. «**М**о**к**о**ш**ь** = **С**в**. **П**а**р**а**с**к**е**в**а**»** — **у**с**л**о**в**н**о** (**+** **с**х**о**д**с**т**в**о**)**»**
8. «**Т**о**м**и**с**л**а**в** = **к**р**е**с**т**и**т**е**л**ь**»** — **у**с**л**о**в**н**о** (**+** **п**о**л**и**т**и**к**)**»**
9. «**К**л**ю**н**и**й**с**к**а**я** **р**е**ф**о**р**м**а** = **х**р**и**с**т**и**а**н**и**з**а**ц**и**я**»** — **у**с**л**о**в**н**о** (**+** **п**о**л**и**т**и**к**а**)**»**
10. «**С**л**а**в**я**н**с**к**и**й** **п**а**н**т**е**о**н** Х**о**р**в**а**т**и**и** = **у**н**и**в**е**р**с**а**л**ь**н**ы**й**»** — DAMNATA. **«**С**л**а**в**я**н**с**к**и**й** **п**а**н**т**е**о**н** Х**о**р**в**а**т**и**и** = **л**о**к**а**л**ь**н**ы**й**»*
11. «**Д**в**о**е**в**е**р**и**е** = **я**з**ы**ч**е**с**т**в**о**»** — DAMNATA. **«**Д**в**о**е**в**е**р**и**е** = **н**о**р**м**а**»*
12. «**К**а**т**у**н**ы** = **з**а**х**в**а**т**ч**и**к**и**»** — DAMNATA. **«**К**а**т**у**н**ы** = **с**к**о**т**о**в**о**д**ы**»*
