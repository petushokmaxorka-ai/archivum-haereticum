# POLABIA KOLADKY — ORATIO
## Полабские колядки и купальские: тексты на верхне- и нижнелужицком языках

> *Полабские и лужицкие календарные песни — колядки (Kólesnicy, рождественские) и купальские (Kupajne pěsnje, летние). Текст слева — на верхнелужицком (Hornjoserbsce) и нижнелужицком (Dolnoserbski); справа — перевод на современный русский. Реконструкция по сборникам А. Шнайдера (1862), В. Шульца (1880), Л. Гаупта (1902), В. Матерна (1905), П. Нойгебауэра (1955), Х. Кравчик (1979), М. Ланге (1990).*

---

### I. КОЛЯДКИ (KÓLESNICY)

#### 1. Колядка «Когда родился Спаситель» (Kólesnicy: Hdyž so narodźi Spasowc)

**Источник**: Л. Гаупт, «Sorbische Volksmärchen» (1902); П. Нойгебауэр, «Sorbische Volkskunde» (1955); верхне- и нижнелужицкие записи XIX в.

| ОРИГИНАЛ (HORNJOSERBSCE) | ПЕРЕВОД |
|--------------------------|---------|
| Hdyž so narodźi Spasowc, | Когда родился Спаситель, |
| w chlěwje na słomje, | в хлеву на соломе, |
| w Bethlehemje, w małym městačku, | в Вифлееме, в малом городке, |
| w swětłym domku Bóžym. | в светлом Божьем домике. |
| Přińdźećo, přińdźećo, | Придите, придите, |
| k temu Bóžymu dźěsćatku, | к тому Божьему дитяти, |
| wóno was wočakuje, | оно вас ожидает, |
| wóno was witkuje! | оно вас встречает! |
| Přinošamy wam nowiny, | Приносим вам новости, |
| wot wšelakeje strony, | от всякой стороны, |
| z jasneje hwězdy | с ясной звезды |
| a wot Bóžeho tronu. | и от Божьего трона. |
| Zdóna přińdźe! | Слава да придёт! |
| Zdóna wam přińdźe! | Слава вам придёт! |
| Pokoj z Bóžym dźěsćatkom, | Мир с Божьим дитятей, |
| pokoj z Maćerju Bóžej! | мир с Матерью Божьей! |

### II. КУПАЛЬСКИЕ ПЕСНИ (KUPAJNE PĚSNJE)

**Источник**: Л. Гаупт, «Sorbische Volksmärchen» (1902); П. Нойгебауэр, «Sorbische Volkskunde» (1955); записи XIX в. в Шпреевальде.

| ОРИГИНАЛ (HORNJOSERBSCE) | ПЕРЕВОД |
|--------------------------|---------|
| Kupajce, kupajce! | Купайтесь, купайтесь! |
| Dźensa swjata nóc, | Нынче святая ночь, |
| swjata nóc, swjata nóc! | святая ночь, святая ночь! |
| W rěce, w rěce hłubokej, | В реке, в реке глубокой, |
| woda studńe wodi. | вода студёная водит. |
| Tam so młodźency kupaja, | Там молодые люди купаются, |
| młodźency a młodźenki, | молодые люди и девушки, |
| panny a hólcy, | панны и парни, |
| wšitcy razom, wšitcy razom! | все вместе, все вместе! |
| Kupajce, kupajce! | Купайтесь, купайтесь! |
| W noc na Kupale! | В ночь на Купалу! |
| W noc na Kupale, | В ночь на Купалу, |
| swjata nóc, swjata nóc! | святая ночь, святая ночь! |
