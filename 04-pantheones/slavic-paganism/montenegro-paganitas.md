### II. ПАНТЕОН — ГЛАВНЫЕ БОГИ

**1. РОД («Всевышний Дед»):
- **«**Р**о**д**»** = **«**р**о**д** = **с**е**м**ь**я**»** (**с**л**а**в**я**н**с**к**о**е**)
- **«**С**л**о**в**о** **И**о**а**н**н**а** **З**л**а**т**о**у**с**т**а**»** (**X**I**I** **в.**) — *«**Р**о**д** с**у**щ**е**с**т**в**у**е**т** **в**о** **в**с**ё**м**»*
- **Ч**е**р**н**о**г**о**р**с**к**а**я** **р**а**з**н**о**в**и**д**н**о**с**т**ь**:** **«**Р**о**д** = **«**С**т**а**р**ы**й** **Б**о**г**»** (в **Л**е**т**о**п**и**с**ц**е** П**о**п**а** **Д**у**к**л**я**н**и**н**а**: **«**С**в**е**т**о**п**е**к** = **«**С**в**е**т** + **п**е**к** = **о**т**е**ц**»**)*
- **С**в**я**т**ы**е** **м**е**с**т**а**:** **д**у**б**о**в**ы**е** **р**о**щ**и** (С**к**у**т**а**р**и**), **г**о**р**ы** (Л**о**в**ч**е**н**), **к**а**т**у**н**ы**

**2. СВАРОГ («Небесный кузнец»):
- **«**С**в**а**р**о**г**»** = **«**с**в**а**р** = **к**у**з**н**е**ц**»** (**с**л**а**в**я**н**с**к**о**е**)
- **«**С**л**о**в**о** **н**е**к**и**е**х** **с**т**а**р**ц**е**в**»** (**X**I**I** **в.**) — *«**С**в**а**р**о**г** — **б**о**г**-**к**у**з**н**е**ц**»*
- **Ч**е**р**н**о**г**о**р**с**к**а**я** **р**а**з**н**о**в**и**д**н**о**с**т**ь**:** **С**в**а**р**о**г** = **к**у**з**н**е**ц** (г**л**а**в**н**ы**й** **с**е**л**и**т**е**б**н**ы**й** **к**у**з**н**е**ц** Д**у**к**л**ь**а**)*
- **С**в**я**т**ы**е** **м**е**с**т**а**:** **к**у**з**н**и**ц**ы** (г. **К**о**т**о**р**, **г. **Б**у**д**в**а**)

**3. МОКОШЬ («Великая богиня»):
- **«**М**о**к**о**ш**ь**»** = **«**м**о**к**о**ш**ь** = **п**р**я**д**и**л**ь**щ**и**ц**а**»** (**с**л**а**в**я**н**с**к**о**е**)
- **«**П**о**в**е**с**т**ь** **в**р**е**м**е**н**н**ы**х** **л**е**т**»** (**9**8**0** **г.**) — *«**М**о**к**о**ш**ь** — **б**о**г**ы**н**я**»*
- **Ч**е**р**н**о**г**о**р**с**к**а**я** **р**а**з**н**о**в**и**д**н**о**с**т**ь**:** **М**о**к**о**ш**ь** = **«**М**а**т**ь**-**С**ы**р**а**-**З**е**м**л**я** + **П**р**я**д**и**л**ь**щ**и**ц**а**»**
- **С**в**я**т**ы**е** **м**е**с**т**а**:** **р**е**к**и** (**Т**а**р**а**, **М**о**р**а**ч**а**), **к**о**л**о**д**ц**ы** (**С**в**. **С**т**е**ф**а**н**), **с**т**а**р**ы**е** **д**у**б**ы** (**С**в**е**т**и** **С**т**е**ф**а**н**)

**4. ВЕЛЕС («Хозяин подземного мира»):
- **«**В**е**л**е**с**»** = **«**в**е**л**е** = **с**к**о**т**»** (**с**л**а**в**я**н**с**к**о**е**)
- **Ч**е**р**н**о**г**о**р**с**к**а**я** **р**а**з**н**о**в**и**д**н**о**с**т**ь**:** **В**е**л**е**с** = **п**а**т**р**о**н** **с**к**о**т**а** (в **Ч**е**р**н**о**г**о**р**и**и** — **к**о**з**ы**, **о**в**ц**ы**, **К**Р**С**)
- **С**в**я**т**ы**е** **м**е**с**т**а**:** **п**а**с**т**б**и**щ**а** (**К**о**м**о**в**и**, **Д**у**р**м**и**т**о**р**), **л**е**с**а** (**К**о**т**о**р**с**к**и**е** **л**е**с**а**), **о**з**ё**р**а** (**С**к**у**т**а**р**и**)
- **С**в**я**т**ы**е** **м**е**с**т**а** **в** **Ч**е**р**н**о**г**о**р**и**и**:** **В**е**л**е**с** (**г**о**р**о**д**), **В**е**л**е**ш**к**а** (**р**е**к**а**), **В**е**л**е**ш**н**и**ц**а** (п**е**щ**е**р**а**)

**5. ДАЖДЬБОГ («Солнечный бог»):
- **«**Д**а**ж**д**ь**б**о**г**»** = **«**д**а**ж**д**ь** + **б**о**г**»** (**с**л**а**в**я**н**с**к**о**е**)
- **«**С**л**о**в**о** **о** **п**о**л**к**у** **И**г**о**р**е**в**е**»** (**о**к**.** **1**1**8**5**) — *«**Д**а**ж**д**ь**б**о**г**»*
- **Ч**е**р**н**о**г**о**р**с**к**а**я** **р**а**з**н**о**в**и**д**н**о**с**т**ь**:** **Д**а**ж**д**ь**б**о**г** = **«**С**у**р**ь**я**»** (**с**о**л**н**ц**е**), **п**а**т**р**о**н** **у**р**о**ж**а**я**
- **С**в**я**т**ы**е** **м**е**с**т**а**:** **с**о**л**н**е**ч**н**ы**е** **к**а**м**н**и** (Л**о**в**ч**е**н**), **к**р**у**г**л**ы**е** **п**л**о**щ**а**д**к**и** (С**т**а**р**а**я** **Ч**е**р**н**о**г**о**р**и**я**)
- **С**в**я**т**ы**е** **м**е**с**т**а** **в** **Ч**е**р**н**о**г**о**р**и**и**:** **Д**а**ж**д**ь**б**о**г** = **«**Д**а**ж**д**ь**б**о**г**»** (**т**о**п**о**н**и**м**ы**)

**6. ПЕРУН («Громовержец»):
- **«**П**е**р**у**н**»** = **«**п**е**р**у**н** = **п**е**р**у**т**ь**»** (**с**л**а**в**я**н**с**к**о**е**)
