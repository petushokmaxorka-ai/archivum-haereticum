**MACEDONIA PAGANITAS** — **«**м**а**к**е**д**о**н**с**к**о**е** **с**л**а**в**я**н**с**к**о**е** **я**з**ы**ч**е**с**т**в**о**»**:

**С**т**р**а**т**е**г**и**я** **REPLICATIO + NODUS + EDITIO**:
- **REPLICATIO**: **м**а**к**е**д**о**н**ц**ы** **к**о**п**и**р**у**ю**т** **о**б**щ**е**с**л**а**в**я**н**с**к**у**ю** **в**е**р**у** (**Р**о**д**, **М**о**к**о**ш**ь**, **С**в**а**р**о**г**, **В**е**л**е**с**)
- **NODUS**: **М**а**к**е**д**о**н**и**я** = **у**з**е**л** **с**л**а**в**я**н**с**к**о**г**о** + **г**р**е**ч**е**с**к**о**г**о** + **ф**р**а**к**и**й**с**к**о**г**о** + **и**л**л**и**р**и**й**с**к**о**г**о**
- **EDITIO**: **о**б**р**я**д**ы** **о**п**у**б**л**и**к**о**в**а**н**ы** **В**е**р**к**о**в**и**ч**е**м** + **М**а**к**е**д**о**н**с**к**о**е** **в**о**з**р**о**ж**д**е**н**и**е** XIX** **в.**

**Цель** **архива** — **через** **македонское** **славянское** **язычество** **увидеть** **«общий** **закон** **Божий** **через** **природные** **силы»**; **доказательство** **«параллельной** **передачи** **общего** **индоевропейского** **культа** **природных** **сил** **в** **Славии»**; **«двоеверие** **=** **«норма»** **IX-XIX** **вв., **языческие** **боги** **под** **«**х**р**и**с**т**и**а**н**с**к**о**й** **о**б**о**л**о**ч**к**о**й** **с**в**я**т**ы**х**; **«общий** **закон** **через** **македонское** **язычество** = **«**о**б**щ**и**й** **з**а**к**о**н** **ч**е**р**е**з** **и**н**д**о**е**в**р**о**п**е**й**с**к**у**ю** **в**е**р**у** **в** **п**р**и**р**о**д**н**ы**е** **с**и**л**ы**»**.

**Г**л**а**в**н**ы**й** **а**р**г**у**м**е**н**т**: **«**ч**е**р**е**з** **м**а**к**е**д**о**н**с**к**о**е** **с**л**а**в**я**н**с**к**о**е** **я**з**ы**ч**е**с**т**в**о** **у**в**и**д**е**т**ь** **о**б**щ**и**й** **з**а**к**о**н** **Б**о**ж**и**й** **ч**е**р**е**з** **п**р**и**р**о**д**н**ы**е** **с**и**л**ы** — **Р**о**д** = **«**П**р**а**б**о**г**»** (**п**а**т**р**и**а**р**х**), **М**о**к**о**ш**ь** = **«**В**е**л**и**к**а**я** **М**а**т**ь**»** (**м**а**т**р**и**а**р**х**), **С**в**а**р**о**г** = **«**к**у**з**н**е**ц**»** (**т**в**о**р**е**ц**), **В**е**л**е**с** = **«**х**о**з**я**и**н** **п**о**д**з**е**м**н**о**г**о** **м**и**р**а**»** (**ж**р**е**ц**), **Д**а**ж**д**ь**б**о**г** = **«**С**о**л**н**ц**е**»** (**ц**а**р**ь**)**; **«**с**л**а**в**я**н**с**к**а**я** **р**е**л**и**г**и**я** **у**р**о**ж**а**я** + **с**к**о**т**а** + **ж**е**н**щ**и**н**ы** + **с**о**л**н**ц**а** + **г**р**о**з**ы**»** = **«**о**б**щ**и**й** **и**н**д**о**е**в**р**о**п**е**й**с**к**и**й** **к**у**л**ь**т** **п**р**и**р**о**д**ы**»**.
