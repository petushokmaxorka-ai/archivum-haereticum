### I. СВОДКА

**Пантеон полабских славян (X-XII вв. н.э.) = один из самых развитых языческих пантеонов Европы**:

**Главные боги** (8 верховных):
1. **Свентовит (Svetovid / Svantevit / Zvantevith)** — *«Ретра, Аркона, верховный бог»*
2. **Яровит (Jarovit / Gerovit)** — *«Вольгаст, бог войны»*
3. **Триглав (Triglav)** — *«Волин, Щецин, трёхглавый»*
4. **Радегаст (Radegast / Redigast)** — *«Гавельберг, бог войны»*
5. **Чернобог (Černobog / Crnobog)** — *«Черняхов, темный бог»*
6. **Белая Богиня (Siva / Živa / Siwa)** — *«мать богов, мать-земля»*
7. **Припегала (Pripegala)** — *«Старгард, богиня весны»*
8. **Поревит (Porevit / Porenut)** — *«Вольгаст, бог плодородия»*

**Главные капища** (8 великих):
1. **Ретра (Radigast / Riedegost)** — *«земля лютичей, главный храм Свентовита»*
2. **Аркона (Arkona / Arkona na Rujane)** — *«остров Рюген, храм Свентовита»*
3. **Щецин (Szczecin / Stettin)** — *«поморяне, храм Триглава»*
4. **Волин (Wolin / Wollin)** — *«поморяне, храм Триглава»*
5. **Кореница (Cörrenz / Corinza / Korinza)** — *«остров Рюген»*
6. **Гавельберг (Havelberg)** — *«древляне, храм Радегаста»*
7. **Черняхов (Czarne Bogi / Tempelberg)** — *«древляне, храм Чернобога»*
8. **Вольгаст (Wolgast / Hologost)** — *«бодричи, храм Яровита и Поревита»*
