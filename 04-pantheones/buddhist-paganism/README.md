# BUDDHIST-PAGANISM
## Буддийское язычество: Буддизм, Бон-Тибет, Нага, Сюгэндо, Чхондоизм
### «Южная нирвана» — от Индии до Тибета, от Японии до Кореи

**CLASSIFICATIO: ◆ FIDELIS (документальная база: Трипитака, Лотосовая сутра, Алмазная сутра, Винаи, Абхидхармакоша) / ◇ SUSPECTUS-FERTILIS (ранний буддизм vs. махаяна vs. ваджраяна — рабочие гипотезы) / ✠ DAMNATUS («Буддизм — это атеизм» / «Бон — это просто шаманизм, а не буддизм» / «Наги — это просто змеи» — DAMNATA как упрощения) / ✚ CREDO (личная позиция владельца: **стратегия NODUS** — как Бактрия, буддизм «узел» между Индией, Тибетом, Китаем, Японией, Кореей, Юго-Восточной Азией; **«закон Божий» через «закон кармы» (дхарма)** — Будда = «Совершенный Просветлённый», Майтрея = «Будда будущего»; **двоеверие** — синкретизм буддизма с бон, синто, даосизмом, конфуцианством)**

### II. БУДДИЗМ (BUDDHISMUS) — Тхеравада, Махаяна, Ваджраяна

**Гаутама** **Будда** (**Siddhartha** **Gautama**, **ок.** **563–483** **до** **н.** **э.**):
- **«**Принц**»** **из** **Капилавасту** (**совр.** **Непал**/**Индия**)
- **«**Отшельник**»** (**бросил** **семью** **и** **трон**)
- **«**Просветлённый**»** **под** **деревом** **Бодхи** (**«**Фикус** **священный**»**)

### VII. СТРАТЕГИЯ NODUS

**Буддийская** **стратегия** **NODUS** (**узел**) — **как** **Бактрия**:

### VIII. ОБЩИЙ «ЗАКОН БОЖИЙ ЧЕРЕЗ ЗАКОН КАРМЫ» В БУДДИЗМЕ

| **Понятие** | **Буддизм** | **Славянский** **аналог** |
|---|---|---|
| **Закон** | **Дхарма** (**«**закон** **бытия**»**) | **«**Правь**-**закон**»** |
| **Страдание** | **Дукха** | **«**Горе**-**беда**»** |
| **Освобождение** | **Нирвана** | **«**Рай**»** |
| **Путь** | **Марга** (**Восьмеричный** **путь**) | **«**Спасение**»** |
| **Просветление** | **Бодхи** | **«**Истина**»** |
| **Перерождение** | **Сансара** | **«**Коло** **года**»** (**годовой** **цикл**)** |
| **Совершенный** | **Будда** | **«**Святой**»** |
| **Будущий** **Будда** | **Майтрея** | **«**Святой** **грядущий**»** (**Китеж**, **Беловодье**)** |

**Главная** **идея**: **«**закон** **кармы** **=** **«**закон** **Божий** **через** **«**закон** **бытия**»**»** — **общий** **для** **всех** **индоевропейцев** **и** **буддистов**, **но** **выраженный** **по-**разному**. **Буддизм** **видит** **«**закон**»** **как** **«**дхарму**»** (**«**закон** **бытия**»**), **а** **не** **как** **«**закон** **Бога**»**; **но** **через** **«**карму**»** **он** **сохраняет** **«**общий** **закон**»**.

### IX. DAMNATA-ПРОТОКОЛ

**DAMNATA (нельзя писать как факт):**

1. «**Буддизм** — **это** **атеизм**»** — DAMNATA. **Буддизм** — **«**не**-теизм**»**, **но** **с** **«**богами**»** (**Брахма**, **Индра**, **Четыре** **небесных** **царя**)
2. «**Бон** — **это** **просто** **шаманизм**, **а** **не** **буддизм**»** — DAMNATA. **Бон** — **«**отдельная** **религия**»**, **но** **с** **«**буддийским** **субстратом**»** (**впитала** **буддизм** **с** **VII** **в.**)
3. «**Наги** — **это** **просто** **змеи**»** — DAMNATA. **Наги** — **«**полузмеи-**полулюди**»**, **«**правители** **подземного** **мира**»**
4. «**Сюгэндо** — **это** **«**японский** **шаманизм**»** — DAMNATA. **Сюгэндо** — **«**четырёхчастный** **синкретизм**»** (**синто** + **буддизм** + **даосизм** + **шаманизм**)
5. «**Чхондоизм** — **«**просто** **«**путь** **небес**»** — DAMNATA. **Чхондоизм** — **«**полноценная** **новая** **религия**»** (**XX** **в.**)
6. «**Буддизм** — **«**индийская** **религия**»** — DAMNATA. **Буддизм** — **«**мировая** **религия**»** (**Китай**, **Япония**, **Корея**, **Тибет**, **Монголия**, **Шри**-**Ланка**, **ЮВА**)
7. «**Далай-**лама** — **«**просто** **политический** **лидер**»** — DAMNATA. **Далай-**лама** — **«**воплощение** **Авалокитешвары**»** (**теологическое** + **политическое**)
