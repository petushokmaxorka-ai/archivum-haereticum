### «Язык-остров и страна Сугаррамурди» — от Сугаара до ламьяков

**CLASSIFICATIO: ◆ FIDELIS (документальная база: Trask 1997, Hualde/Lakarra 2006, Caro Baroja 1945, Mythologia Vasconum) / ◇ SUSPECTUS-FERTILIS (баскский язык как «изолят» vs. связи с иберийским/кавказским — рабочие гипотезы) / ✠ DAMNATUS («Баски — это «до-индоевропейский» / «Баски = иберы» / «Баски = «отсталое» племя» — DAMNATA как упрощения) / ✚ CREDO (личная позиция владельца: **стратегия ISOLATIO + NODUS** — баски изолировались в Пиренеях, иберийцы «узел» между Африкой, Средиземноморьем, Кельтикой; **«закон Божий через Мари/Сугаара/ламьяков»** — Мари = «Верховный бог», Сугаар = «змей-муж», ламьяки = «демоницы»; **двоеверие** — синкретизм с христианством)**

### III. СТРАТЕГИЯ ISOLATIO + NODUS

**Баскская** **стратегия** **ISOLATIO** + **NODUS** (**изоляция**/**узел**):

> **«**Баски** **изолировались** **в** **Пиренеях**; **баскский** **язык** **—** **«**язык**-**остров**»** (**до**-**индоевропейский**); **басконы** **—** **«**узел**»** **между** **Африкой**, **Средиземноморьем**, **Кельтикой**; **«**Мари**»** **=** **«**общий** **Верховный** **бог**»**; **«**Сугаар**»** **=** **«**змей**-**муж**»**; **двоеверие** **с** **христианством** **сохранило** **языческий** **субстрат** **до** **XXI** **в.**»** — **это** **ISOLATIO** **+** **NODUS**.

**Сравнение** **с** **другими** **стратегиями**:
- **Баски** — **ISOLATIO** (**изоляция**)
- **Иберия** — **NODUS** (**узел**)
- **Айны** — **ISOLATIO** (**изоляция**)
- **Эфиопия**/**Семитское** — **ISOLATIO**/**RESISTENTIA**
- **Средиземноморье** — **CONCURRENTIA** (**соперничество**)
- **Славия** — **REPLICATIO** (**копирование**)

### V. DAMNATA-ПРОТОКОЛ

**DAMNATA (нельзя писать как факт):**

1. «**Баски** — **«**до**-**индоевропейский** **народ**»** — DAMNATA. **Баски** = **«**полноценная** **культура**»** (**«**до**-**индоевропейский** **язык**»** — **гипотеза**)
2. «**Баски** = **«**иберы**»** — DAMNATA. **Баски** ≠ **иберы** (**разные** **языки**)
3. «**Баски** — **«**отсталое** **племя**»** — DAMNATA. **Баски** = **«**полноценная** **культура**»** (**с** **мифологией**, **обрядами**, **языком**)
4. «**Мари** = **«**просто** **«**бог**»** — DAMNATA. **Мари** = **«**Верховный** **бог**»** **с** **олитическим** **пантеоном** (**Сугаар**, **Ама** **Лур**, **Эац**, **Ил**)
5. «**Сугаар** = **«**дьявол**»** — DAMNATA. **Сугаар** = **«**змей**-**муж**»** (**«**муж** **Мари**»**) — **общий** **индоевропейский** **мотив**
6. «**Ламьяки** = **«**русалки**»** — DAMNATA. **Ламьяки** = **«**духи** **предков**»** (**«**не** **русалки**»**)
7. «**Итурри**-**О**-**Си** = **«**просто** **«**три** **головы**»** — DAMNATA. **Итурри**-**О**-**Си** = **«**триединый** **бог**»** (**«**общий** **индоевропейский** **мотив**»**)
