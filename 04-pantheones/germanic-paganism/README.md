### VI. СТРАТЕГИЯ REPLICATIO + NODUS

**Германо-скандинавская** **стратегия** **REPLICATIO** + **NODUS** (**копирование**/**узел**):

> **«**Германо**-**скандинавский** **пантеон** **—** **«**копия**»** **индо**-**европейского** **прототипа** (**Dieur** ***Patēr***, ***Deiuos** **Pater*** **=** **«**Юпитер**»**); **«**северный** **узел**»** **между** **Германией**, **Скандинавией**, **Исландией**, **Гренландией**; **двоеверие** **с** **христианством** (**крещение** **Дании** **965**, **Норвегии** **995**, **Исландии** **1000**)**»** — **это** **REPLICATIO** **+** **NODUS**.

**Сравнение** **с** **другими** **стратегиями**:
- **Скандинавия** — **REPLICATIO** + **NODUS** (**копирование** + **узел**)
- **Славия** — **REPLICATIO** (**копирование**)
- **Германия** — **REPLICATIO** (**копирование**)
- **Кельтика** — **REPLICATIO** (**копирование**)
- **Баски** — **ISOLATIO** + **NODUS** (**изоляция** + **узел**)
- **Средиземноморье** — **CONCURRENTIA** (**соперничество**)
