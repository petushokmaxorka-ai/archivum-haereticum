### V. СТРАТЕГИЯ REPLICATIO + NODUS

**Скандинавская** **стратегия** **REPLICATIO** + **NODUS** (**копирование**/**узел**):

> **«**Скандинавский** **пантеон** — **«**копия**»** **индо**-**европейского** **прототипа** (**Dieur** ***Patēr***, ***Deiuos** **Pater*** **=** **«**Юпитер**»**); **«**северный** **узел**»** **между** **Германией**, **Скандинавией**, **Исландией**, **Гренландией**; **двоеверие** **с** **христианством** (**крещение** **Дании** **965**, **Норвегии** **995**, **Исландии** **1000**)**»** — **это** **REPLICATIO** **+** **NODUS**.

**Сравнение** **с** **другими** **стратегиями**:
- **Скандинавия** — **REPLICATIO** + **NODUS** (**копирование** + **узел**)
- **Славия** — **REPLICATIO** (**копирование**)
- **Германия** — **REPLICATIO** (**копирование**)
- **Кельтика** — **REPLICATIO** (**копирование**)
- **Баски** — **ISOLATIO** + **NODUS** (**изоляция** + **узел**)
- **Средиземноморье** — **CONCURRENTIA** (**соперничество**)
- **Эфиопия**/**Семитское** — **ISOLATIO**/**RESISTENTIA**

### VII. DAMNATA-ПРОТОКОЛ

**DAMNATA (нельзя писать как факт):**

1. «**Викинги** — **«**дикие** **завоеватели**»** — DAMNATA. **Викинги** = **«**полноценная** **цивилизация**»** (**торговля**, **ремесло**, **поэзия**, **законодательство**)
2. «**Скандинавское** **язычество** — **«**примитивное**»** — DAMNATA. **Скандинавское** **язычество** = **«**полноценная** **религия**»** (**с** **пантеоном**, **мифологией**, **обрядами**, **этикой**)
3. «**Скандинавские** **боги** — **«**демоны**»** — DAMNATA. **Скандинавские** **боги** = **«**полноценные** **боги**»** (**как** **греческие**/**римские**/**славянские**)
4. «**Один** = **«**просто** **«**бог**»** — DAMNATA. **Один** = **«**Всеотец**»** (**«**Wōden**»** — **«**Wuotan**»**) — **общий** **индо**-**европейский** **бог**
5. «**Тор** = **«**громовержец**»** — DAMNATA. **Тор** = **«**громовержец**»** (**«**Мьёлльнир**»** = **«**молот**»**) — **общий** **индо**-**европейский** **бог** (**как** **Перун**/**Зевс**/**Таранис**)
6. «**Локи** = **«**злой** **дух**»** — DAMNATA. **Локи** = **«**трикстер**»** (**«**обманщик**»**/**«**помощник**»**) — **общий** **индо**-**европейский** **архетип** (**как** **Гермес**/**Луг**)
7. «**Старшая** **Эдда** — **«**примитивный** **текст**»** — DAMNATA. **Старшая** **Эдда** = **«**высокая** **поэзия**»** (**как** **«**Илиада**»**/**«**Одиссея**»**)
8. «**Руны** — **«**магические** **знаки**»** — DAMNATA. **Руны** = **«**алфавит**»** (**«**футарк**»**) + **«**священное** **письмо**»**
