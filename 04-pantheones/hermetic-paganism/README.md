### VIII. DAMNATA-ПРОТОКОЛ

**DAMNATA (нельзя писать как факт):**

3. «**Яхве**-**южный** = **«**арабский** **Аллах**»** — DAMNATA. **Аллах** = **«**Бог**»** **в** **исламе**, **YHWH** = **«**имя**»** **в** **иудаизме**; **разные** **традиции**
4. «**«**Изумрудная** **скрижаль**»** — **«**древний** **египетский** **текст**»** — DAMNATA. **«**Изумрудная** **скрижаль**»** **—** **II** **в.** **н.** **э.**, **арабский** **текст** **Аполлония** **Тианского**
5. «**Гермес** **Трисмегист** — **«**историческое** **лицо**»** — DAMNATA. **Гермес** **Трисмегист** = **«**сплав** **Гермеса** **и** **Тота**»**, **спорная** **фигура**
6. «**Тот** — **«**просто** **бог** **письма**»** — DAMNATA. **Тот** = **«**писец** **богов** **на** **психостазисе**»** (**суд** **Осириса**), **«**изобретатель** **иероглифов**»**, **«**бог** **мудрости** **и** **магии**»**
