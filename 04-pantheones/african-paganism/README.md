# AFRICAN-PAGANISM
## Африканское язычество: банту, малагасийский, ифа-сантерия
### «Анимизм предков» — от Западной Африки до Мадагаскара, от Кубы до Гаити

**CLASSIFICATIO: ◆ FIDELIS (документальная база: «Одун Ифа» (Ифа) XVIII в., коды Анга, традиция Вуду/Водун, «Hainteny» Мадагаскара, «Сказки дядюшки Римуса» Джоэля Чандлера Харриса 1880) / ◇ SUSPECTUS-FERTILIS (банту vs. йоруба vs. фон — рабочие гипотезы) / ✠ DAMNATUS («Африканское язычество — это «примитивный анимизм» / «Вуду — это «чёрная магия» / «Ифа — это «гадание» — DAMNATA как упрощения) / ✚ CREDO (личная позиция владельца: **стратегия NODUS** — как Бактрия, африканские пантеоны «узел» между Западной Африкой, Мадагаскаром, Кубой, Гаити, Бразилией (Кандумбле), США (Новоорлеан); **«закон Божий через Ориша»** — Ориша = «святые предки»; **двоеверие** — синкретизм с христианством)**
Datum: 022.M3

> *Заявление владельца: **африканское** **язычество** — **«**Анимизм** **предков**»**, **стратегия** **NODUS** **как** **Бактрия**/**буддизм**/**индуизм**/**дальневосточное**/**шаманизм**. **3** **досье**: **bantu** (**Зулусы**/**Ашанти**/**Йоруба**/**Ибо**), **malagasy** (**Мадагаскар**), **ifa-santeria** (**Ифа**/**Сантерия**/**Вуду**/**Кандумбле**). **Помним** **цель**: **«**через** **африканское** **язычество** **увидеть** **«**общий** **закон** **Божий** **через** **Ориша**»**.*
>

### I. СВОДКА

**Африканское** **язычество** — **«**Анимизм** **предков**»** (**палеолит** — **XXI** **в.**).

### V. СТРАТЕГИЯ NODUS

**Африканская** **стратегия** **NODUS** (**узел**) — **как** **Бактрия**/**буддизм**/**индуизм**/**дальневосточное**/**шаманизм**:

> **«**Африканские** **пантеоны** **были** **«**узлом**»** **между** **Западной** **Африкой**, **Карибами**, **Южной** **Америкой**, **Северной** **Америкой**; **трансатлантическая** **работорговля** **перенесла** **Ориша**/**Вуду**/**Кандумбле** **в** **Новый** **Свет**; **Ифа**/**Вуду**/**Сантерия**/**Кандумбле** = **«**новоафриканские** **религии**»** **с** **общим** **корнем**»** — **это** **NODUS**.

**Сравнение** **с** **другими** **стратегиями**:
- **Африканское** — **NODUS** (**узел**)
- **Буддизм**/**Индуизм**/**Дальневосточное**/**Шаманизм** — **NODUS** (**узел**)
- **Средиземноморье** — **CONCURRENTIA** (**соперничество**)
- **Славия** — **REPLICATIO** (**копирование**)
- **Церковь** **Востока** — **TRANSMISSIO** (**передача**)
- **Эфиопия**/**Семитское** — **ISOLATIO**/**RESISTENTIA**

### VII. DAMNATA-ПРОТОКОЛ

**DAMNATA (нельзя писать как факт):**

1. «**Африканское** **язычество** — **«**примитивный** **анимизм**»** — DAMNATA. **Африканские** **религии** = **«**полноценные** **религии**»** (**с** **пантеоном**, **мифологией**, **обрядами**)
2. «**Вуду** — **«**чёрная** **магия**»** — DAMNATA. **Вуду** = **«**религия**»** (**«**Духи** **Вуду**»** = **«**лоа**»** = **«**святые** **предки**»**)
3. «**Ифа** — **«**просто** **гадание**»** — DAMNATA. **Ифа** = **«**оракул** **+** **256** **знаков** **+** **теология**»**
4. «**Сантерия** — **«**смешение** **культов**»** — DAMNATA. **Сантерия** = **«**полноценная** **новоафриканская** **религия**»**
5. «**Фамадихана** — **«**ужасный** **языческий** **обряд**»** — DAMNATA. **Фамадихана** = **«**перезахоронение** **предков**»** (**«**память** **о** **предках**»**)
6. «**Олорун** — **«**просто** **«**бог** **неба**»** — DAMNATA. **Олорун** = **«**Верховный** **бог**»** **с** **«**олитическим** **пантеоном**»** (**Ориша**)
